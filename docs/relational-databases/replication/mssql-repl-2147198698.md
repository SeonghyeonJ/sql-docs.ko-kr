|attribute|값|  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|-2147198698|  
|이벤트 원본|MSSQLServer|  
|심볼 이름||  
|메시지 텍스트|이 게시에 대한 스냅샷이 최신 스냅샷이 아닙니다. 구독을 동기화하려면 스냅샷 에이전트를 다시 실행해야 합니다.|  
  
## <a name="explanation"></a>설명

## <a name="user-action"></a>사용자 동작  
 먼저 스냅샷 에이전트를 다시 실행해야만 구독을 동기화할 수 있습니다.
