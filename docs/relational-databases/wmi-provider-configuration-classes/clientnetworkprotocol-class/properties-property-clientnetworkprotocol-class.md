# <a name="properties-property-clientnetworkprotocol-class"></a>Properties 속성(ClientNetworkProtocol 클래스)

  [클라이언트 프로토콜 구성](../../../database-engine/configure-windows/configure-client-protocols.md)에서 지정한 현재 클라이언트 네트워크 프로토콜과 연결된 속성을 가져옵니다.

