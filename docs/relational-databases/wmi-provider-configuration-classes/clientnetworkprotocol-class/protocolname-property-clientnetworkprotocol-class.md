# <a name="protocolname-property-clientnetworkprotocol-class"></a>ProtocolName 속성(ClientNetworkProtocol 클래스)
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  [클라이언트 프로토콜 구성](../../../database-engine/configure-windows/configure-client-protocols.md)에서 지정한 현재 네트워크 프로토콜의 이름을 가져옵니다.

```  
  
object.ProtocolName [= value]  
```

 *object*
