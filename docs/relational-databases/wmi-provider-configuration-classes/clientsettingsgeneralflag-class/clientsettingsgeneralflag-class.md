# <a name="clientsettingsgeneralflag-class"></a>ClientSettingsGeneralFlag 클래스
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  [ClientSettingsGeneralFlag 클래스](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md) 는 클라이언트 인스턴스의 일반 플래그 설정을 나타냅니다 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
 [ClientSettingsGeneralFlag 클래스](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md) 를 사용 하 여 다음을 수행할 수 있습니다.  
  
-   플래그 이름을 봅니다.  
  
-   플래그 인덱스 값을 봅니다.  
  
-   플래그 값을 보고 수정합니다.
