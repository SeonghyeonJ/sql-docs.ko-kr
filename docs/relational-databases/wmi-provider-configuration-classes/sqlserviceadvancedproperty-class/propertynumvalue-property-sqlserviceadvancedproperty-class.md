# <a name="propertynumvalue-property-sqlserviceadvancedproperty-class"></a>PropertyNumValue 속성(SqlServiceAdvancedProperty 클래스)

  고급 속성의 숫자 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object.PropertyNumValue [= value]  
```
