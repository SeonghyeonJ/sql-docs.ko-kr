# <a name="propertyname-property-sqlserviceadvancedproperty-class"></a>PropertyName 속성(SqlServiceAdvancedProperty 클래스)
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  고급 속성의 이름을 가져오거나 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object.PropertyName [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 고급 속성을 나타내는 [SqlServiceAdvancedProperty 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 고급 속성의 이름을 지정하는 문자열 값입니다.
