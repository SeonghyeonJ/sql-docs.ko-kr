# <a name="sqlserviceadvancedproperty-class"></a>SqlServiceAdvancedProperty 클래스
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  합니다 [SqlServiceAdvancedProperty 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) 에서 참조 되는 서비스의 고급 속성을 나타내는 합니다 [SqlService 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) 개체입니다.  
  
 합니다 [AdvancedProperties 속성 (SqlService 클래스)](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/advancedproperties-property-sqlservice-class.md) 배열을 참조 [SqlServiceAdvancedProperty 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) 개체입니다.  
  
 합니다 [서비스 시작 및 중지](https://technet.microsoft.com/library/ms174886\(v=sql.105\).aspx) 클래스는 서비스에 고유한 속성을 나타냅니다. 이러한 속성은 연결 된 속성 목록에는 [SqlService 클래스](https://technet.microsoft.com/library/ms186497.aspx) 클래스입니다. 합니다 [SqlServiceAdvancedProperty 클래스](https://technet.microsoft.com/library/ms182447.aspx) 클래스를 사용 하면 문자열, 숫자 또는 부울 속성의 표현입니다. 지정한 서비스의 고유한 속성을 볼 수 있습니다.
