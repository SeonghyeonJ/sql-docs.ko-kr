# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>PropertyStrValue 속성(SqlServiceAdvancedProperty 클래스)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/applies-to-version/sqlserver.md)]
  고급 속성의 문자열 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 고급 속성을 나타내는 [SqlServiceAdvancedProperty 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 고급 속성의 값을 지정하는 문자열 값입니다.
