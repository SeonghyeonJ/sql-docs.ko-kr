# <a name="localdb_error_xevent_failed"></a>LOCALDB_ERROR_XEVENT_FAILED
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>세부 정보  
  
| attribute | 값 |
| --------- | ----- |
|제품 이름|SQL Server|  
|이벤트 ID|279|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|로컬 데이터베이스 인스턴스 API 내에서 XEvent 엔진을 시작하지 못했습니다.|  
  
## <a name="explanation"></a>설명

 로컬 데이터베이스 런타임을 다시 설치하고 작업을 다시 시도하십시오.
