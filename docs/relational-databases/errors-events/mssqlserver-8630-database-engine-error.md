## <a name="details"></a>세부 정보  
  
| attribute | 값 |  
| :-------- | :---- |  
|제품 이름|SQL Server|  
|이벤트 ID|8630|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|EXECUTION_ERR|  
|메시지 텍스트|내부 쿼리 프로세서 오류: 쿼리 프로세서 실행 중 오류가 발생했습니다.|
