# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  SQLNumParams을 통해 [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)] 데이터베이스 엔진의 개선으로 인해 예상결과에 대한 보다 정확한 설명을 얻을 수 있습니다. 이러한 보다 정확한 결과는 이전 버전의 에서 SQLNumParams에서 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]반환된 값과 다를 수 있습니다.  
  
 자세한 내용은 [메타데이터 검색](../../relational-databases/native-client/features/metadata-discovery.md)을 참조하세요.
