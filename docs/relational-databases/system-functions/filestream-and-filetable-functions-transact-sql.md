# <a name="filestream-and-filetable-functions-transact-sql"></a>Filestream 및 FileTable 함수(Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  이 섹션에서는 FILESTREAM 및 FileTable 기능과 관련된 시스템 함수에 대해 설명합니다.  
  
 [FileTableRootPath&#40;Transact-SQL&#41;](../../relational-databases/system-functions/filetablerootpath-transact-sql.md)  
 특정 FileTable 또는 현재 데이터베이스에 대한 루트 수준 UNC 경로를 반환합니다.  
  
 [GetFileNamespacePath&#40;Transact-SQL&#41;](../../relational-databases/system-functions/getfilenamespacepath-transact-sql.md)  
 FileTable의 파일 또는 디렉터리에 대한 UNC 경로를 반환합니다.  
  
 [GetPathLocator&#40;Transact-SQL&#41;](../../relational-databases/system-functions/getpathlocator-transact-sql.md)  
 FileTable의 지정된 파일 또는 디렉터리에 대한 경로 로케이터 ID 값을 반환합니다.  
  
 [PathName&#40;Transact-SQL&#41;](../../relational-databases/system-functions/pathname-transact-sql.md)  
 FILESTREAM BLOB(Binary Large Object)의 경로를 반환합니다.  
  
 FileTables 기능에 대한 자세한 내용은 [FileTables &#40;SQL Server&#41;](../../relational-databases/blob/filetables-sql-server.md)를 참조하세요.
