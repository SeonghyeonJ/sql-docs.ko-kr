# <a name="spatial-data-catalog-views"></a>공간 데이터 카탈로그 뷰
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  이 섹션에서는 공간 데이터에 대한 정보를 제공하는 카탈로그 뷰에 대해 설명합니다.  
  
 [sys.spatial_index_tessellations&#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-spatial-index-tessellations-transact-sql.md)  
 공간 분할 구성표 및 각 공간 인덱스의 매개 변수에 대한 정보를 나타냅니다.  
  
 [sys.spatial_indexes&#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-spatial-indexes-transact-sql.md)  
 공간 인덱스의 주요 인덱스 정보를 나타냅니다.  
  
 [spatial_reference_systems &#40;Transact-sql&#41;](../../relational-databases/system-catalog-views/sys-spatial-reference-systems-transact-sql.md)  
 지원되는 공간 참조 시스템(SRID)을 나열합니다.
