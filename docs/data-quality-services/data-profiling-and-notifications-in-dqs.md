# <a name="data-profiling-and-notifications-in-data-quality-services-dqs"></a>DQS (Data Quality Services)의 데이터 프로 파일링 및 알림

[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

[!INCLUDE[ssDQSnoversion](../includes/ssdqsnoversion-md.md)]DQS ()는 데이터 프로 파일링 서비스입니다. DQS는 원본에서 데이터를 분석 하 고 DQS 작업의 데이터에 대 한 통계를 표시 합니다.

## <a name="attributes-goals-benefits"></a>특성, 목표, 이점

### <a name="attributes-of-dqs"></a>DQS의 특성

DQS 프로 파일링에는 다음과 같은 특성이 있습니다.

- 데이터 품질의 자동 측정을 제공 합니다.
- 는 DQS 기술 자료 관리 및 데이터 품질 프로젝트에 통합 되어 있습니다.
- 는 동적 이며 조정 가능 합니다.

### <a name="goals-of-dqs"></a>DQS 목표

DQS를 사용한 프로 파일링에는 다음과 같은 두 가지 주요 목표가 있습니다.

- 데이터 품질 프로세스를 안내 하 고 결정을 지원 합니다.
- 프로세스의 효율성을 평가 합니다.

### <a name="benefits-of-dqs"></a>DQS의 이점

DQS 프로 파일링에는 다음과 같은 이점이 있습니다.

- 원본 데이터의 품질에 대 한 통찰력을 제공 하 고 데이터 품질 문제를 식별 하는 데 도움이 됩니다.

- 는 데이터 품질 프로세스를 평가 하 고 다음을 안내 합니다.
  - 기술 자료 검색
  - 데이터 정리
  - 일치 정책
  - 일치 작업

- 가장 관련성이 높은 정보를 제공 합니다.

- 동작을 보증할 수 있는 중요 한 통계 또는 이벤트를 강조 하는 알림을 생성 합니다. DQS 알림은 종종 조건을 나타내고 정정 작업을 권장 합니다.

DQS 프로 파일링은 기술 자료 검색, 데이터 정리 및 데이터 일치를 제공 합니다. DQS는 데이터 분석 도구 이기도 합니다. 분석에 대 한 기술 자료를 만들 수 있습니다. 그런 다음 기술 자료를 사용 하 여 기술 자료 검색을 실행할 수 있습니다. 기술 자료 검색에서는 프로 파일링 통계를 사용 하 여 기술 자료가 검색, 정리 및 일치에 대 한 요구 사항을 충족 하는지 여부를 확인 합니다.

## <a name="how-dqs-profiling-works"></a><a name="How"></a>DQS 프로 파일링 작동 방법

DQS 프로 파일링은 기술 자료의 품질을 평가 하지 않습니다. 프로 파일링은 원본 데이터의 품질을 측정 합니다. 프로 파일링은 다음 활동의 영향을 나타내는 통계를 제공 합니다.

- 기술 자료 관리에서 특정 작업을 수행 합니다.
- 원본 데이터에 대 한 데이터 품질 프로젝트입니다.

### <a name="activity-context"></a>작업 컨텍스트

프로파일링은 항상 현재 수행 중인 특정 작업의 컨텍스트에서 실행됩니다. 화면에서 프로 파일링 탭을 클릭 하 여 프로 파일링 데이터를 표시할 수 있습니다. 이 클릭은 수행 하는 작업의 단계에서 이동 하지 않습니다. 프로 파일링 테이블은 프로세스가 수행 되는 동안 실시간으로 채워집니다. 따라서 데이터 품질 태스크를 수행 하는 동안 평가할 수 있습니다.

정리 또는 중복 제거 후 원본 데이터가 개선되었는지, 그렇다면 얼마나 개선되었는지 확인할 수 있습니다.

### <a name="profiling-is-based-on-counts"></a>프로 파일링은 개수를 기반으로 합니다.

모든 프로 파일링 숫자는 특정 값의 모양새 수를 나타냅니다. 대부분의 경우 숫자는 합계의 백분율을 참조 합니다. 고유성 메트릭은 예외입니다. 고유성 메트릭은 값의 모양 수에 관계 없이 값의 절대 개수를 나타냅니다.

### <a name="knowledge-driven-solution"></a>기술 자료 기반 솔루션

프로파일링은 DQS 기술 자료 기반 솔루션의 일부입니다. 프로 파일링은 기술 자료, 일치 또는 데이터 정리 프로세스에 대 한 정보를 제공 합니다. 이 정보는 데이터 원본 필드와 기술 자료 도메인 간의 매핑을 기반으로 합니다. 프로 파일링은 매핑이 완료 된 후에만 수행 됩니다. 모든 활동의 매핑 단계 중에는 프로 파일링이 수행 되지 않습니다.

프로파일링은 항상 특정 작업에 연결됩니다. 프로 파일링 프로세스는 도메인 _의_ 데이터가 아니라 도메인에 _매핑된_ 데이터에 대해 수행 됩니다.

프로파일링은 다음과 같은 작업 단계에 통합되어 있습니다.

- 기술 자료 검색 작업의 **도메인 값** **검색** 및 관리 단계입니다.

- 정리 **작업의 정리 및** **결과 관리 및 보기** 단계

- 일치 정책 작업의 일치 **정책** 및 **일치 결과** 단계입니다.

- 일치 작업의 **일치** 및 **내보내기** 단계입니다.

DQS에서는 도메인 관리 작업에 대한 프로파일링 통계를 제공하지 않습니다.

## <a name="profiling-data-by-activity"></a><a name="Activity"></a>작업 별로 데이터 프로 파일링

DQS 프로 파일링에서는 다음 표준 데이터 품질 차원을 사용 하 여 데이터의 품질을 나타냅니다.

- 완전성-데이터가 존재 하는 정도입니다.
- 정확도-데이터를 용도에 맞게 사용할 수 있는 정도입니다.
- 고유성-다른 값이 다른 엔터티를 나타내는 범위입니다.

기본적으로 NULL과 빈 값은 누락 된 것으로 간주 되거나 완전성 비율이 낮아집니다. 그러나 다른 값을 NULL에 해당 하는 값으로 정의할 수 있습니다. 이러한 값은 누락 된 것으로 간주 됩니다.

프로파일링에서 프로세스를 평가하는 데 필요한 통계를 제공하지만 통계 해석은 사용자가 수행해야 합니다. 통계를 열 단위로 보면서 프로파일링의 결과를 이해하세요.

### <a name="differing-sets-of-profiling-statistics"></a>서로 다른 프로 파일링 통계 집합

DQS 작업에는 다음과 같은 여러 프로파일링 통계 집합이 있습니다.

- 정리 작업에만 정확도에 대한 프로파일링 통계가 있습니다(도메인별 백분율). 정확도는 유효성, 일관성, 구문 오류 및 도메인 규칙의 영향을 받습니다.

- 정리 작업에만 원본의 올바름, 수정됨 및 제안과 도메인별 수정됨 및 제안 값에 대한 프로파일링 통계가 있습니다(모두 백분율).

- 정리 및 기술 자료 검색 작업에는 유효성에 대한 프로파일링 통계가 있습니다(레코드별 정리, 레코드 및 도메인별 기술 자료 검색). 일치 정책과 일치 작업에는 유효성에 대한 통계가 없습니다.

- 프로 파일링 통계는 원본 및 도메인에 대해 고유 하 게 제공 됩니다. 이 문은 다음 활동에 적용 됩니다.
  - 기술 자료 검색.
  - 일치 정책.
  - 일치.
  - 정리 작업에는 사용할 _수 없습니다_ .

DQS는 작업과 관련 된 특정 프로 파일링 통계를 제공 합니다. 자세한 내용은 다음 항목의 **프로 파일링** 섹션을 참조 하십시오.

- [기술 자료 검색 수행](../data-quality-services/perform-knowledge-discovery.md)

- [DQS &#40;내부&#41; 기술 자료를 사용 하 여 데이터 정리](../data-quality-services/cleanse-data-using-dqs-internal-knowledge.md)

- [일치 정책 만들기](../data-quality-services/create-a-matching-policy.md)

## <a name="profiling-data-in-activity-monitoring"></a><a name="Monitoring"></a>작업 모니터링의 데이터 프로 파일링

프로 파일링 정보는 Data Quality 클라이언트의 작업 페이지 뿐만 아니라 작업 모니터링 에서도 사용할 수 있습니다. 정보는 기술 자료 검색, 일치 정책, 일치 및 정리 작업과 관련 됩니다. 정보는 Data Quality 클라이언트의 작업 페이지 및 작업 모니터링에서 사용할 수 있습니다.

한 위치에서 다음 항목을 모두 볼 수 있습니다.

- 속성
- 활동의 관련 계산 프로세스
- 각 작업에 대해 생성 된 프로 파일링 정보

작업 테이블에서 작업을 선택 하 여 두 번째 테이블에 프로 파일링 결과를 표시할 수 있습니다. 또한 프로파일링 결과를 내보낼 수도 있습니다.

자세한 내용은 [DQS Administration](../data-quality-services/dqs-administration.md)을 참조하세요.

## <a name="notifications"></a><a name="Notifications"></a>알림을

DQS는 프로 파일링 된 통계를 기반으로 작업을 수행할 수 있는 시기를 표시 하는 알림을 생성 합니다. DQS는 데이터 원본에 대 한 중요 한 팩트에 대해 알림을 사용 합니다. 이러한 팩트는 실행 된 용도를 기준으로 현재 활동의 효율성을 보여 줍니다. 알림은 조건을 나타내는 팁과 권장 사항을 제공 합니다. 알림은 기술 자료 검색, 데이터 정리 또는 데이터 일치 작업을 개선할 수 있는 방법을 권장할 수도 있습니다.

DQS는 사용자에 게 중요할 수 있는 문제를 발견 하면 알림을 보냅니다. 예를 들어 완전성 및 정확성이 모두 100% 이면 데이터 정리 작업에서 수정 되거나 제안 된 값이 생성 되지 않을 수 있습니다. DQS에서 해당 상황에 대 한 알림을 게시할 수 있습니다. 이 알림은 작업이 필요 하지 않을 수 있음을 표시 합니다. 활동의 실행 여부는 결정에 따라 결정 됩니다.

알림은 **프로 파일링** 탭에서 느낌표가 있는 도구 설명으로 표시 됩니다. 알림의 통계 근거를 나타내기 위해 알림과 관련 된 통계가 빨간색으로 표시 됩니다.

### <a name="disable-notifications"></a>알림 사용 안 함

알림은 기본적으로 사용 하도록 설정 되어 있습니다. 그러나 Data Quality Client 홈 페이지를 사용 하 여 알림을 사용 하지 않도록 설정할 수 있습니다. **관리** 섹션에서 **일반 설정** 탭을 사용 합니다.

알림을 사용 하지 않도록 설정 하면 도구 설명이 표시 되지 않고 통계가 빨간색으로 표시 되지 않습니다. 알림을 사용 하지 않도록 설정 하는 동안 프로 파일링은 계속 작동 합니다. 알림을 사용 하지 않도록 설정 하면 성능이 향상 되지 않습니다.

특정 작업에 대한 알림과 관련된 특정 상태는 다음을 참조하세요.
