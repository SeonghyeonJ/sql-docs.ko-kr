# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
서버에 반환 되어야 하는 레코드를 지정 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|기본값 서버에 대 한 모든 행을 반환 합니다.|  
|**adMarshalModifiedOnly**|1|서버에 수정 된 행만 반환 합니다.|
