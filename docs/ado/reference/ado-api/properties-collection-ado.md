# <a name="properties-collection-ado"></a>Properties 컬렉션(ADO)
개체의 특정 인스턴스에 대 한 모든 [속성](../../../ado/reference/ado-api/property-object-ado.md) 개체를 포함 합니다.  
  
## <a name="remarks"></a>설명  
 일부 ADO 개체에는 **속성** 개체로 구성 된 **속성** 컬렉션이 있습니다. 각 **속성** 개체는 공급자와 관련 된 ADO 개체의 특성에 해당 합니다.  
  
> [!NOTE]
>  **속성** 개체를 사용 하는 방법에 대 한 자세한 설명은 [속성](../../../ado/reference/ado-api/property-object-ado.md) 개체 항목을 참조 하세요.  
  
 **레코드 집합 개체의** **동적 속성** 은 **레코드 집합** 을 닫을 때 범위 (사용할 수 없음)로 이동 합니다.
