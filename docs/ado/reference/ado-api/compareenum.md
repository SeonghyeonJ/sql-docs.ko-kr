# <a name="compareenum"></a>CompareEnum
책갈피로 표시 되는 두 레코드의 상대 위치를 지정 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adCompareEqual**|1|책갈피가 동일 함을 나타냅니다.|  
|**adCompareGreaterThan**|2|첫 번째 책갈피가 두 번째 책갈피 임을 나타냅니다.|  
|**adcompa**|0|첫 번째 책갈피가 두 번째 책갈피 앞에 있음을 나타냅니다.|  
|**adCompareNotComparable**|4|책갈피를 비교할 수 없음을 나타냅니다.|  
|**adCompareNotEqual**|3|책갈피가 동일 하지 않고 순서가 지정 되지 않았음을 나타냅니다.|
