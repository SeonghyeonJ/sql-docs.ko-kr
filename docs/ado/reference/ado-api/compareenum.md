# <a name="compareenum"></a>CompareEnum
해당 책갈피를 나타내는 두 레코드의 상대적 위치를 지정 합니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adCompareEqual**|1|책갈피 같은지를 나타냅니다.|  
|**adCompareGreaterThan**|2|두 번째 후 첫 번째 책갈피 임을 나타냅니다.|  
|**adCompareLessThan**|0|첫 번째 책갈피 두 번째 임을 나타냅니다.|  
|**adCompareNotComparable**|4|책갈피를 비교할 수 없음을 나타냅니다.|  
|**adCompareNotEqual**|3|책갈피에 같고 순차적 하지 없음을 나타냅니다.|
