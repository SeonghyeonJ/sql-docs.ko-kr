# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
텍스트에서 줄 구분 기호로 사용 되는 문자를 지정 [Stream](../../../ado/reference/ado-api/stream-object-ado.md) 개체입니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adCR**|13|캐리지 리턴을 나타냅니다.|  
|**adCRLF**|-1|기본. 캐리지 리턴 줄 바꿈을 나타냅니다.|  
|**adLF**|10|줄 바꿈을 나타냅니다.|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 해당  
 이러한 상수는 ADO/wfc 필요가 없습니다.
