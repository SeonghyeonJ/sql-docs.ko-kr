# <a name="ado-properties"></a>ADO 속성

|속성|설명|  
|-|-|  
|[AbsolutePage](./absolutepage-property-ado.md)|현재 레코드가 있는 페이지를 나타냅니다.|  
|[AbsolutePosition](./absoluteposition-property-ado.md)|**레코드 집합** 개체에 대 한 현재 레코드의 서 수 위치를 나타냅니다.|  
|[ActiveCommand](./activecommand-property-ado.md)|연결 된 **레코드 집합** 개체를 만든 **명령** 개체를 나타냅니다.|  
|[ActiveConnection](./activeconnection-property-ado.md)|지정 된 **명령**, **레코드 집합**또는 **Record** 개체가 현재 속해 있는 **연결** 개체를 나타냅니다.|  
|[ActualSize](./actualsize-property-ado.md)|필드 값의 실제 길이를 나타냅니다.|  
|[특성](./attributes-property-ado.md)|개체의 특성을 하나 이상 나타냅니다.|  
|[BOF 및 EOF](./bof-eof-properties-ado.md)|**BOF** 는 현재 레코드 위치가 레코드 집합 개체의 첫 번째 레코드 앞에 있음을 나타냅니다.<br /><br /> **EOF** 는 현재 레코드 위치가 레코드 집합 개체의 마지막 레코드 뒤에 있음을 나타냅니다.|  
|[책갈피](./bookmark-property-ado.md)|**레코드 집합** 개체의 현재 레코드를 고유 하 게 식별 하거나 **레코드 집합** 개체의 현재 레코드를 유효한 책갈피로 식별 되는 레코드로 설정 하는 책갈피를 나타냅니다.|  
|[CacheSize](./cachesize-property-ado.md)|메모리에 로컬로 캐시 된 레코드 **집합** 개체의 레코드 수를 나타냅니다.|  
|[장](./chapter-property-ado.md)|**ADORecordsetConstruction** 개체의/에서 OLE DB **챕터** 개체를 가져오거나 설정 합니다.|  
|[집합이](./charset-property-ado.md)|텍스트 **스트림의** 내용을 번역할 문자 집합을 나타냅니다.|  
|[CommandStream](./commandstream-property-ado.md)|**명령** 개체에 대 한 입력으로 사용 되는 스트림을 나타냅니다.|  
|[CommandText](./commandtext-property-ado.md)|공급자에 대해 실행할 명령의 텍스트를 나타냅니다.|  
|[CommandTimeout](./commandtimeout-property-ado.md)|시도를 종료 하 고 오류를 생성 하기 전에 명령을 실행 하는 동안 대기 하는 시간을 나타냅니다.|  
|[CommandType](./commandtype-property-ado.md)|**명령** 개체의 유형을 나타냅니다.|  
|[ConnectionString 속성](./connectionstring-property-ado.md)|데이터 원본에 대 한 연결을 설정 하는 데 사용 되는 정보를 나타냅니다.|  
|[ConnectionTimeout](./connectiontimeout-property-ado.md)|시도를 종료 하 고 오류를 생성 하기 전에 연결을 설정 하는 동안 대기 하는 시간을 나타냅니다.|  
|[Count](./count-property-ado.md)|컬렉션의 개체 수를 나타냅니다.|  
|[CursorLocation](./cursorlocation-property-ado.md)|커서 서비스의 위치를 나타냅니다.|  
|[CursorType](./cursortype-property-ado.md)|**레코드 집합** 개체에 사용 되는 커서 유형을 나타냅니다.|  
|[DataMember](./datamember-property.md)|**DataSource** 속성이 참조 하는 개체에서 검색 되는 데이터 멤버의 이름을 나타냅니다.|  
|[DataSource](./datasource-property-ado.md)|**레코드 집합** 개체로 표시할 데이터를 포함 하는 개체를 나타냅니다.|  
|[DefaultDatabase](./defaultdatabase-property.md)|**연결** 개체에 대 한 기본 데이터베이스를 나타냅니다.|  
|[DefinedSize](./definedsize-property.md)|**필드** 개체의 데이터 용량을 나타냅니다.|  
|[설명](./description-property.md)|**오류** 개체에 대해 설명 합니다.|  
|[지역](./dialect-property.md)|공급자가 **CommandText** 또는 **commandstream** 속성을 구문 분석 하는 데 사용할 구문 및 일반 규칙을 나타냅니다.|  
|[Direction](./direction-property.md)|**매개** 변수가 입력 매개 변수, 출력 매개 변수 또는 둘 다를 나타내는지, 아니면 매개 변수가 저장 프로시저의 반환 값 인지를 나타냅니다.|  
|[EditMode](./editmode-property.md)|현재 레코드의 편집 상태를 나타냅니다.|  
|[EOS](./eos-property.md)|현재 위치가 스트림의 끝에 있는지 여부를 나타냅니다.|  
|[Filter](./filter-property.md)|**레코드 집합**의 데이터에 대 한 필터를 나타냅니다.|  
|[HelpContext 및 HelpFile](./helpcontext-helpfile-properties.md)|**오류** 개체와 연결 된 도움말 파일 및 항목을 나타냅니다.<br /><br /> **HelpContextID** 는 도움말 파일의 항목에 대 한 컨텍스트 ID를 **Long** 값으로 반환 합니다.<br /><br /> **HelpFile** 는 도움말 파일의 완전히 확인 된 경로로 계산 되는 **문자열** 값을 반환 합니다.|  
|[Index](./index-property.md)|**레코드 집합** 개체에 현재 적용 되는 인덱스의 이름을 나타냅니다.|  
|[IsolationLevel](./isolationlevel-property.md)|**연결** 개체에 대 한 격리 수준을 나타냅니다.|  
|[항목](./item-property-ado.md)|이름 또는 서 수를 기준으로 컬렉션의 특정 멤버를 나타냅니다.|  
|[LineSeparator](./lineseparator-property-ado.md)|텍스트 **스트림** 개체에서 줄 구분 기호로 사용할 이진 문자를 나타냅니다.|  
|[LockType](./locktype-property-ado.md)|편집 하는 동안 레코드에 적용 되는 잠금 유형을 나타냅니다.|  
|[MarshalOptions](./marshaloptions-property-ado.md)|서버에 다시 마샬링할 레코드를 나타냅니다.|  
|[Records](./maxrecords-property-ado.md)|쿼리에서 **레코드 집합** 으로 반환할 최대 레코드 수를 나타냅니다.|  
|[모드](./mode-property-ado.md)|**연결**, **레코드**또는 **스트림** 개체의 데이터를 수정 하는 데 사용할 수 있는 권한을 나타냅니다.|  
|[이름](./name-property-ado.md)|개체의 이름을 나타냅니다.|  
|[NativeError](./nativeerror-property-ado.md)|특정 **오류** 개체에 대 한 공급자별 오류 코드를 나타냅니다.|  
|[Number](./number-property-ado.md)|**오류** 개체를 고유 하 게 식별 하는 번호를 나타냅니다.|  
|[NumericScale](./numericscale-property-ado.md)|**매개 변수** 또는 **필드** 개체에서 숫자 값의 소수 자릿수를 나타냅니다.|  
|[OriginalValue](./originalvalue-property-ado.md)|변경을 수행 하기 전에 레코드에 있던 **필드** 의 값을 나타냅니다.|  
|[PageCount](./pagecount-property-ado.md)|**레코드 집합** 개체가 포함 하는 데이터 페이지 수를 나타냅니다.|  
|[PageSize](./pagesize-property-ado.md)|레코드 **집합**에서 한 페이지를 나타내는 레코드 수를 나타냅니다.|  
|[ParentRow](./parentrow-property-ado.md)|**ADORecordConstruction** 개체에 대 한 OLE DB **row** 개체의 컨테이너를 설정 하 여 행의 부모를 ADO **Record** 개체로 설정 합니다.|  
|[ParentURL](./parenturl-property-ado.md)|현재 **record** 개체의 부모 **레코드** 를 가리키는 절대 URL 문자열을 나타냅니다.|  
|[위치](./position-property-ado.md)|**스트림** 개체 내의 현재 위치를 나타냅니다.|  
|[정밀도](./precision-property-ado.md)|**매개 변수** 개체 또는 숫자 **필드** 개체의 숫자 값에 대 한 전체 자릿수를 나타냅니다.|  
|[Prepared](./prepared-property-ado.md)|실행 전에 명령의 컴파일된 버전을 저장할지 여부를 나타냅니다.|  
|[공급 기업](./provider-property-ado.md)|**연결** 개체에 대 한 공급자의 이름을 나타냅니다.|  
|[RecordCount](./recordcount-property-ado.md)|레코드 **집합** 개체의 레코드 수를 나타냅니다.|  
|[RecordType](./recordtype-property-ado.md)|**Record** 개체의 유형을 나타냅니다.|  
|[행당](./row-property-ado.md)|**ADORecordConstruction** 개체에서/에 대 한 OLE DB **행** 개체를 가져오거나 설정 합니다.|  
|[행 위치](./rowposition-property-ado.md)|**ADORecordsetConstruction** 개체에서/에 대 한 OLE DB **rowposition** 개체를 가져오거나 설정 합니다.|  
|[행 집합](./rowset-property-ado.md)|**ADORecordsetConstruction** 개체에서/에 대 한 OLE DB **행 집합** 개체를 가져오거나 설정 합니다.|  
|[원본 (ADO 오류)](./source-property-ado-error.md)|원래 오류를 생성 한 개체 또는 응용 프로그램의 이름을 나타냅니다.|  
|[원본 (ADO 레코드)](./source-property-ado-record.md)|**Record** 개체가 나타내는 엔터티를 나타냅니다.|  
|[원본 (ADO 레코드 집합)](./source-property-ado-recordset.md)|**레코드 집합** 개체의 데이터 원본을 나타냅니다.|  
|[SQLState](./sqlstate-property.md)|특정 **오류** 개체에 대 한 SQL 상태를 나타냅니다.|  
|[State](./state-property-ado.md)|개체의 상태가 열리거나 닫혀 있는지 여부에 관계 없이 모든 개체를 나타냅니다. 비동기 메서드를 실행 하는 모든 개체 (개체의 현재 상태에 연결, 실행 또는 검색)가 있는지 여부를 나타냅니다.|  
|[상태 (ADO 필드)](./status-property-ado-field.md)|**필드** 개체의 상태를 나타냅니다.|  
|[상태 (ADO 레코드 집합)](./status-property-ado-recordset.md)|일괄 업데이트 또는 기타 대량 작업과 관련 된 현재 레코드의 상태를 나타냅니다.|  
|[StayInSync](./stayinsync-property.md)|부모 행 위치가 변경 될 때 계층 구조 **레코드 집합** 개체에서 기본 자식 레코드에 대 한 참조 (즉, *장*)가 변경 되는지 여부를 나타냅니다.|  
|[Stream 속성](./stream-property.md)|**ADOStreamConstruction** 개체의/에서 OLE DB **Stream** 개체를 가져오거나 설정 합니다.|  
|[형식](./type-property-ado.md)|**매개 변수**, **필드**또는 **속성** 개체의 작동 유형 또는 데이터 형식을 나타냅니다.|  
|[형식 (ADO 스트림)](./type-property-ado-stream.md)|**스트림에** 포함 된 데이터의 형식 (이진 또는 텍스트)을 나타냅니다.|  
|[UnderlyingValue](./underlyingvalue-property.md)|**필드** 개체에 대 한 데이터베이스의 현재 값을 나타냅니다.|  
|[값](./value-property-ado.md)|**필드**, **매개 변수**또는 **속성** 개체에 할당 된 값을 나타냅니다.|  
|[Version](./version-property-ado.md)|ADO 버전 번호를 나타냅니다.|
