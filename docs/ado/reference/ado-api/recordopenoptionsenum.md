|상수|값|Description|

|**adDelayFetchFields**|0x8000|필드와 연결 공급자에 게 나타내는 합니다 **레코드** 처음에 검색할 필요는 없지만 필드에 액세스 하는 첫 번째 시도에서 검색할 수 있습니다. 이 플래그가 없는 것으로 표시 된 기본 동작을 모두 검색 하는 것은 **레코드** 필드 개체입니다.|
