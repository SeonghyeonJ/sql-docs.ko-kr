# <a name="bookmarkenum"></a>BookmarkEnum
작업을 시작할 위치를 나타내는 책갈피를 지정 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**Ad책갈피 현재**|0|현재 레코드에서 시작 합니다.|  
|**먼저 Ad책갈피**|1|첫 번째 레코드에서 시작 합니다.|  
|**마지막으로 Ad책갈피**|2|마지막 레코드에서 시작 합니다.|
