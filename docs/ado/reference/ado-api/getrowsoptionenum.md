# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
레코드 [집합](../../../ado/reference/ado-api/recordset-object-ado.md)에서 검색할 레코드 수를 지정 합니다.  
  
|지속적임|값|Description|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|현재 위치나 [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) 메서드의 *Start* 매개 변수로 지정 된 책갈피에서 레코드 **집합**의 나머지 레코드를 검색 합니다.|
