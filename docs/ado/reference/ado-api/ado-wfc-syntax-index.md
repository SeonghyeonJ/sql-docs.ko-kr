# <a name="ado---wfc-syntax-index"></a>ADO - WFC 구문 인덱스
ADO 언어 참조에서는 Microsoft Visual Basic 프로그래밍 언어를 사용 하 여 ADO 메서드와 속성 구문을 보여 줍니다. 이 인덱스는 Windows Foundation 클래스에 대 한 ADO (ADO/WFC)를 기반으로 하는 ADO 언어 참조 항목에 대 한 상호 참조입니다. 구문의 차이가 발생 하는 경우 언어 참조 항목의 구문 목록과 달리이 인덱스의 함수 시그니처를 사용 합니다.
