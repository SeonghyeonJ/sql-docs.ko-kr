# <a name="ado-for-visual-c-syntax-index-for-com"></a>COM용 Visual C++ 구문 인덱스에 대한 ADO
이 인덱스는 Microsoft Visual C++ 기반으로 하는 ADO 언어 참조에 대 한 상호 참조입니다.  
  
 응용 프로그램에서 **#import** 지시어를 사용 하는 경우 Microsoft Visual Basic와 유사한 구문을 사용할 수 있도록 헤더 파일이 생성 됩니다. _Propertyname 및_ **put_**_propertyname_ **get_** 폼의 속성 이름은 단순히 *propertyname*으로 선언 된 것 처럼 처리 될 수 있습니다. 그런 다음 속성을 함수 대신 데이터 멤버 처럼 처리할 수 있습니다.  
  
 모든 메서드, 속성 및 이벤트는 **HRESULT**를 반환 하는 함수입니다 .이 함수를 테스트 하 여 함수가 성공적으로 실행 되었는지 여부를 확인할 수 있습니다.
