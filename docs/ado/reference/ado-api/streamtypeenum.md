# <a name="streamtypeenum"></a>StreamTypeEnum
[스트림](./stream-object-ado.md) 개체에 저장 된 데이터의 형식을 지정 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1|이진 데이터를 나타냅니다.|  
|**adTypeText**|2|기본값 문자 집합에 지정 된 문자 집합에 있는 텍스트 데이터를 [나타냅니다.](./charset-property-ado.md)|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 동급  
 이러한 상수에는 ADO/WFC 해당 항목이 없습니다.
