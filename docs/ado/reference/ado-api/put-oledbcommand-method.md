# <a name="put_oledbcommand-method"></a>put_OLEDBCommand 메서드
이 메서드는 작업을 수행 하지 않으며 항상 S_OK을 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
HRESULT put_OLEDBCommand(  
      IUnknown *pOLEDBCommand  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 *pOLEDBCommand*  
 진행 OLE DB Command 개체에 대 한 포인터입니다.
