# <a name="datafactory-object-rdsserver-properties-methods-and-events"></a>DataFactory 개체(RDSServer) 속성, 메서드 및 이벤트
> [!IMPORTANT]
>  Windows 8 및 Windows Server 2012 부터는 RDS 서버 구성 요소는 더 이상 포함 된 Windows 운영 체제에서 (Windows 8을 참조 하 고 [Windows Server 2012 호환성 설명서](https://www.microsoft.com/download/details.aspx?id=27416) 자세한). RDS 클라이언트 구성 요소는 Windows의 이후 버전에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. RDS를 사용 하는 응용 프로그램을 마이그레이션해야 [WCF 데이터 서비스](https://go.microsoft.com/fwlink/?LinkId=199565)합니다.

## <a name="methods"></a>메서드  
  
|||  
|-|-|  
|[ConvertToString 메서드(RDS)](../../../ado/reference/rds-api/converttostring-method-rds.md)|레코드 집합을 MIME64 문자열로 변환합니다.|  
|[CreateRecordset 메서드(RDS)](../../../ado/reference/rds-api/createrecordset-method-rds.md)|만들고 빈 레코드 집합을 반환 합니다.|  
|[Execute 메서드(RDS)](../../../ado/reference/rds-api/execute-method-rds.md)|요청을 실행 하 고 (예: ADO 2.5 이상을 사용)는 고급 데이터 행 집합을 만듭니다.|  
|[Execute21 메서드(RDS)](../../../ado/reference/rds-api/execute21-method-rds.md)|요청을 실행 하 고 (예: ADO 2.1 사용)는 고급 데이터 행 집합을 만듭니다.|  
|[Query 메서드(RDS)](../../../ado/reference/rds-api/query-method-rds.md)|요청을 실행 하 고는 고급 데이터 행 집합을 만듭니다.|  
|[SubmitChanges 메서드(RDS)](../../../ado/reference/rds-api/submitchanges-method-rds.md)|이 메서드는 보류 중인 변경 내용이 있는 레코드 집합이 들어 연결 문자열에서 식별 되는 데이터베이스로 전송 합니다.|  
|[Synchronize 메서드(RDS)](../../../ado/reference/rds-api/synchronize-method-rds.md)|(2.5 이상 ADO와 함께 사용) 연결 문자열에서 지정한 데이터베이스를 사용 하 여 지정 된 레코드 집합을 동기화 합니다.|  
|[Synchronize21 메서드(RDS)](../../../ado/reference/rds-api/synchronize21-method-rds.md)|(사용 하 여 ADO 2.1 사용 하 여) 연결 문자열에서 지정한 데이터베이스를 사용 하 여 지정 된 레코드 집합을 동기화 합니다.|
