# <a name="type-property-ado-md"></a>Type 속성(ADO MD)
현재 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md)의 형식을 나타냅니다.  
  
## <a name="return-values"></a>반환 값  
 는 [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md) 값을 반환 하 고 읽기 전용입니다.  
  
## <a name="remarks"></a>설명  
 이 속성은 [수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체에 속한 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체에 대해서만 지원 됩니다. [Position](../../../ado/reference/ado-md-api/position-object-ado-md.md) 개체에 속하는 **멤버** 개체에서이 속성을 참조 하는 경우 오류가 발생 합니다.
