# <a name="position-object-ado-md"></a>Position 개체(ADO MD)
축의 한 지점을 정의 하는 여러 차원의 멤버 집합을 나타냅니다.  
  
## <a name="remarks"></a>설명  
 **Position** 개체의 속성 및 컬렉션을 사용 하 여 다음을 수행할 수 있습니다.  
  
-   **Ordinal** 속성을 사용 하 여 [축을](../../../ado/reference/ado-md-api/axis-object-ado-md.md)따라 **위치의** 서 수 위치를 반환 합니다.  
  
-   [Members](../../../ado/reference/ado-md-api/members-collection-ado-md.md) 컬렉션을 사용 하 여 **축을**따라 위치를 구성 하는 멤버를 반환 합니다.  
  
 이 섹션에는 다음 항목이 포함 되어 있습니다.  
  
-   [속성, 메서드 및 이벤트](../../../ado/reference/ado-md-api/position-object-properties-methods-and-events.md)
