# <a name="membertypeenum"></a>MemberTypeEnum
[멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체의 [Type](../../../ado/reference/ado-md-api/type-property-ado-md.md) 속성에 대 한 설정을 지정 합니다.  
  
|지속적임|값|Description|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|**멤버** 개체가 수준의 모든 멤버를 나타내는지 여부를 나타냅니다.|  
|**adMemberFormula**|3|**멤버** 개체가 수식 식을 사용 하 여 계산 됨을 나타냅니다.|  
|**adMemberMeasure**|2|**멤버** 개체가 측정값 차원에 속하고 양적 특성을 나타내는 것을 나타냅니다.|  
|**adMemberRegular**|1|Default. **멤버** 개체가 비즈니스 엔터티의 인스턴스를 나타내는지 여부를 나타냅니다.|  
|**adMemberUnknown**|0|멤버의 형식을 확인할 수 없습니다.|
