# <a name="formattedvalue-property-ado-md"></a>FormattedValue 속성(ADO MD)
[셀](./cell-object-ado-md.md) 값의 형식이 지정 된 표시를 나타냅니다.  
  
## <a name="return-values"></a>반환 값  
 는 **문자열** 을 반환 하며 읽기 전용입니다.  
  
## <a name="remarks"></a>설명  
 **FormattedValue** 속성을 사용 하 여 [셀](./cell-object-ado-md.md) 개체의 [value](./value-property-ado-md.md) 속성에 대 한 형식이 지정 된 표시 값을 가져옵니다. 예를 들어 셀의 값이 1056.87이 고이 값이 달러 금액을 나타내는 경우 **FormattedValue** 는 $1056.87이 됩니다.  
  
## <a name="applies-to"></a>적용 대상
