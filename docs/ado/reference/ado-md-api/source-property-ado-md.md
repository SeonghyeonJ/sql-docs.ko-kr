# <a name="source-property-ado-md"></a>Source 속성(ADO MD)
[셀 집합](./cellset-object-ado-md.md)의 데이터 원본을 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 **Variant**를 설정 또는 반환 하 고, 닫힌 [셀 집합](./cellset-object-ado-md.md) 개체에 대 한 읽기/쓰기 및 열린 **셀 집합** 개체에 대 한 읽기 전용입니다. **Variant** 는 MDX 쿼리와 같은 유효한 **문자열**을 포함 해야 합니다.
