# <a name="parent-property-ado-md"></a>Parent 속성(ADO MD)
현재 부모 멤버를 나타내는 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 계층에서입니다.  
  
## <a name="return-values"></a>반환 값  
 반환 된 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체 및 읽기 전용입니다.  
  
## <a name="remarks"></a>설명  
 (루트) 계층의 최상위 수준에 있는 멤버에는 부모가 없습니다. 이 속성의 경우에 지원 됩니다 **멤버** 에 속하는 개체를 [수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체입니다. 이 속성에서 참조 되는 동안 오류가 발생 **멤버** 에 속하는 개체를 [위치](../../../ado/reference/ado-md-api/position-object-ado-md.md) 개체입니다.
