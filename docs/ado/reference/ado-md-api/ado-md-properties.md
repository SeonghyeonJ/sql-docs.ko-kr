|[서 수 (위치)](../../../ado/reference/ado-md-api/ordinal-property-ado-md-position.md)|축 따라 위치를 고유 하 게 식별합니다.|

|[ParentSameAsPrev](../../../ado/reference/ado-md-api/parentsameasprev-property-ado-md.md)|이 위치 멤버의 부모를 바로 앞에 있는 멤버의 부모로 같은지 여부를 나타냅니다.|  
|[원본](../../../ado/reference/ado-md-api/source-property-ado-md.md)|셀 집합에서 데이터 원본을 나타냅니다.|
