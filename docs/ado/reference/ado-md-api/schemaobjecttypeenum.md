# <a name="schemaobjecttypeenum"></a>SchemaObjectTypeEnum

|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adObjectTypeDimension**|1|[Dimension](./dimension-object-ado-md.md) 개체입니다.|  
|**adObjectTypeHierarchy**|2|[계층](./hierarchy-object-ado-md.md) 개체입니다.|  
|**adObjectTypeLevel**|3|[수준](./level-object-ado-md.md) 개체입니다.|  
|**adObjectTypeMember**|4|[멤버](./member-object-ado-md.md) 개체입니다.|
