# <a name="schemaobjecttypeenum"></a>SchemaObjectTypeEnum

|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adObjectTypeDimension**|1|[Dimension](../../../ado/reference/ado-md-api/dimension-object-ado-md.md) 개체입니다.|  
|**adObjectTypeHierarchy**|2|[계층](../../../ado/reference/ado-md-api/hierarchy-object-ado-md.md) 개체입니다.|  
|**adObjectTypeLevel**|3|[수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체입니다.|  
|**adObjectTypeMember**|4|[멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체입니다.|
