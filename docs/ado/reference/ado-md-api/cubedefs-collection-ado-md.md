# <a name="cubedefs-collection-ado-md"></a>CubeDefs 컬렉션(ADO MD)
다차원 카탈로그에서 큐브를 나타내는 [CubeDef](./cubedef-object-ado-md.md) 개체를 포함 합니다.  
  
## <a name="remarks"></a>설명  
 **CubeDefs** 는 표준 ADO 컬렉션입니다. 컬렉션의 속성과 메서드를 사용 하 여 다음을 수행할 수 있습니다.  
  
-   [Count](../ado-api/count-property-ado.md) 속성을 사용 하 여 컬렉션의 개체 수를 가져옵니다.  
  
-   기본 [항목](../ado-api/item-property-ado.md) 속성을 사용 하 여 컬렉션에서 개체를 반환 합니다.  
  
-   [Refresh](../ado-api/refresh-method-ado.md) 메서드를 사용 하 여 공급자에서 컬렉션의 개체를 업데이트 합니다.  
  
 이 섹션에는 다음 항목이 포함 되어 있습니다.  
  
-   [속성, 메서드 및 이벤트](./cubedefs-collection-properties-methods-and-events.md)
