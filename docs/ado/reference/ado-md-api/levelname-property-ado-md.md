# <a name="levelname-property-ado-md"></a>LevelName 속성(ADO MD)
[멤버](./member-object-ado-md.md)수준의 이름을 나타냅니다.  
  
## <a name="return-values"></a>반환 값  
 는 **문자열** 을 반환 하며 읽기 전용입니다.  
  
## <a name="remarks"></a>설명  
 **LevelName** 속성을 사용 하 여 멤버가 속한 수준의 이름을 검색 합니다. 이는 [수준](./level-object-ado-md.md) 개체의 [Name](./name-property-ado-md.md) 속성에 해당 합니다.
