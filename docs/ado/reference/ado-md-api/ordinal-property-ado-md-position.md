# <a name="ordinal-property-ado-md-position"></a>Ordinal 속성(ADO MD Position)
축을 따라 [위치](./position-object-ado-md.md) 를 고유 하 게 식별 합니다.  
  
## <a name="return-values"></a>반환 값  
 는 **Long** 정수를 반환 하며 읽기 전용입니다.  
  
## <a name="remarks"></a>설명  
 [Position](./position-object-ado-md.md) 개체의 **Ordinal** [속성은 위치 컬렉션 내](./positions-collection-ado-md.md) **위치의** 인덱스에 해당 합니다.  
  
 셀은 셀 [집합](./cellset-object-ado-md.md) 개체의 [Item](./item-property-ado-md-cellset.md) 속성을 사용 하 여 각 축의 **위치** **서 수** 를 사용 하 여 신속 하 게 검색할 수 있습니다.
