# <a name="actionenum"></a>ActionEnum
경우에 수행할 동작의 유형을 지정 [SetPermissions](../../../ado/reference/adox-api/setpermissions-method-adox.md) 라고 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adAccessDeny**|3|그룹 또는 사용자 지정 된 사용 권한을 거부 됩니다.|  
|**adAccessGrant**|1|그룹 또는 사용자 적어도 요청 된 권한을 갖습니다.|  
|**adAccessRevoke**|4|그룹 또는 사용자의 모든 명시적 액세스 권한이 해지 됩니다.|  
|**adAccessSet**|2|그룹 또는 사용자 요청된 된 권한을 정확 하 게 해야 합니다.|
