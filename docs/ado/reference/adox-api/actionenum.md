# <a name="actionenum"></a>ActionEnum
[SetPermissions](./setpermissions-method-adox.md) 가 호출 될 때 수행할 동작의 유형을 지정 합니다.  
  
|상수|값|설명|  
|--------------|-----------|-----------------|  
|**adAccessDeny**|3|그룹이 나 사용자에 게 지정 된 사용 권한이 거부 됩니다.|  
|**adAccessGrant**|1|그룹이 나 사용자에 게 적어도 요청 된 사용 권한이 있습니다.|  
|**adAccessRevoke**|4|그룹이 나 사용자에 대 한 모든 명시적 액세스 권한은 해지 됩니다.|  
|**adAccessSet**|2|그룹 또는 사용자에 게 요청 된 권한이 정확히 포함 됩니다.|
