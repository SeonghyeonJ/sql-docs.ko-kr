# <a name="sortorder-property-adox"></a>SortOrder 속성(ADOX)
열에 대 한 정렬 순서를 나타냅니다 (인덱스 열에만 해당).  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 [Sortorderenum](./sortorderenum.md) 상수 중 하나일 수 있는 **Long** 값을 설정 하 고 반환 합니다. 기본값은 **adSortAscending**입니다.  
  
## <a name="remarks"></a>설명  
 이 속성은 [인덱스](./index-object-adox.md)의 [Columns](./columns-collection-adox.md) 컬렉션에 있는 [열](./column-object-adox.md) 개체에만 적용 됩니다.  
  
## <a name="applies-to"></a>적용 대상  
 [열 개체(ADOX)](./column-object-adox.md)
