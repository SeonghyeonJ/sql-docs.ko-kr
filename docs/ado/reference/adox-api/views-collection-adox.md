## <a name="remarks"></a>설명  
 합니다 [추가](../../../ado/reference/adox-api/append-method-adox-views.md) 에 대 한 메서드는 **뷰** 컬렉션이 ADOX에 대 한 고유 합니다. 다음 작업을 수행할 수 있습니다.  
  
-   새 뷰를 사용 하 여 컬렉션에 추가 합니다 **Append** 메서드.
