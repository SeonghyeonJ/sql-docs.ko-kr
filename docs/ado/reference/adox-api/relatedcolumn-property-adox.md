# <a name="relatedcolumn-property-adox"></a>RelatedColumn 속성(ADOX)
관련 테이블에 있는 ADOX (관련 [열 개체](../../../ado/reference/adox-api/column-object-adox.md) )의 이름을 나타냅니다 (키 열에만 해당).  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 관련 테이블에 있는 관련 열의 이름인 **문자열** 값을 설정 하 고 반환 합니다.  
  
## <a name="remarks"></a>설명  
 기본값은 빈 문자열("")입니다.  
  
 컬렉션에 이미 추가 된 [열](../../../ado/reference/adox-api/column-object-adox.md) 개체의 경우이 속성은 읽기 전용입니다.
