# <a name="type-property-table-adox"></a>Type 속성(테이블)(ADOX)
테이블의 유형을 나타냅니다.  
  
## <a name="return-values"></a>반환 값  
 테이블의 유형을 지정 하는 **문자열** 값을 반환 합니다. 예를 들어 "테이블", "시스템 테이블" 또는 "전역 임시"입니다.  
  
## <a name="remarks"></a>설명  
 이 속성은 읽기 전용입니다.
