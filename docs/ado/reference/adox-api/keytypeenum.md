# <a name="keytypeenum"></a>KeyTypeEnum
유형을 지정 [키](../../../ado/reference/adox-api/key-object-adox.md): 기본, 외래, 또는 고유 합니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|기본. 키에는 기본 키가입니다.|  
|**adKeyForeign**|2|키를 외래 키가입니다.|  
|**adKeyUnique**|3|키는 고유 합니다.|
