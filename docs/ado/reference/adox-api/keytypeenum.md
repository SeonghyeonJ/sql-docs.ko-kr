# <a name="keytypeenum"></a>KeyTypeEnum
기본, 외부 또는 고유 [키](../../../ado/reference/adox-api/key-object-adox.md)의 유형을 지정 합니다.  
  
|지속적임|값|Description|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|Default. 키가 기본 키입니다.|  
|**adKeyForeign**|2|키가 외래 키 인 경우|  
|**adKeyUnique**|3|키가 고유 합니다.|
