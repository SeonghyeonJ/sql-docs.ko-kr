# <a name="unique-property-adox"></a>Unique 속성(ADOX)
인덱스 키가 고유 해야 하는지 여부를 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 **부울** 값을 설정 하 고 반환 합니다.  
  
## <a name="remarks"></a>설명  
 기본값은 **False**입니다.  
  
 이 속성은 이미 컬렉션에 추가 된 [인덱스](./index-object-adox.md) 개체에서 읽기 전용입니다.
