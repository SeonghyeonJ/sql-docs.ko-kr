# <a name="activeconnection-property-adox"></a>ActiveConnection 속성(ADOX)
[카탈로그가](./catalog-object-adox.md) 속한 ADO [연결](../ado-api/connection-object-ado.md) 개체를 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 **연결 개체 또는** 연결에 대 한 정의를 포함 하는 **문자열** 을 설정 합니다. 활성 **연결** 개체를 반환 합니다.  
  
## <a name="remarks"></a>설명  
 기본값은 null 개체 참조입니다.  
  
## <a name="applies-to"></a>적용 대상  
 [카탈로그 개체(ADOX)](./catalog-object-adox.md)
