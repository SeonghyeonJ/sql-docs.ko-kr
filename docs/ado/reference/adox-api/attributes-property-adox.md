## <a name="settings-and-return-values"></a>설정 및 반환 값  
 **Long** 값을 설정 하거나 반환 합니다. [열](../../../ado/reference/adox-api/column-object-adox.md) 개체가 나타내는 테이블의 특징을 지정 하는 값입니다. 값은 [ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md) 상수를 조합 하 여 사용할 수 있습니다. 기본값은 영 (**0**)으로, **adcolfixed** 또는 **adcolfixed**이 아닙니다.
