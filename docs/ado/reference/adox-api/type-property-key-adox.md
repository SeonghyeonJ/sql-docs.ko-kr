# <a name="type-property-key-adox"></a>Type 속성(키)(ADOX)
키의 유형을 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 [KeyTypeEnum](./keytypeenum.md) 상수 중 하나일 수 있는 **Long** 값을 설정 하거나 반환 합니다. 기본값은 **adKeyPrimary**입니다.  
  
## <a name="remarks"></a>설명  
 이 속성은 컬렉션에 이미 추가 된 [키](./key-object-adox.md) 개체에 대해 읽기 전용입니다.
