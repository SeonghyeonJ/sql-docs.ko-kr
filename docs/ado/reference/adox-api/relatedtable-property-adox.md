# <a name="relatedtable-property-adox"></a>RelatedTable 속성(ADOX)
관련 테이블의 이름을 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 **문자열** 값을 설정 하 고 반환 합니다.  
  
## <a name="remarks"></a>설명  
 기본값은 빈 문자열("")입니다.  
  
 키가 외래 키인 경우 **RelatedTable** 는 키가 포함 된 테이블의 이름입니다.
