# <a name="data-providers"></a>데이터 공급자
데이터 공급자는 다양 한 SQL 데이터베이스, 인덱싱된 순차 파일, 스프레드시트, 문서 저장소 및 메일 파일과 같은 데이터 원본을 나타냅니다. 공급자는 행 집합 이라고 하는 일반적인 추상화를 사용 하 여 균일 하 게 데이터를 표시 합니다.  
  
 여러 다른 데이터 공급자에 연결 하 고 여전히 모든 지정 된 공급자의 특정 기능에 관계 없이 동일한 프로그래밍 모델을 노출할 수 있으므로 ADO는 강력 하 고 유연 합니다. 그러나 각 데이터 공급자가 고유 하므로 데이터 공급자에 따라 달라 집니다 ADO를 사용 하 여 응용 프로그램 상호 작용 하는 방법.  
  
 기능 및 특징의 OLE DB Provider for SQL Server에 사용 되는 Microsoft SQL Server 데이터베이스에 액세스 하는 상당히 다른 Microsoft OLE DB Provider for Internet Publishing, 파일에 액세스 하는 데 사용 되는 예를 들어, 웹 서버에 저장 합니다.
