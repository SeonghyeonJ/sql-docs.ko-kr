# <a name="updating-joined-results-unique-table"></a>조인된 결과 업데이트: 고유한 테이블
ADO를 사용 하면 고유 테이블 동적 속성을 사용 하 여 여러 기본 테이블에 대 한 조인 작업으로 구성 된 레코드 집합의 특정 기본 테이블에 대 한 수정을 엄격 하 게 제어할 수 있습니다. 고유 테이블 사용에 대 한 자세한 내용은 Unique 테이블의 ADO 프로그래머 참조 항목 및 업데이트 Resynch 동적 속성을 참조 하세요.
