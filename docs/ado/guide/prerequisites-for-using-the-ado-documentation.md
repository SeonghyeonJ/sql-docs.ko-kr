# <a name="prerequisites-for-using-the-ado-documentation"></a>ADO 설명서 사용을 위한 필수 구성 요소
이 설명서를 사용 하려면 최소한 Microsoft Visual Basic를 사용 하 여 응용 프로그램을 개발 하는 경험이 있어야 합니다 .이 가이드의 대부분 예제는이 언어로 작성 되었기 때문입니다. 또한 VBScript (Microsoft Visual C++, Java, Visual Basic Scripting Edition) 및 Microsoft JScript로 작성 된 몇 가지 예제가 있습니다.  
  
 ADO는 다양 한 원본의 데이터에 액세스 하는 데 사용 되기 때문에 관계형 데이터베이스 관리 시스템 및 OLAP (온라인 분석 처리)에 대 한 이해가 필요 하 고 인터넷 및 인터넷 프로토콜에 대 한 기본 지식이 필요 합니다.  
  
 ADO와 관련 OLE DB 기술은 Microsoft COM (구성 요소 개체 모델)을 기반으로 하기 때문에 COM을 통해 ADO의 고급 개념을 더 잘 이해할 수 있습니다.
