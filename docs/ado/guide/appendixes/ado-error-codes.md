# <a name="capture-ado-error-codes"></a>ADO 오류 코드 캡처
[오류 컬렉션의](../../../ado/reference/ado-api/errors-collection-ado.md) [오류](../../../ado/reference/ado-api/error-object.md) 개체에서 반환 되는 공급자 오류 외에도 ADO 자체에서 런타임 환경의 예외 처리 메커니즘에 대 한 오류를 반환할 수 있습니다. Microsoft® Visual Basic의 **On error** 문과 같이 프로그래밍 언어의 오류 트래핑 메커니즘을 사용 하거나 Microsoft Visual C++®의 **try-catch 블록을 사용 하 여 ADO** 오류를 캡처합니다.

 ADO 오류 코드 목록은 [Errorvalueenum](../../../ado/reference/ado-api/errorvalueenum.md)을 참조 하십시오.
