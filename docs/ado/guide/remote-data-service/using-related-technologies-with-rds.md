# <a name="using-related-technologies-with-rds"></a>RDS에서 관련 기술 사용
이 섹션에서는 다음 항목을 다룹니다.  
  
-   [ODBC 연결 풀링에서 RDS 사용](../../../ado/guide/remote-data-service/using-rds-with-odbc-connection-pooling.md)  
  
-   [구성 요소 서비스에서 비즈니스 개체 실행](../../../ado/guide/remote-data-service/running-business-objects-in-component-services.md)  
  
> [!IMPORTANT]
>  Windows 8 및 Windows Server 2012 부터는 RDS 서버 구성 요소는 더 이상 포함 된 Windows 운영 체제에서 (Windows 8을 참조 하 고 [Windows Server 2012 호환성 설명서](https://www.microsoft.com/download/details.aspx?id=27416) 자세한). RDS 클라이언트 구성 요소는 Windows의 이후 버전에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. RDS를 사용 하는 응용 프로그램을 마이그레이션해야 [WCF 데이터 서비스](https://go.microsoft.com/fwlink/?LinkId=199565)합니다.  
  
 매우 다양 한 기술이 원격 데이터 서비스와 상호 작용 하거나 해당 구현에 사용 됩니다.  
  
 **인터넷 정보 서비스** Microsoft 인터넷 정보 서비스 (IIS)에 대 한 자세한 내용은 Microsoft 인터넷 정보 서비스 웹 사이트를 참조 하세요.  
  
 **Active Server Pages** 자세한 정보에 대 한 Microsoft ASP Active Server Pages (), Microsoft Active Server 페이지 웹 사이트를 참조 하세요.  
  
 **Microsoft 구성 요소 서비스** 구성 요소 서비스에 대 한 자세한 내용은 Microsoft Transaction Server 웹 사이트를 참조 하세요.  
  
 **Microsoft SQL Server** Microsoft SQL Server에 대 한 자세한 내용은 Microsoft SQL Server 웹 사이트를 참조 하세요.  
  
 **Microsoft Internet Explorer** Microsoft Internet Explorer에 대 한 자세한 내용은 Microsoft 인터넷 ExplorerWeb 사이트 및 MSDN Library 및 워크샵 웹 사이트를 참조 하세요.  
  
 **Microsoft Windows NT Server/Windows 2000 Server** Microsoft Windows NT Server 또는 Windows 2000 Server의 보안에 대 한 자세한 내용은 Microsoft Windows 웹 사이트를 참조 하세요.
