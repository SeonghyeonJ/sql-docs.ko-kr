# <a name="configurationsetting-property---machineaccountidentity"></a>ConfigurationSetting 속성 - MachineAccountIdentity
  보고서 서버가 설치되어 있는 컴퓨터의 컴퓨터 계정 ID를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>속성 값  
 보고서 서버가 설치되어 있는 컴퓨터의 컴퓨터 계정 ID가 들어 있는 **String** 값입니다.
