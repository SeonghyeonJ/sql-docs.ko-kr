# <a name="controlling-page-breaks-headings-columns-and-rows-report-builder-and-ssrs"></a>페이지 나누기, 머리글, 열 및 행 제어(보고서 작성기 및 SSRS)
  페이지 나누기를 사용하여 보고서를 개별 페이지로 나누어 보고 인쇄할 수 있습니다. 페이지 나누기는 보고서를 미리 볼 때나 다른 파일 형식으로 내보낼 때 최적의 상태로 표시하기 위해 내용을 보고서 페이지에 맞추는 방식을 결정합니다.  
  
 또한 페이지 나누기를 추가하면 큰 보고서를 처리할 때 성능이 향상됩니다. 렌더링된 페이지가 표시되는 동안 나머지 페이지는 백그라운드에서 렌더링됩니다. 이렇게 하면 다른 페이지를 렌더링하는 동안 사용자는 초기 페이지를 보면서 기다릴 수 있습니다.  
  
 페이지 나누기는 테이블, 행렬, 목록, 차트, 계기 또는 이미지와 같은 보고서 항목에 추가할 수 있습니다. 또한 테이블, 행렬 또는 목록의 그룹에도 페이지 나누기를 추가할 수 있습니다. 페이지 나누기는 그룹 전, 후, 사이에 추가할 수 있습니다. 그룹 간 페이지 나누기는 기본적으로 보고서에 추가되지 않습니다.
