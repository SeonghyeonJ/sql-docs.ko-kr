# <a name="manage-deployed-tabular-model-databases"></a>배포 된 테이블 형식 모델 데이터베이스 관리
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]
