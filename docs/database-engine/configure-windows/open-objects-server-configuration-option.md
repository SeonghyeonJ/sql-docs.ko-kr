# <a name="open-objects-server-configuration-option"></a>open objects 서버 구성 옵션
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  이 옵션은 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]에서 기능이 해제되었지만 **sp_configure**에는 여전히 유지됩니다. 있습니다(설정의 영향 없음). [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]에서 열린 데이터베이스 개체 수는 동적으로 관리되며 사용 가능한 메모리에 따라서만 제한됩니다. **open objects** 옵션은 기존 스크립트와의 호환성을 위해 **sp_configure**에서 사용할 수 있습니다.
