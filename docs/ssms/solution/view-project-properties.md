# <a name="view-project-properties"></a>프로젝트 속성 보기
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
프로젝트와 연관된 속성을 속성 창에서 볼 수 있습니다.  
  
### <a name="to-view-project-properties"></a>프로젝트 속성을 보려면  
  
1.  솔루션 탐색기에서 프로젝트를 선택합니다.  
  
2.  속성 창이 열리지 않으면 **보기** 메뉴에서 **속성 창**을 클릭합니다.  
  
3.  속성 창에서 프로젝트의 속성을 확인합니다.  
  
만든 날짜와 같은 프로젝트 속성은 읽기 전용이므로 속성 창에서 변경할 수 없습니다.
