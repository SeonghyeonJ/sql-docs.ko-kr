# <a name="open-database-diagram-designer-visual-database-tools"></a>데이터베이스 다이어그램 디자이너 열기(Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
데이터베이스 다이어그램 디자이너는 연결할 데이터베이스를 디자인하고 시각화할 수 있게 하는 비주얼 도구입니다. 데이터베이스를 디자인할 때 데이터베이스 디자이너를 사용하여 테이블, 열, 키, 인덱스, 관계, 제약 조건 등을 만들거나 편집하거나 삭제할 수 있습니다. 데이터베이스를 시각화하기 위해 테이블, 열, 키, 관계의 일부 또는 전부를 표시하는 다이어그램을 하나 이상 만들 수 있습니다.  
  
새 다이어그램이나 기존 다이어그램을 열어 데이터베이스 다이어그램 디자이너를 열 수 있습니다.  
  
### <a name="open-a-new-database-diagram"></a>새 데이터베이스 다이어그램 열기  
  
1.  개체 탐색기에서 데이터베이스 다이어그램 노드를 마우스 오른쪽 단추로 클릭합니다.  
  
2.  드롭다운 메뉴에서 **새 데이터베이스 다이어그램**을 클릭합니다.  
  
3.  **테이블 추가** 대화 상자에서 다이어그램에 사용할 테이블을 선택합니다.  
  
    > [!NOTE]  
    > 테이블을 추가하지 않고 **닫기** 를 선택하면 이전 상태로 되돌아가고 나중에 테이블을 추가할 수 있습니다.  
  
4.  **데이터베이스 다이어그램** 메뉴가 주 메뉴에 추가되고 디자이너 창이 열립니다.  
  
### <a name="open-an-existing-database-diagram"></a>기존 데이터베이스 다이어그램 열기  
  
1.  개체 탐색기에서 데이터베이스 다이어그램 노드 아래에 있는 다이어그램을 마우스 오른쪽 단추로 클릭합니다.  
  
2.  드롭다운 메뉴에서 **데이터베이스 다이어그램 디자인**을 클릭합니다.  
  
3.  **데이터베이스 다이어그램** 메뉴가 주 메뉴에 추가되고 디자이너 창에 다이어그램이 열립니다.
