# <a name="copy-query-results-visual-database-tools"></a>쿼리 결과 복사(Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
**결과** 창에서 탭으로 구분된 텍스트로 결과 집합을 복사할 수 있습니다. 전체 결과 집합, 연속된 모든 숫자 또는 연속되지 않은 행이나 셀을 복사할 수 있습니다.  
  
### <a name="to-copy-result-set-text-from-the-results-pane"></a>결과 창에서 결과 집합 텍스트를 복사하려면  
  
1.  복사할 데이터가 포함된 쿼리나 뷰를 실행합니다.  
  
2.  **결과** 창에서 전체 결과 집합을 선택하려면 왼쪽 위 빈 셀을 클릭합니다.  
  
3.  행을 선택하려면  
  
    -   왼쪽의 확인란을 클릭하여 하나의 행을 선택합니다.  
  
    -   하나의 행을 선택한 다음 마우스를 끌어 추가 행을 선택하여 여러 개의 행을 연속적으로 선택합니다.  
  
    -   하나의 행을 선택한 다음 Ctrl 키를 누른 채 추가 행을 클릭하여 여러 개의 행을 비연속적으로 선택합니다.  
  
    선택한 행은 강조 표시됩니다.  
  
4.  셀을 선택하려면  
  
    -   값을 클릭하여 단일 셀을 선택합니다.  
  
    -   인접한 셀로 마우스를 끌어 여러 개의 셀을 선택합니다.  
  
    -   단일 셀을 선택한 다음 Ctrl 키를 누른 채 추가 셀을 클릭하여 여러 개의 셀을 비연속적으로 선택합니다.  
  
    선택한 셀은 강조 표시됩니다.  
  
5.  마우스 오른쪽 단추를 클릭한 다음 **복사**를 선택합니다.
