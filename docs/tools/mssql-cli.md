# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>mssql-SQL Server에 대 한 cli 명령줄 쿼리 도구
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

mssql-cli는 SQL Server 쿼리, Windows, macOS 또는 Linux에 설치 하는 대화형 명령줄 도구입니다.

## <a name="install-mssql-cli"></a>Mssql-cli 설치

자세한 설치 지침은 [설치 가이드](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md)를 참조 하거나 pip를 알고 있는 경우 다음 명령을 실행 하 여 설치 합니다.

```$ pip install mssql-cli```
