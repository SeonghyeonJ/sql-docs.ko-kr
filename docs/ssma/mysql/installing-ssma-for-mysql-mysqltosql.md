# <a name="installing-ssma-for-mysql-mysqltosql"></a>MySQL용 SSMA 설치(MySqlToSql)
MySQL 용 SSMA (SQL Server Migration Assistant)는 MySQL에서 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 또는 SQL Azure로의 마이그레이션을 수행 하는 데 사용 하는 클라이언트 응용 프로그램으로 구성 됩니다. 또한 마이그레이션된 데이터베이스에서 데이터 마이그레이션 및 MySQL 시스템 기능 사용을 지 원하는 확장 팩이 포함 되어 있습니다.  
  
마이그레이션 단계를 수행 하는 컴퓨터에 클라이언트 응용 프로그램을 설치 합니다. 마이그레이션된 데이터베이스를 호스팅할 컴퓨터에 확장 팩 파일을 설치 해야 합니다.  해당 컴퓨터에서 SQL Server를 실행 해야 합니다.  
  
## <a name="upgrading-ssma-for-mysql"></a>MySQL 용 SSMA 업그레이드  
최신 버전의 MySQL 용 SSMA로 업그레이드 하려는 경우 먼저 클라이언트와 서버 확장 팩을 제거한 다음 최신 버전을 설치 해야 합니다.
