# <a name="installing-ssma-for-db2-db2tosql"></a>D b 2 용 SSMA 설치 (DB2ToSQL)
[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]D b 2 용 SSMA (Migration Assistant)는 DB2에서 및 Azure SQL DB로 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 의 마이그레이션을 수행 하는 데 사용 하는 클라이언트 응용 프로그램으로 구성 됩니다. 또한 마이그레이션된 데이터베이스에서 데이터 마이그레이션을 지 원하는 확장 팩과 DB2 시스템 함수를 사용할 수 있습니다.  
  
마이그레이션 단계를 수행 하는 컴퓨터에 클라이언트 응용 프로그램을 설치 합니다. 확장 팩은 클라이언트에서 배포 되며 별도의 설치를 요구 하지 않습니다.  
  
## <a name="upgrading-ssma-for-db2"></a>D b 2 용 SSMA 업그레이드  
최신 버전의 d b 2 용 SSMA로 업그레이드 하려면 먼저 클라이언트와 서버 확장 팩을 제거한 다음 최신 버전을 설치 해야 합니다.
