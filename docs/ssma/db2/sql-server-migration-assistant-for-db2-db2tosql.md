# <a name="sql-server-migration-assistant-for-db2-db2tosql"></a>DB2 용 SQL Server Migration Assistant (DB2ToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] D b 2 용 ssma (Migration Assistant)는 windows 및 linux의 2012 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2014 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , 2016, 2017, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] windows 및 linux의 2019 또는 Azure SQL Database으로 db2 데이터베이스를 마이그레이션하기 위한 도구입니다. D b 2 용 SSMA는 DB2 데이터베이스 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 개체를 데이터베이스 개체로 변환 하 고 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)],에서 해당 개체를 만든 후 DB2 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 에서 또는 Azure SQL Database로 데이터를 마이그레이션합니다.  
  
이 설명서에서는 d b 2 용 SSMA를 소개 하 고 DB2 데이터베이스를로 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]마이그레이션하는 방법에 대 한 단계별 지침을 제공 합니다. 다음 표에서는 자세한 정보를 확인할 수 있는 문서를 보여 줍니다.
