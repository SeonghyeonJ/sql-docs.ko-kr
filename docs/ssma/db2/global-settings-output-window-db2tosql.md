# <a name="global-settings-output-window-db2tosql"></a>전역 설정 (출력 창) (DB2ToSQL)
사용 된 **전역 설정** SSMA에 대 한 설정을 지정 하려면 대화 상자.  
  
출력 창 설정에 액세스 하는 **도구** 메뉴에서 **전역 설정**, 클릭 **GUI** 선택 고 왼쪽된 창 맨 아래에 **출력 창**합니다.  
  
## <a name="options"></a>변수  
**오류 목록 창 표시**  
선택 **True** 메시지가 있을 때 오류 목록 창을 표시 하려면 SSMA를 하려는 경우.  
  
**출력 창 표시**  
선택 **True** 메시지가 있을 때 출력 창을 표시 하려면 SSMA를 하려는 경우.
