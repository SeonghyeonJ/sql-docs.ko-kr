# <a name="o2ss0352-bulk-collect-into-clause-in-select-statement-not-converted-error"></a>O2SS0352: 변환 되지 않은 SELECT 문의 BULK COLLECT INTO 절 (오류)

이 문서에서는 `SELECT` `BULK COLLECT INTO` 절과 `DISTINCT` 절이 사용 되는 경우 Oracle 용 SSMA (SQL Server Migration Assistant)에서 PL/SQL 블록을 변환 하지 않는 이유에 대해 설명 합니다.

## <a name="background"></a>배경

컬렉션은 요소의 순서가 지정 된 그룹입니다. 목록, 배열 및 기타 친숙 한 데이터 형식을 포함 하는 일반적인 개념입니다. 각 요소에는 컬렉션에서의 위치를 결정 하는 고유한 첨자가 있습니다. 절 `DISTINCT` 은 결과 집합에 고유한 값만 나타날 수 있도록 지정 합니다. 절 `BULK COLLECT INTO` 은 쿼리 출력을 컬렉션에 바인딩하여 PL/SQL 및 sql 엔진 간의 통신을 감소 합니다.

SSMA는 Oracle 컬렉션에 대 한 마이그레이션 경로를 제공 합니다. 그러나 컬렉션을 채우는 `SELECT` 데 사용 되는 문에서 절을 `DISTINCT` 사용 하는 경우 ssma는 오류 메시지를 생성 합니다.

## <a name="example"></a>예제

Where 절과 `BULK COLLECT INTO` 절이 사용 `SELECT` 되는 다음 예를 살펴보겠습니다. `DISTINCT`

```sql
CREATE OR REPLACE PROCEDURE BULK_COLLECT_WITH_DISTINCT
AS
BEGIN
    DECLARE
        TYPE tbl_type_DNAME IS TABLE OF varchar2(14);
        var_dname tbl_type_DNAME;
    BEGIN
        SELECT DISTINCT DNAME
        BULK COLLECT INTO var_dname
        FROM dept;

        IF var_dname.COUNT = 0 THEN
            DBMS_OUTPUT.PUT_LINE('No results!');
        ELSE
            DBMS_OUTPUT.PUT_LINE('Results:');

            FOR i IN var_dname.FIRST..var_dname.LAST LOOP
                DBMS_OUTPUT.PUT_LINE(' DEPT NAME #' || var_dname(i));
            END LOOP;
        END IF;
    END;
END;
```

SSMA에서 위의 코드를 변환 하려고 하면 다음과 같은 오류 메시지가 생성 됩니다.

> O2SS0352: SELECT 문의 대량 COLLECT INTO 절이 변환 되지 않았습니다.

## <a name="possible-remedies"></a>가능한 해결 방법

위의 PL/SQL 블록을 변환 하려면 아래와 같이 절 `DISTINCT` 을 절 `UNIQUE` (의 `DISTINCT`동의어)으로 바꾸어야 합니다.

```sql
CREATE OR REPLACE PROCEDURE BULK_COLLECT_WITH_DISTINCT
AS
BEGIN
    DECLARE
        TYPE tbl_type_DNAME IS TABLE OF varchar2(14);
        var_dname tbl_type_DNAME;
    BEGIN
        SELECT UNIQUE DNAME
        BULK COLLECT INTO var_dname
        FROM dept;

        IF var_dname.COUNT = 0 THEN
            DBMS_OUTPUT.PUT_LINE('No results!');
        ELSE
            DBMS_OUTPUT.PUT_LINE('Results:');

            FOR i IN var_dname.FIRST..var_dname.LAST LOOP
                DBMS_OUTPUT.PUT_LINE(' DEPT NAME #' || var_dname(i));
            END LOOP;
        END IF;
    END;

END;
```
