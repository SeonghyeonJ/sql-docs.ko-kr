# <a name="project-settings-gui-oracletosql"></a>프로젝트 설정(GUI)(OracleToSQL)
GUI 프로젝트 설정에 나타나는 데이터를 구성할 수는 **데이터** 여부 변환 후 평가 보고서를 표시 하려면 여부와 탭 합니다.  
  
GUI 창에서 사용할 수는 **프로젝트 설정** 하 고 **기본 프로젝트 설정** 대화 상자.  
  
-   사용 된 **프로젝트 설정** 현재 프로젝트에 대 한 사용자 인터페이스 옵션을 설정 하려면 대화 상자. GUI 설정에 액세스할 수는 **도구** 메뉴에서 **프로젝트 설정**를 클릭 하 고 **GUI** 왼쪽 창의 맨 아래에서.  
  
-   사용 된 **기본 프로젝트 설정** 모든 프로젝트에 대 한 사용자 인터페이스 옵션을 설정 하려면 대화 상자. GUI 설정에 액세스 하는 **도구** 메뉴에서 **기본 프로젝트 설정**, 설정을 보거나에서 변경 하는 데 필요한는 마이그레이션 프로젝트 형식을 선택 **마이그레이션 버전 대상 지정** 을 드롭다운 하 고 클릭 **GUI** 왼쪽 창의 맨 아래에 있습니다.  
  
## <a name="options"></a>변수  
**원본에 대 한 최대 행 수**  
에 표시 되는 데이터의 행 수를 구성 합니다 **데이터** 선택한 원본 테이블에 대 한 탭 합니다.  
  
**기본값**: 100  
  
**대상에 대 한 최대 행 수**  
에 표시 되는 데이터의 행 수를 구성 합니다 **데이터** 선택한 대상 테이블에 대 한 탭 합니다.  
  
**기본값**: 100  
  
**변환 후 보고서를 보여 줍니다.**  
스키마를 변환한 후 보고서를 표시할 True를 선택 합니다. 결과 변환 보고서를 동일한 레이아웃 및 평가 보고서와 정보를 포함합니다.  
  
**기본값**: False
