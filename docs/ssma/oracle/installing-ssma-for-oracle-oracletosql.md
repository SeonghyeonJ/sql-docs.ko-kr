# <a name="installing-ssma--for-oracle-oracletosql"></a>Oracle용 SSMA 설치(OracleToSQL)
[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]Oracle 용 SSMA (Migration Assistant)는 Oracle에서 Azure SQL DB로 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 마이그레이션을 수행 하는 데 사용 하는 클라이언트 응용 프로그램으로 구성 됩니다. 또한 마이그레이션된 데이터베이스에서 데이터 마이그레이션 및 Oracle 시스템 기능 사용을 지 원하는 확장 팩이 포함 되어 있습니다.  
  
마이그레이션 단계를 수행 하는 컴퓨터에 클라이언트 응용 프로그램을 설치 합니다. 마이그레이션된 데이터베이스를 호스팅할 컴퓨터에 확장 팩 파일을 설치 해야 합니다. 해당 컴퓨터는를 실행 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]해야 합니다.  
  
## <a name="upgrading-ssma-for-oracle"></a>Oracle 용 SSMA 업그레이드  
최신 버전의 Oracle 용 SSMA로 업그레이드 하려면 먼저 클라이언트와 서버 확장 팩을 제거한 다음 최신 버전을 설치 해야 합니다.
