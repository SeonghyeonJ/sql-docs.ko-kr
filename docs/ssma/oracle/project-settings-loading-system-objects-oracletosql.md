-   모든 SSMA 프로젝트에 대 한 설정을 지정 하는 **도구** 메뉴에서 **기본 프로젝트 설정**, 설정을 보거나 에서변경하는데필요한는마이그레이션프로젝트형식을선택**마이그레이션 대상 버전** 클릭 드롭다운 **일반** 클릭 한 다음 확인 하 고 왼쪽된 창 맨 아래에 **시스템 개체 로드**합니다.
