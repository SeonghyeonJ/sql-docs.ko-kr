# <a name="viewing-test-case-reports-oracletosql"></a>테스트 사례 보고서 보기(OracleToSQL)
테스트 사례 보고서에는 테스트 확인 결과 정보와 일반 테스트 보여 줍니다. 테스트 실패 시 확인 된 개체에 일치 하지 않는 데이터에 대 한 내용은 표시 됩니다.  
  
## <a name="report-structure"></a>보고서 구조  
보고서의 맨 위에 이러한 통계를 보여 줍니다.  
  
-   테스트 개체 및 테스트에 성공 하는 개체 수의 총 수입니다.  
  
-   확인 된 테이블과 외래 키의 총 수와 테이블 및 성공적으로 일치 하는 외래 키의 수입니다.  
  
-   시작 시간, 종료 시간 테스트 사례 및 실행에 걸린 총 시간입니다.  
  
보고서의 나머지 부분에는 네 가지에서 정보가 표시 됩니다.  
  
**필수 구성 요소 오류**  
발생 한 모든 오류를 보여 줍니다는 **필수 조건 단계입니다.** 일반적으로 생략 됩니다.  
  
**초기화**  
실행의 상태를 보여 줍니다 **성공** 하거나 **오류**합니다.  
  
**테스트 개체 결과**  
결과 (성공 또는 실패) 및 SSMA 테스터 오류 발생 시 발견 된 불일치 항목을 비교 합니다.  
  
**종료**  
실행의 상태를 보여 줍니다 **성공** 하거나 **오류**합니다.
