# <a name="global-settings-output-window-accesstosql"></a>전역 설정 (출력 창) (AccessToSQL)
**전역 설정** 대화 상자를 사용 하 여 ssma에 대 한 설정을 지정할 수 있습니다.  
  
출력 창 설정에 액세스 하려면 **도구** 메뉴에서 **전역 설정**을 선택 하 고 왼쪽 창의 맨 아래에 있는 **GUI** 를 클릭 한 다음 **출력 창**을 선택 합니다.  
  
## <a name="options"></a>옵션  
**오류 목록 창 표시**  
메시지에 오류 목록 창이 표시 되도록 하려면 **True** 를 선택 합니다.  
  
**출력 창 표시**  
메시지가 있을 때 SSMA에서 출력 창을 표시 하도록 하려면 **True** 를 선택 합니다.
