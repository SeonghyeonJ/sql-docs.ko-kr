# <a name="global-settings-editor-accesstosql"></a>전역 설정 (편집기) (AccessToSQL)
**전역 설정** 대화 상자의 편집기 페이지를 사용 하 여 코드 편집기 창에 대 한 설정을 지정할 수 있습니다.  
  
편집기 설정에 액세스 하려면 **도구** 메뉴에서 **전역 설정**을 선택 하 고 왼쪽 창의 맨 아래에 있는 **GUI** 를 클릭 한 다음 **편집기**를 선택 합니다.  
  
## <a name="options"></a>옵션  
**편집기에서 줄 번호 표시**  
코드 편집기 창에 줄 번호를 표시 하려면 **True**를 선택 합니다.  
  
이 설정을 변경할 때 코드 편집기 창을 선택 하면 현재 개체를 다시 선택 하 여 디스플레이를 업데이트 해야 할 수 있습니다.
