# <a name="convert-load-and-migrate-accesstosql"></a>변환, 로드 및 마이그레이션 (AccessToSQL)

변환, **로드 및 마이그레이션 대화 상자** 는 변환, 로드 및 마이그레이션 단추 또는 마이그레이션 마법사를 사용 하는 경우 마이그레이션 상태를 보고 합니다.  
  
마이그레이션이 완료 되 면 각 섹션의 링크를 선택 하 여 정보, 경고 및 오류 메시지를 볼 수 있습니다. 각 링크 뒤의 세부 정보에는 사용 가능한 오류, 경고 및 정보 메시지 (예: "0 오류, 4 개 경고")가 표시 됩니다.  
  
페이지의 **선택한 개체에 대 한 데이터 마이그레이션** 섹션에도 **보고서** 단추가 포함 됩니다. 이 단추를 선택 하 여 데이터 마이그레이션 보고서를 볼 수 있습니다.  
  
## <a name="options"></a>옵션

**Report**  
연결 된 변환 또는 마이그레이션 보고서를 표시 합니다.
