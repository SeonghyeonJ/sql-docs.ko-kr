# <a name="find-databases-wizard-select-files-accesstosql"></a>데이터베이스 찾기 마법사 (파일 선택) (AccessToSQL)
데이터베이스 찾기 마법사의 파일 선택 페이지에는 검색 중에 발견 된 파일이 나열 됩니다. SSMA에 추가할 파일을 선택 하려면 파일 이름을 포함 하는 행을 선택 합니다. 파일 선택을 완료 하면 **다음** 을 클릭 하 여 선택 확인 페이지를 표시 합니다.  
  
파일에 대 한 네트워크 검색 마법사를 열려면 **파일** 메뉴에서 **데이터베이스 찾기**를 클릭 합니다.  
  
SSMA에 추가할 파일을 선택 하려면 파일 이름 옆의 확인란을 선택 합니다. 파일 선택을 완료 하면 **다음** 을 클릭 하 여 완료 페이지를 확인 합니다.  
  
## <a name="options"></a>옵션  
**모두 선택**  
목록에서 모든 데이터베이스를 선택 하려면이 확인란을 선택 합니다.  
  
**모두 선택 취소**  
목록에서 모든 데이터베이스를 선택 취소 하려면이 확인란을 선택 합니다.  
  
**파일 이름**  
SSMA에 추가할 Access 데이터베이스 파일의 이름을 표시 합니다.  
  
**소유자**  
파일의 소유자를 표시 합니다.  
  
**파일 경로**  
Access 데이터베이스 파일의 경로를 표시 합니다.  
  
**크기**  
데이터베이스의 크기 (바이트)입니다.  
  
**만든 날짜**  
데이터베이스를 만든 날짜와 시간입니다.  
  
**수정한 날짜**  
데이터베이스를 마지막으로 업데이트 한 날짜와 시간입니다.
