# <a name="initializing-test-cases-sybasetosql"></a>테스트 사례 초기화(SybaseToSQL)
이 단계에서 새 테스트 사례에 대 한 초기 정보를 지정할 수 있습니다.  
  
## <a name="parameters"></a>매개 변수  
**테스트 사례 이름**  
테스트 사례를 식별 하는 이름을 입력 합니다.  
  
**만든 날짜**  
오늘날의 현재 날짜를 자동으로 정의 합니다.  
  
**마지막으로 수정한 날짜**  
자동으로 입력 변경할 수 없습니다.  
  
**테스트 사례 설명**  
테스트 사례의 용도 파악 하려면 추가 정보를 입력 합니다.
