# <a name="customizing-calls-order-sybasetosql"></a>호출 순서 사용자 지정(SybaseToSQL)
이 단계에서는 테스트할 개체가 호출 되는 순서를 선택 합니다.  
  
## <a name="customizing-order"></a>순서 사용자 지정  
개체 목록을 사용 하 여 표 맨 위에 있는 단추를 사용 하 여 개체를 올바른 순서로 배치할 수 있습니다. 또는 행이 선택 될 때 오른쪽에 표시 되는 단추를 사용할 수 있습니다.
