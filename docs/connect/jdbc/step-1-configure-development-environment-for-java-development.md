* 해당하는 JDBC 드라이버를 다운로드하여 설치할 수 있는 곳:  [SQL Server용 Microsoft JDBC 드라이버 다운로드](../../connect/jdbc/download-microsoft-jdbc-driver-for-sql-server.md)
