# <a name="getbigdecimal-method-sqlserverresultset"></a>getBigDecimal 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 java.math.BigDecimal로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|설명|  
|----------|-----------------|  
|[getBigDecimal 메서드&#40;int&#41;&#40;SQLServerResultSet&#41;](../../../connect/jdbc/reference/getbigdecimal-method-int-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 검색합니다.|  
|[getBigDecimal 메서드&#40;int, int&#41;&#40;SQLServerResultSet&#41;](../../../connect/jdbc/reference/getbigdecimal-method-int-int-sqlserverresultset.md)|(사용되지 않음) 지정된 소수 자릿수를 사용하여 이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 검색합니다.|  
|[getBigDecimal 메서드&#40;java.lang.String&#41;&#40;SQLServerResultSet&#41;](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 검색합니다.|  
|[getBigDecimal 메서드&#40;java.lang.String, int&#41;&#40;SQLServerResultSet&#41;](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string-int-sqlserverresultset.md)|(사용되지 않음) 지정된 소수 자릿수를 사용하여 이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 검색합니다.|
