# <a name="getbigdecimal-method-sqlserverresultset"></a>getBigDecimal 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 java.math.BigDecimal로 검색합니다.

|[getSQLServerResultSet decimal 메서드 &#40;int, int&#41; &#40;&#41;](../../../connect/jdbc/reference/getbigdecimal-method-int-int-sqlserverresultset.md)|(사용되지 않음) 지정된 소수 자릿수를 사용하여 이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 검색합니다.|
