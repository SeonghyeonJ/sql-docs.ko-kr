# <a name="getasciistream-method-sqlserverresultset"></a>getAsciiStream 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열의 값을 ASCII 문자의 스트림으로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[getAsciiStream(int)](../../../connect/jdbc/reference/getasciistream-method-int.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 ASCII 문자의 스트림으로 검색합니다.|  
|[getAsciiStream(java.lang.String)](../../../connect/jdbc/reference/getasciistream-method-java-lang-string.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 ASCII 문자의 스트림으로 검색합니다.|
