# <a name="clearwarnings-method-sqlserverresultset"></a>clearWarnings 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체에 대해 보고된 모든 경고를 지웁니다.  
  
> [!NOTE]  
>  [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]에서는 현재 이 메서드가 구현되지 않습니다. 이 메서드를 호출하면 항상 null이 반환됩니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 clearWarnings 메서드는 java.sql.ResultSet 인터페이스의 clearWarnings 메서드에 의해 지정됩니다.
