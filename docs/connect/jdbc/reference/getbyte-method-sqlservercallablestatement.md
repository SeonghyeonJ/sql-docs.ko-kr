# <a name="getbyte-method-sqlservercallablestatement"></a>getByte 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수의 값을 검색하여 **byte** 값으로 반환합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[getByte(int)](../../../connect/jdbc/reference/getbyte-method-int.md)|매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 **byte** 값으로 검색합니다.|  
|[getByte(java.lang.String)](../../../connect/jdbc/reference/getbyte-method-java-lang-string.md)|매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 **byte** 값으로 검색합니다.|
