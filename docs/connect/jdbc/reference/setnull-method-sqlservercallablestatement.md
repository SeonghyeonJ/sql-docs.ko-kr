# <a name="setnull-method-sqlservercallablestatement"></a>setNull 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  설정할 매개 변수 형식이 지정된 경우 지정된 매개 변수를 null 값으로 설정합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[setNull(java.lang.String, int)](../../../connect/jdbc/reference/setnull-method-java-lang-string-int.md)|설정할 매개 변수 형식이 지정된 경우 지정된 매개 변수를 null 값으로 설정합니다.|  
|[setNull(java.lang.String, int, java.lang.String)](../../../connect/jdbc/reference/setnull-method-java-lang-string-int-java-lang-string.md)|설정할 매개 변수의 형식 및 이름이 지정된 경우 지정된 매개 변수를 null 값으로 설정합니다.|
