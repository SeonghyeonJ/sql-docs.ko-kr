# <a name="getsavepointid-method-sqlserversavepoint"></a>getSavepointId 메서드(SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  저장점의 ID를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getSavepointId()  
```  
  
## <a name="return-value"></a>Return Value  
 **int** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 getSavepointId 메서드는 java.sql.Savepoint 인터페이스의 getSavepointId 메서드에 의해 지정됩니다.
