# <a name="setlogwriter-method-sqlserverdatasource"></a>setLogWriter 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 메서드는 내부 전용입니다. 로깅에 대한 자세한 내용은 [드라이버 작업 추적](../../../connect/jdbc/tracing-driver-operation.md)을 참조하세요.  
  
## <a name="syntax"></a>구문  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *out*  
  
 PrintWriter 개체입니다.  
  
## <a name="remarks"></a>설명  
 이 setLogWriter 메서드는 javax.sql.DataSource 인터페이스의 setLogWriter 메서드에 의해 지정됩니다.
