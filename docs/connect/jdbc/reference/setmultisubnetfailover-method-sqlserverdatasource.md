# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>setMultiSubnetFailover 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  **MultiSubnetFailover** connection 속성의 값을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>매개 변수  
 *multiSubnetFailover*  
  
 **MultiSubnetFailover** connection 속성의 새 값입니다.  
  
## <a name="remarks"></a>Remarks  
 **MultiSubnetFailover** connection 속성에 대 한 자세한 내용은 [연결 속성 설정](../../../connect/jdbc/setting-the-connection-properties.md)을 참조 하세요.
