# <a name="getapplicationintent-method-sqlserverdatasource"></a>getApplicationIntent 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  **applicationIntent** 연결 속성의 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>Return Value  
 **applicationIntent** 연결 속성의 값을 반환합니다.  
  
## <a name="remarks"></a>설명  
 **applicationIntent** 연결 속성에 대한 자세한 내용은 [연결 속성 설정](../../../connect/jdbc/setting-the-connection-properties.md)을 참조하세요.
