# <a name="setapplicationintent-method-sqlserverdatasource"></a>setApplicationIntent 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  **applicationIntent** 연결 속성의 값을 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>매개 변수  
 *applicationIntent*  
  
 **applicationIntent** 연결 속성의 새 값입니다.  
  
## <a name="remarks"></a>설명  
 **applicationIntent** 연결 속성에 대한 자세한 내용은 [연결 속성 설정](../../../connect/jdbc/setting-the-connection-properties.md)을 참조하세요.
