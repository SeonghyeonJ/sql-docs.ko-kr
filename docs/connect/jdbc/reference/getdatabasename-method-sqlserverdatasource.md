# <a name="getdatabasename-method-sqlserverdatasource"></a>getDatabaseName 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  데이터베이스 이름을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.lang.String getDatabaseName()  
```  
  
## <a name="return-value"></a>Return Value  
 데이터베이스 이름이 들어 있는 **String**이며, 값이 설정되어 있지 않은 경우 null입니다.
