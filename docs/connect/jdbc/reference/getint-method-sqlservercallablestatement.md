# <a name="getint-method-sqlservercallablestatement"></a>getInt 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수의 값을 Java 프로그래밍 언어의 **int**로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|설명|  
|----------|-----------------|  
|[getInt(int)](../../../connect/jdbc/reference/getint-method-int.md)|매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 Java 프로그래밍 언어의 **int**로 검색합니다.|  
|[getInt(java.lang.String)](../../../connect/jdbc/reference/getint-method-java-lang-string.md)|매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 Java 프로그래밍 언어의 **int**로 검색합니다.|
