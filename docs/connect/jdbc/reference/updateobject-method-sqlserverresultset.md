# <a name="updateobject-method-sqlserverresultset"></a>updateObject 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 열을 **Object** 값으로 업데이트합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[updateObject(int, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object.md)|열 인덱스가 지정된 경우 지정된 열을 **Object** 값으로 업데이트합니다.|  
|[updateObject(int, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object-int.md)|열 인덱스와 소수 자릿수가 지정된 경우 지정된 열을 **Object** 값으로 업데이트합니다.|  
|[updateObject(java.lang.String, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object.md)|열 이름이 지정된 경우 지정된 열을 **Object** 값으로 업데이트합니다.|
