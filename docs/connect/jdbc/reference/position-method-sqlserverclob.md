# <a name="position-method-sqlserverclob"></a>position 메서드(SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 시작 위치에 따라 지정된 CLOB 개체 또는 해당 CLOB에 있는 부분 문자열의 문자 위치를 반환합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[position(java.sql.Clob, long)](../../../connect/jdbc/reference/position-method-java-sql-clob-long.md)|지정된 시작 위치에 따라 CLOB에서 지정된 CLOB 개체가 있는 문자 위치를 반환합니다.|  
|[position(java.lang.String, long)](../../../connect/jdbc/reference/position-method-java-lang-string-long.md)|지정된 시작 위치에 따라 CLOB에서 지정된 부분 문자열이 있는 문자 위치를 반환합니다.|
