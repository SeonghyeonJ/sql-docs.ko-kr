# <a name="getobject-method-sqlservercallablestatement"></a>getObject 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수의 값을 검색하여 Java 프로그래밍 언어의 개체로 반환합니다.

|속성|설명|  
|----------|-----------------|  
|[getObject(int)](../../../connect/jdbc/reference/getobject-method-int.md)|매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 검색하여 Java 프로그래밍 언어의 개체로 반환합니다.|  
|[getObject(int, java.util.Map)](../../../connect/jdbc/reference/getobject-method-int-java-util-map.md)|매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 지정된 Map 개체를 사용하여 Java 프로그래밍 언어의 개체로 검색합니다.|  
|[getObject(java.lang.String)](../../../connect/jdbc/reference/getobject-method-java-lang-string.md)|매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 검색하여 Java 프로그래밍 언어의 개체로 반환합니다.|  
|[getObject(java.lang.String, java.util.Map)](../../../connect/jdbc/reference/getobject-method-java-lang-string-java-util-map.md)|매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 지정된 Map 개체를 사용하여 Java 프로그래밍 언어의 개체로 검색합니다.|
