# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 메서드는 내부 전용입니다. 로깅에 대한 자세한 내용은 [드라이버 작업 추적](../../../connect/jdbc/tracing-driver-operation.md)을 참조하세요.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>Return Value  
 PrintWriter 개체입니다.  
  
## <a name="remarks"></a>설명  
 이 getLogWriter 메서드는 javax.sql.DataSource 인터페이스의 getLogWriter 메서드에 의해 지정됩니다.
