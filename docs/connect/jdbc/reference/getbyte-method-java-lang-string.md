# <a name="getbyte-method-javalangstring"></a>getByte 메서드(java.lang.String)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 **byte** 값으로 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public byte getByte(java.lang.String sCol)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *sCol*  
  
 매개 변수 이름이 들어 있는 **문자열**입니다.  
  
## <a name="return-value"></a>반환 값  
 **byte** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 이 getByte 메서드는 java.sql.CallableStatement 인터페이스의 getByte 메서드에 의해 지정됩니다.
