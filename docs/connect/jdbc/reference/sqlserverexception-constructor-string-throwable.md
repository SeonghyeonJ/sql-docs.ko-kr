# <a name="sqlserverexception-constructor-javalangstring-javalangthrowable"></a>SQLServerException 생성자 (java.lang.String, java.lang.Throwable)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  새 인스턴스를 초기화 합니다 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) 클래스를 **문자열** 개체와 **throw 할 수 있는** 개체.

## <a name="syntax"></a>구문  
  
```  
public SQLServerException(java.lang.String errText,
            java.lang.Throwable cause)
            
```  
  
#### <a name="parameters"></a>매개 변수  
 *errText*  
  
 오류 텍스트를 포함 하는 문자열입니다.
 
 *cause*  
  
 예외의 원인을 포함 하는 throw 할 수 있는 개체입니다.
