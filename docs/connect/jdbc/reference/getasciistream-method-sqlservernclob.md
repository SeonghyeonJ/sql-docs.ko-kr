# <a name="getasciistream-method-sqlservernclob"></a>getAsciiStream 메서드(SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 **nclob** 개체에서 ASCII 스트림으로 지정한 **nclob** 값을 검색 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.sql.InputStream getAsciiStream()  
```  
  
## <a name="return-value"></a>반환 값  
 NCLOB 데이터를 포함 하는 InputStream 개체입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 이 getAsciiStream 메서드는 SQLServerNClob 인터페이스의 getAsciiStream 메서드에 의해 지정 됩니다.
