# <a name="getasciistream-method-sqlservernclob"></a>getAsciiStream 메서드(SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 **NClob** 개체에 의해 ASCII 스트림으로 지정된 **NCLOB** 값을 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.sql.InputStream getAsciiStream()  
```  
  
## <a name="return-value"></a>Return Value  
 NCLOB 데이터를 포함하는 InputStream 개체입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 getAsciiStream 메서드는 java.sql.SQLServerNClob 인터페이스의 getAsciiStream 메서드에 의해 지정됩니다.
