# <a name="supportsbatchupdates-method-sqlserverdatabasemetadata"></a>supportsBatchUpdates 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스에서 일괄 처리 업데이트를 지원하는지 여부를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean supportsBatchUpdates()  
```  
  
## <a name="return-value"></a>Return Value  
 지원되는 경우 **true**입니다. 그렇지 않으면 **false**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 supportsBatchUpdates 메서드는 java.sql.DatabaseMetaData 인터페이스의 supportsBatchUpdates 메서드에 의해 지정됩니다.
