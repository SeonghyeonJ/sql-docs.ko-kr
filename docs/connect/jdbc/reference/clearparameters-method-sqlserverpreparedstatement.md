# <a name="clearparameters-method-sqlserverpreparedstatement"></a>clearParameters 메서드(SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  현재 매개 변수 값을 즉시 지웁니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public final void clearParameters()  
```  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 이 clearParameters 메서드는 java.sql.PreparedStatement 인터페이스의 clearParameters 메서드에 의해 지정 됩니다.
