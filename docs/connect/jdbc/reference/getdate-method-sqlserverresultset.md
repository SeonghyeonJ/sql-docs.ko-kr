# <a name="getdate-method-sqlserverresultset"></a>getDate 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열의 값을 Java 프로그래밍 언어의 java.sql.Date 개체로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|Description|  
|----------|-----------------|  
|[getDate 메서드(int)](../../../connect/jdbc/reference/getdate-method-int-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 Java 프로그래밍 언어의 java.sql.Date 개체로 검색합니다.|  
|[getDate(int, java.util.Calendar)](../../../connect/jdbc/reference/getdate-method-int-java-util-calendar-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 지정된 Calendar 개체를 사용하여 Java 프로그래밍 언어의 java.sql.Date 개체로 검색합니다.|  
|[getDate(java.lang.String)](../../../connect/jdbc/reference/getdate-method-java-lang-string-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 Java 프로그래밍 언어의 java.sql.Date 개체로 검색합니다.|  
|[getDate(java.lang.String, java.util.Calendar)](../../../connect/jdbc/reference/getdate-method-java-lang-string-java-util-calendar-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 지정된 Calendar 개체를 사용하여 Java 프로그래밍 언어의 java.sql.Date 개체로 검색합니다.|
