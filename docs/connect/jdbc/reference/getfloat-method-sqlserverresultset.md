# <a name="getfloat-method-sqlserverresultset"></a>getFloat 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열의 값을 Java 프로그래밍 언어의 **float**로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|설명|  
|----------|-----------------|  
|[getFloat(int)](../../../connect/jdbc/reference/getfloat-method-int-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 Java 프로그래밍 언어의 **float**로 검색합니다.|  
|[getFloat(java.lang.String)](../../../connect/jdbc/reference/getfloat-method-java-lang-string-sqlserverresultset.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 Java 프로그래밍 언어의 **float**로 검색합니다.|
