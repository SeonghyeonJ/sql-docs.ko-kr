# <a name="getminorversion-method-sqlserverdriver"></a>getMinorVersion 메서드(SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]의 부 버전 번호를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getMinorVersion()  
```  
  
## <a name="return-value"></a>Return Value  
 JDBC 드라이버의 부 버전 번호가 들어 있는 **int**입니다.  
  
## <a name="remarks"></a>설명  
 이 getMinorVersion 메서드는 java.sql.Driver 인터페이스의 getMinorVersion 메서드에 의해 지정됩니다.
