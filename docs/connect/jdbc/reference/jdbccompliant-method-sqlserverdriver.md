# <a name="jdbccompliant-method-sqlserverdriver"></a>jdbcCompliant 메서드(SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]가 JDBC 사양을 준수하는지를 확인합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean jdbcCompliant()  
```  
  
## <a name="return-value"></a>Return Value  
 JDBC 드라이버가 최소 요구 사항을 충족하면 **true**이고, 그렇지 않으면 **false**입니다.  
  
## <a name="remarks"></a>설명  
 이 jdbcCompliant 메서드는 java.sql.Driver 인터페이스의 jdbcCompliant 메서드에 의해 지정됩니다.
