# <a name="getmaxcharliterallength-method-sqlserverdatabasemetadata"></a>getMaxCharLiteralLength 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스가 문자 리터럴에 허용하는 최대 문자 수를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getMaxCharLiteralLength()  
```  
  
## <a name="return-value"></a>Return Value  
 허용되는 최대 문자 수를 나타내는 **int**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 getMaxCharLiteralLength 메서드는 java.sql.DatabaseMetaData 인터페이스의 getMaxCharLiteralLength 메서드에 의해 지정됩니다.
