# <a name="getfetchsize-method-sqlserverresultset"></a>getFetchSize 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 인출 크기를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getFetchSize()  
```  
  
## <a name="return-value"></a>반환 값  
 현재 인출 크기를 나타내는 **int**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 이 getFetchSize 메서드는 java.sql.ResultSet 인터페이스의 getFetchSize 메서드가 지정 됩니다.
