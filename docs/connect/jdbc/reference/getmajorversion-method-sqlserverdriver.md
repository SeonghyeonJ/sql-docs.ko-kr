# <a name="getmajorversion-method-sqlserverdriver"></a>getMajorVersion 메서드(SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]의 주 버전 번호를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getMajorVersion()  
```  
  
## <a name="return-value"></a>반환 값  
 JDBC 드라이버의 주 버전 번호가 들어 있는 **int**입니다.  
  
## <a name="remarks"></a>Remarks  
 이 getMajorVersion 메서드는 java.sql.Driver 인터페이스의 getMajorVersion 메서드에 의해 지정 됩니다.
