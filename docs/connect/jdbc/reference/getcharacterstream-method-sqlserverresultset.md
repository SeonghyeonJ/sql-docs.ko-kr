# <a name="getcharacterstream-method-sqlserverresultset"></a>getCharacterStream 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열의 값을 java.io.Reader 개체로 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|설명|  
|----------|-----------------|  
|[getCharacterStream(int)](../../../connect/jdbc/reference/getcharacterstream-method-int.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 인덱스의 값을 java.io.Reader 개체로 검색합니다.|  
|[getCharacterStream(java.lang.String)](../../../connect/jdbc/reference/getcharacterstream-method-java-lang-string.md)|이 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체의 현재 행에서 지정된 열 이름의 값을 java.io.Reader 개체로 검색합니다.|
