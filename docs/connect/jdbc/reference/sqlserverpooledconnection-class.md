# <a name="sqlserverpooledconnection-class"></a>SQLServerPooledConnection 클래스
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  연결 풀의 실제 데이터베이스 연결을 나타냅니다.  
  
 **패키지:** com.microsoft.sqlserver.jdbc  
  
 **확장:** java.lang.Object  
  
 **구현:** javax.sql.PooledConnection  
  
## <a name="syntax"></a>구문  
  
```  
  
public final class SQLServerPoolingDataSource  
```  
  
## <a name="remarks"></a>설명  
 SQLServerPooledConnection 클래스는 연결 풀 관리자가 연결 풀을 관리하는 데 사용할 수 있는 메서드를 제공합니다. 애플리케이션에서는 일반적으로 이러한 연결을 직접 인스턴스화하지 않습니다.
