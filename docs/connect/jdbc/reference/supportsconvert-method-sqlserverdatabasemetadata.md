# <a name="supportsconvert-method-sqlserverdatabasemetadata"></a>supportsConvert 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스에서 SQL 형식 간의 CONVERT 기능을 지원하는지 여부를 검색합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|설명|  
|----------|-----------------|  
|[supportsConvert()](../../../connect/jdbc/reference/supportsconvert-method.md)|이 데이터베이스에서 SQL 형식 간의 CONVERT 기능을 지원하는지 여부를 검색합니다.|  
|[supportsConvert(int, int)](../../../connect/jdbc/reference/supportsconvert-method-int-int.md)|이 데이터베이스에서 지정된 두 SQL 형식의 CONVERT 기능을 지원하는지 여부를 검색합니다.|
