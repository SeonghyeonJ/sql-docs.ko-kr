# <a name="free-method-sqlserverblob"></a>free 메서드(SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 메서드는 BLOB 개체 및 이 개체가 보유한 리소스를 해제합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public void free()  
```  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 free 메서드는 java.sql.Blob 인터페이스의 free 메서드에 의해 지정됩니다.
