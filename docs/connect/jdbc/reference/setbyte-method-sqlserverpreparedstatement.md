# <a name="setbyte-method-sqlserverpreparedstatement"></a>setByte 메서드(SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수를 지정된 **byte** 값으로 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public final void setByte(int n,  
                          byte x)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *n*  
  
 매개 변수 번호를 나타내는 **int**입니다.  
  
 *x*  
  
 **byte** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 setByte 메서드는 java.sql.PreparedStatement 인터페이스의 setByte 메서드에 의해 지정됩니다.
