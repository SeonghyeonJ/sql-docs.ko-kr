# <a name="supportsstatementpooling-method-sqlserverdatabasemetadata"></a>supportsStatementPooling 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스에서 문 풀링을 지원하는지 여부를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean supportsStatementPooling()  
```  
  
## <a name="return-value"></a>Return Value  
 지원되는 경우 **true**입니다. 그렇지 않으면 **false**입니다.  
  
## <a name="exceptions"></a>예외  
 java.sql.SQLException  
  
## <a name="remarks"></a>설명  
 이 supportsStatementPooling 메서드는 java.sql.DatabaseMetaData 인터페이스의 supportsStatementPooling 메서드에 의해 지정됩니다.
