# <a name="getdrivermajorversion-method-sqlserverdatabasemetadata"></a>getDriverMajorVersion 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 JDBC 드라이버의 주 버전 번호를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getDriverMajorVersion()  
```  
  
## <a name="return-value"></a>Return Value  
 JDBC 드라이버 주 버전을 나타내는 **int**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>설명  
 이 getDriverMajorVersion 메서드는 java.sql.DatabaseMetaData 인터페이스의 getDriverMajorVersion 메서드에 의해 지정됩니다.
