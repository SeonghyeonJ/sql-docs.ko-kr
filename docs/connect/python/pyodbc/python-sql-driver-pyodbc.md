## <a name="support"></a>Support(지원)

PyODBC는 커뮤니티에서 지원하는 소프트웨어입니다. Microsoft는 pyODBC 오픈 소스 커뮤니티에 기여하며 [https://github.com/mkleehammer/pyodbc/](https://github.com/mkleehammer/pyodbc/)의 리포지토리에 적극적으로 참가합니다. 그러나 Microsoft는 이 소프트웨어에 대한 지원을 제공하지 않습니다.

도움을 받으려면 [pyODBC GitHub 리포지토리에 문제를 등록](https://github.com/mkleehammer/pyodbc/issues)하거나 다른 Python 커뮤니티 리소스를 방문해 보세요.
