# <a name="sql-data-developer"></a>SQL 데이터 개발자
Microsoft의 SQL data platform을 사용 하 여 웹 서버, 엔터프라이즈 서버 및 클라우드의 모바일 장치 및 데스크톱에서 데이터 중심 솔루션을 만들 수 있습니다.  

## <a name="sql-data-storage"></a>SQL 데이터 스토리지
* [데이터베이스 엔진 SQL Server](../database-engine/configure-windows/sql-server-database-engine.md): SQL Server 데이터베이스 엔진를 사용 하 여 온라인 트랜잭션 처리 또는 온라인 분석 처리 데이터에 대 한 관계형 데이터베이스를 만듭니다. 
* [SQL Azure](https://docs.microsoft.com/azure/sql-database/): SQL Azure를 사용 하 여 데이터베이스를 클라우드로 이동 
* [SQL Compact](https://www.microsoft.com/download/details.aspx?id=17876): 모바일 장치, 데스크톱 및 웹 클라이언트를 위한 독립 실행형 응용 프로그램 및 간헐적으로 연결 되는 응용 프로그램을 빌드하 SQL Server Compact 합니다.
* [Localdb](../database-engine/configure-windows/sql-server-2016-express-localdb.md): SQL Server의 전체 서버 인스턴스를 관리할 필요 없이 개발 중에 localdb를 사용 하 여 transact-sql 코드를 작성 하 고 테스트 합니다.

## <a name="sql-data-tools"></a>SQL Data tools
* [SQL Server Data Tools](../ssdt/download-sql-server-data-tools-ssdt.md) : SSDT를 사용 하 여 관계형 데이터베이스, Azure SQL 데이터베이스, Integration Services 패키지, Analysis Services 데이터 모델 및 Reporting Services 보고서를 작성 합니다.
* [SQL Server 관리 도구](../ssms/download-sql-server-management-studio-ssms.md): SSMS를 사용 하 여 SQL Server 인스턴스를 구성, 모니터링 및 관리 합니다.

## <a name="sql-data-access"></a>SQL 데이터 액세스
* [Sql 클라이언트 드라이버](sql-connection-libraries.md): sql 드라이버를 사용 하 여 sql database에서 데이터를 연결, 쿼리, 업데이트, 삽입 또는 삭제할 수 있습니다.
* [Entity Framework](https://msdn.microsoft.com/library/gg696172.aspx): Entity Framework를 사용 하 여 LINQ를 사용 하 여 데이터베이스에 쉽게 액세스 하 고 SQL Server에 직접 액세스 하 고 엔터티 데이터 모델 (EDM)와 원시 관계형 구조 간의 매핑을 제어 하는 옵션을 사용 합니다. 
* [Windows Communication Foundation (WCF)](https://msdn.microsoft.com/library/dd456779.aspx): 거의 턴키 솔루션을 위해 Data Services를 사용 하 여 웹 및 인트라넷에서 OData 서비스를 쉽게 만들고 사용할 수 있습니다.
* [Sync framework](https://msdn.microsoft.com/library/jj839436.aspx): 동기화 프레임 워크를 사용 하 여 모든 데이터 형식, 모든 데이터 저장소, 전송 프로토콜 및 네트워크 토폴로지에 대 한 오프 라인 액세스를 사용 하도록 설정 합니다.
* [사후 확장](https://msdn.microsoft.com/library/hh242985.aspx): 대응식 확장 (Rx)을 사용 하 여 이벤트 스트림 프로그래밍을 수행 하 고 예측 가능한 시퀀스 및 LINQ 스타일 쿼리 연산자를 사용 하 여 비동기 프로그래밍을 간소화 합니다.  JavaScript 용 대응식 확장 (RxJS)을 사용 하면 관찰 가능한 시퀀스를 사용 하 여 웹에서 비동기 콜백 기반 프로그래밍 및 이벤트 구동 프로그래밍을 단순화할 수 있습니다.
* [CLR 통합](../relational-databases/clr-integration/common-language-runtime-clr-integration-programming-concepts.md): CLR 통합을 통해 Microsoft Visual Basic .NET 및 Microsoft Visual C#을 포함한 모든 .NET Framework 언어를 사용하여 저장 프로시저, 트리거, 사용자 정의 형식, 사용자 정의 함수, 사용자 정의 집계 및 스트리밍 테이블 반환 함수를 작성할 수 있습니다. 
* [Sqlxml 4.0](../relational-databases/sqlxml/sqlxml-4-0-programming-concepts.md): sqlxml 4.0을 사용 하 여 관계형 데이터를 XML로 내보냅니다.
