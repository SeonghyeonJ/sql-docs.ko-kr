# <a name="support-resources-for-the-microsoft-drivers-for-php-for-sql-server"></a>Microsoft Drivers for PHP for SQL Server에 대한 지원 리소스
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

이 항목에서는 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]를 사용하는 애플리케이션을 개발할 때 유용할 수 있는 리소스를 나열합니다.  
  
## <a name="ssdriverphp-support-resources"></a>[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] 지원 리소스  
[에 대한 최신 설명서는](microsoft-php-driver-for-sql-server.md) Microsoft PHP Driver for SQL Server [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]를 참조하세요.  
  
피어 투 피어 지원은 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] [포럼](https://social.msdn.microsoft.com/Forums/sqlserver/home?forum=sqldriverforphp)을 방문하세요.  
  
피드백 제공, 질문하기 또는 개발 팀이 고려하고 있는 내용에 대해 알아보려면 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] [SQL Server 드라이버 블로그](https://techcommunity.microsoft.com/t5/sql-server/bg-p/SQLServer/label-name/SQLServerDrivers)를 방문하세요.  
  
[GitHub의 SQL Server용 Microsoft PHP Driver 소스 코드](https://github.com/Microsoft/msphpsql)  
  
## <a name="sql-server-and-transact-sql-support-resources"></a>SQL Server 및 Transact-SQL 지원 리소스
SQL Server 및 Transact-SQL 설명서는 [SQL Server 설명서](../../sql-server/index.yml)에서 찾을 수 있습니다.
  
피어 투 피어 지원은 [MSDN SQL Server 포럼](https://social.msdn.microsoft.com/Forums/sqlserver/home)을 방문하세요.  
  
## <a name="internet-information-services-iis-support-resources"></a>IIS(인터넷 정보 서비스) 지원 리소스  
최신 IIS 뉴스를 보려면 [IIS 홈](https://www.iis.net/)을 방문하세요.  
  
피어 투 피어 지원은 [IIS 포럼](https://forums.iis.net/)을 방문하세요.  
  
## <a name="php-support-resources"></a>PHP 지원 리소스  
[Windows용 PHP 설명서](https://windows.php.net/)  
  
PHP에 대한 최신 정보는 [http://www.php.net/](http://www.php.net)을 방문하세요.  
  
PHP 설명서는 [http://www.php.net/docs.php](http://www.php.net/docs.php)를 방문하세요.
