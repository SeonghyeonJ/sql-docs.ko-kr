# <a name="overview-of-the-microsoft-drivers-for-php-for-sql-server"></a>Microsoft Drivers for PHP for SQL Server 개요

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]는 Azure SQL Database를 비롯한 SQL Server 2005 이상 버전에 대한 데이터 액세스를 제공하는 PHP 확장입니다. 확장은 SQLSRV 드라이버와 PDO_SQLSRV 드라이버를 사용 하는 프로시저 인터페이스를 제공 하 여 Express를 비롯 한 모든 SQL Server 버전의 데이터에 액세스 합니다 (2005 SQL Server부터). 3\.1 이상 버전의 드라이버에 대 한 지원은 SQL Server 2008로 시작 합니다. [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] API에는 Windows 인증, 트랜잭션, 매개 변수 바인딩, 스트리밍, 메타데이터 액세스 및 오류 처리에 대한 지원이 포함됩니다.  
  
을 사용 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]하려면 PHP가 실행 중인 동일한 컴퓨터에 올바른 버전의 SQL Server Native Client 또는 Microsoft ODBC 드라이버가 설치 되어 있어야 합니다.  자세한 내용은 [Microsoft Drivers FOR PHP for SQL Server의 시스템 요구 사항](../../connect/php/system-requirements-for-the-php-sql-driver.md)을 참조 하세요.
