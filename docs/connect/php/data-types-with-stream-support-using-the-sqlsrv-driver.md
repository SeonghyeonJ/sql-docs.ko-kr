# <a name="data-types-with-stream-support-using-the-sqlsrv-driver"></a>SQLSRV 드라이버를 사용하여 스트림으로 데이터 형식 지원
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

스트림으로 데이터 검색은 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]의 SQLSRV 드라이버에서만 사용 가능하고 PDO_SQLSRV 드라이버에서는 사용할 수 없습니다.  
  
다음 SQL Server 데이터 형식은 SQLSRV 드라이버를 사용하여 스트림으로 검색할 수 있습니다.  
  
-   binary  
  
-   char  
  
-   이미지  
  
-   nchar  
  
-   ntext  
  
-   nvarchar  
  
-   text  
  
-   UDT  
  
-   varbinary  
  
-   varchar  
  
-   XML
