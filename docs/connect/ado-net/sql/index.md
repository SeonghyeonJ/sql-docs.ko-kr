# <a name="sql-server-and-adonet"></a>SQL Server 및 ADO.NET

[!INCLUDE[Driver_ADONET_Download](../../../includes/driver_adonet_download.md)]

## <a name="microsoftdatasqlclient"></a>Microsoft.Data.SqlClient

<xref:Microsoft.Data.SqlClient>는 데이터베이스 관련 프로토콜을 캡슐화하는 SQL Server 버전에 대한 액세스를 제공합니다. <xref:Microsoft.Data.SqlClient>에는 SQL Server와 직접 통신하기 위한 TDS(Tabular Data Stream) 파서가 포함되어 있습니다.  
  
> [!NOTE]
> SQL Server에 Microsoft SqlClient Data Provider를 사용하려면 애플리케이션에서 <xref:Microsoft.Data.SqlClient> 네임스페이스를 참조해야 합니다.
