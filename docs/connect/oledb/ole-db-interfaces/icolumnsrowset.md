# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  OLE DB Driver for SQL Server는 DBCOLUMN_BASETABLEINSTANCE 열을 IColumnsRowset::GetColumnRowset에 추가합니다. 이 열은 DBTYPE_I2를 반환하며 Microsoft에서 사용하도록 예약되어 있습니다. 이 열의 정보는 이후 릴리스에서 변경될 수 있습니다.
