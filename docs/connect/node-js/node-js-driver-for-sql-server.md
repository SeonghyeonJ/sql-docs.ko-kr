# <a name="nodejs-driver-for-sql-server"></a>SQL Server용 Node.js 드라이버

![Download-DownArrow-Circled](../../ssms/media/download-icon.png)[Node.js SQL 드라이버 다운로드](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Tedious 모듈은 모든 최신 버전의 SQL Server에서 지원되는 TDS 프로토콜(JavaScript로 구현)입니다. 드라이버는 오픈 소스 프로젝트이며 GitHub에서 제공됩니다.  
  
Windows, Linux 또는 macOS에서 Node.js를 사용하여 SQL 데이터베이스에 연결할 수 있습니다.

## <a name="support"></a>Support(지원)

Tedious for Node.js는 커뮤니티에서 지원하는 소프트웨어입니다. Microsoft는 tedious 오픈 소스 커뮤니티에 기여하며 [https://github.com/tediousjs/tedious](https://github.com/tediousjs/tedious)의 리포지토리에 적극적으로 참가합니다. 그러나 Microsoft는 이 소프트웨어에 대한 지원을 제공하지 않습니다.

도움을 받으려면 [tedious GitHub 리포지토리에 문제를 등록](https://github.com/tediousjs/tedious/issues)하거나 다른 Node.js 커뮤니티 리소스를 방문해 보세요.
