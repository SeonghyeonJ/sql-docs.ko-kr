# <a name="nodejs-driver-for-sql-server"></a>SQL Server용 Node.js 드라이버

![다운로드-아래쪽 화살표-원으로 둘러싸인](../../ssdt/media/download.png)[NODE.JS SQL 드라이버 다운로드](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

번거로운 모듈은 모든 최신 버전의 SQL Server에서 지원되는 TDS 프로토콜의 javascript 구현입니다. 드라이버는 Github에서 사용할 수 있는 오픈 소스 프로젝트입니다.  
  
Windows, Linux 또는 Mac에서 node.js를 사용 하 여 SQL Database에 연결할 수 있습니다.
