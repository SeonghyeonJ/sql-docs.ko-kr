- [https://docs.microsoft.com/previous-versions/sql/](/previous-versions/sql/)

이러한 보관 파일에는 최소한 다음과 같은 이전 버전에 대한 설명서가 포함되어 있습니다.

- SQL Server 2014
- SQL Server 2012
- SQL Server 2008 R2
- SQL Server 2008
- SQL Server 2005
