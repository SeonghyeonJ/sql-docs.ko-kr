Linux 컴퓨터에서 이전 섹션에서 사용된 온라인 리포지토리에 액세스할 수 없는 경우 패키지 파일을 직접 다운로드할 수 있습니다. 이러한 패키지는 Microsoft 리포지토리를에 위치한 [ https://packages.microsoft.com ](https://packages.microsoft.com)합니다.

> [!TIP]
> 이전 섹션의 단계로 설치에 성공한 경우 아래 패키지를 다운로드하거나 수동으로 설치할 필요가 없습니다. 오프라인 시나리오에만 해당됩니다.
