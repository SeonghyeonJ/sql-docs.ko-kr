# <a name="odbc-test"></a>ODBC 테스트
Microsoft® ODBC 테스트는 ODBC 드라이버및 ODBC 드라이버 관리자를 테스트하는 데 사용할 수 있는 ODBC 지원 응용 프로그램입니다. ODBC 3.51에는 ANSI 및 유니코드 지원 ODBC 테스트 버전이 모두 포함되어 있습니다. 해당 파일은 다음과 같습니다.  
  
-   ANSI 버전에 대한 Odbcte32.exe 및 Gtrtst32.dll.  
  
-   유니코드 버전의 경우 Odbct32w.exe 및 Gtrts32w.dll입니다.  
  
 ODBC 테스트를 사용하려면 ODBC API, C 언어 및 SQL을 이해해야 합니다. ODBC API에 대한 자세한 내용은 [ODBC 프로그래머의 참조를](../odbc/reference/odbc-programmer-s-reference.md)참조하십시오.  
  
 문서의 이 섹션에 이전에 포함되었던 도움말 항목은 이제 ODBC 테스트 프로그램에 포함됩니다. Odbcte32.exe 또는 Odbct32w.exe를 열고 **도움말** 메뉴를 연 다음 **도움말 항목을 클릭합니다.**  
  
 64비트 Microsoft Windows 운영 체제를 위한 이러한 응용 프로그램의 64비트 버전은 별도의 파일임에도 불구하고 32비트 버전과 동일한 이름을 가수 있습니다. 즉, ODBC 테스트의 64 비트 버전의 유니코드 버전의 이름은 odbct32w.exe입니다.
