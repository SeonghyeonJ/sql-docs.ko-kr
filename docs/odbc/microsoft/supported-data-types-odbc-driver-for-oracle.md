# <a name="supported-data-types-odbc-driver-for-oracle"></a>지원되는 데이터 형식(Oracle용 ODBC 드라이버)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거 될 예정입니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 Oracle에서 제공 하는 ODBC 드라이버를 사용 합니다.  
  
 Oracle 용 ODBC 드라이버는 모든 Oracle 7.3 데이터 형식을 지원 합니다. 그러나 여기에 나열 된 새로운 Oracle8 데이터 형식을 지원 하지 않습니다.  
  
|데이터 형식|Oracle 7.3|Oracle8|  
|---------------|----------------|-------------|  
|BFILE|해당 없음|지원되지 않음|  
|BLOB|해당 없음|지원되지 않음|  
|CHAR|지원됨|지원됨|  
|CLOB|해당 없음|지원되지 않음|  
|DATE|지원됨|지원됨|  
|FLOAT|지원됨|지원됨|  
|정수|지원됨|지원됨|  
|LONG|지원됨|지원됨|  
|LONG RAW|지원됨|지원됨|  
|NCHAR|해당 없음|지원되지 않음|  
|NCLOB|해당 없음|지원되지 않음|  
|NUMBER|지원됨|지원됨|  
|NVARCHAR2|해당 없음|지원되지 않음|  
|RAW|지원됨|지원됨|  
|VARCHAR2|지원됨|지원됨|  
|MLSLABEL|지원되지 않습니다.|지원되지 않습니다.|  
  
> [!NOTE]  
>  VARCHAR 열에 허용 되는 크기에 대 한 자세한 내용은이 가이드의 [Varchar 열 크기](../../odbc/microsoft/varchar-column-size-odbc-driver-for-oracle.md) 를 참조 하십시오.
