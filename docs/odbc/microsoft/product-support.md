# <a name="product-support"></a>기술 지원 서비스
ODBC에 대한 제품 지원은 Microsoft 제품 지원 서비스에서 제공합니다. 많은 Microsoft 제품이 ODBC를 핵심 구성 요소로 사용하고 ODBC 드라이버를 재배포하기 때문에 Microsoft 제품 지원 서비스는 ODBC를 구성 요소로 통합하는 각 제품의 일부로 간주합니다. ODBC에 대한 지원은 해당 제품의 지원 계약에 따라 제공됩니다. 자세한 내용은 개별 제품 설명서를 참조하십시오.
