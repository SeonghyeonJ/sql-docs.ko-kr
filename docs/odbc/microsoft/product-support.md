# <a name="product-support"></a>기술 지원 서비스
ODBC에 대 한 제품 지원은 Microsoft 기술 지원 서비스에서 제공 합니다. 많은 Microsoft 제품은 ODBC를 핵심 구성 요소로 사용 하 고 ODBC 드라이버도 재배포 하므로 Microsoft 기술 지원 서비스는 odbc를 구성 요소로 ODBC를 통합 하는 각 제품의 일부로 간주 합니다. ODBC에 대 한 지원은 해당 제품의 지원 계약에 따라 제공 됩니다. 자세한 내용은 개별 제품 설명서를 참조 하십시오.
