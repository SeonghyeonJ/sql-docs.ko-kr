# <a name="convert-function-limitations"></a>CONVERT 함수 제한 사항
형식 변환 실패가 발생 하면 영향을 받는 열이 NULL로 설정 됩니다.  
  
 CONVERT 함수를 통해 날짜 또는 타임 스탬프 데이터 형식을 다른 데이터 형식으로 변환할 수 없습니다.
