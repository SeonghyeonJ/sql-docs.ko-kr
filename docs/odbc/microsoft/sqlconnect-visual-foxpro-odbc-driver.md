# <a name="sqlconnect-visual-foxpro-odbc-driver"></a>SQLConnect(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 [데이터베이스](../../odbc/microsoft/visual-foxpro-terminology.md) 또는 [테이블](../../odbc/microsoft/visual-foxpro-terminology.md)디렉터리일 수 있는 데이터 원본에 연결합니다. 비주얼 FoxPro ODBC 드라이버는 *szUID,* *cbUID,* *szAuthStr*및 *cbAuthStr* 인수를 무시합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLConnect를](../../odbc/reference/syntax/sqlconnect-function.md) 참조하십시오.
