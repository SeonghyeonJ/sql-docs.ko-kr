# <a name="sqlconnect-visual-foxpro-odbc-driver"></a>SQLConnect(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 [데이터베이스](../../odbc/microsoft/visual-foxpro-terminology.md) 또는 [테이블](../../odbc/microsoft/visual-foxpro-terminology.md)의 디렉터리 일 수 있는 데이터 원본에 연결 합니다. Visual FoxPro ODBC 드라이버는 *Szuid*, *cbuid*, *szauthstr*및 *cbauthstr* 인수를 무시 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [SQLConnect](../../odbc/reference/syntax/sqlconnect-function.md) 를 참조 하세요.
