 지원: 전체

 명령문 핸들, *hstmt에서*처리를 취소합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLCancel을](../../odbc/reference/syntax/sqlcancel-function.md) 참조하십시오.
