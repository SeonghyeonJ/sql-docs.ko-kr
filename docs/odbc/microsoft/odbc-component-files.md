# <a name="odbc-component-files"></a>ODBC 구성 요소 파일
Odbc 드라이버 파일은 ODBC 데스크톱 데이터베이스 드라이버 4.0의 각 구성 요소에 필요 합니다.  
  
 일부 파일은 둘 이상의 구성 요소에 필요 합니다. 상용 응용 프로그램을 사용 하 여 ODBC 데스크톱 데이터베이스 드라이버를 재배포 하려는 경우 이러한 파일을 재배포 해야 합니다.  
  
 이러한 파일은 Windows 95 이상에 대 한 \Windows\System 디렉터리 또는 Windows NT 3.51 이상 및 Windows 2000의 \Windows\System32 디렉터리에 설치 됩니다. ODBC 파일이 이전에 다른 디렉터리에 설치 된 경우에는 \Windows\System (또는 System32) 디렉터리에서 최신 파일을 사용 해야 합니다.
