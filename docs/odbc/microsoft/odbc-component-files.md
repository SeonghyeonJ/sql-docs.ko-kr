# <a name="odbc-component-files"></a>ODBC 구성 요소 파일
ODBC 드라이버 파일은 ODBC 데스크톱 데이터베이스 드라이버 4.0의 각 구성 요소에 필요합니다.  
  
 일부 파일은 둘 이상의 구성 요소에 필요합니다. 이러한 파일은 상용 응용 프로그램과 ODBC 데스크톱 데이터베이스 드라이버를 재배포하려는 경우 재배포해야 합니다.  
  
 이러한 파일은 Windows 95 이상용 \Windows\System 디렉토리에 설치되거나 Windows NT 3.51 이상 및 Windows 2000용 \Windows\System32 디렉토리에 설치됩니다. ODBC 파일이 이전에 다른 디렉터리에 설치된 경우 \Windows\System(또는 System32) 디렉터리에서 최신 파일을 사용해야 합니다.
