# <a name="sqlgetcursorname-visual-foxpro-odbc-driver"></a>SQLGetCursorName(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 지정된 *hstmt와*연결된 커서의 이름을 반환합니다. **SQLGetCursorName은** 코어 레벨 API 기능의 일부이기 때문에 Visual FoxPro ODBC 드라이버 API에 포함됩니다. 드라이버는 위치 업데이트를 지원하지 않으므로 다른 API 함수와 함께 사용할 수 없습니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLGetCursorName을](../../odbc/reference/syntax/sqlgetcursorname-function.md) 참조하십시오.
