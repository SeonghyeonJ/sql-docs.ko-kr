# <a name="sqlgetcursorname-visual-foxpro-odbc-driver"></a>SQLGetCursorName(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 지정 된 *hstmt*와 연결 된 커서의 이름을 반환 합니다. **SQLGetCursorName** 는 핵심 수준 API 기능의 일부 이므로 VISUAL FoxPro ODBC 드라이버 API에 포함 되어 있습니다. 이 드라이버는 위치가 지정 된 업데이트를 지원 하지 않기 때문에 다른 API 함수와 함께 사용할 수 없습니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [SQLGetCursorName](../../odbc/reference/syntax/sqlgetcursorname-function.md) 를 참조 하세요.
