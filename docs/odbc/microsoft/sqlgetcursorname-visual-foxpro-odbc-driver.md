# <a name="sqlgetcursorname-visual-foxpro-odbc-driver"></a>SQLGetCursorName(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에서는 Visual FoxPro ODBC 드라이버 관련 정보를 포함합니다. 이 함수에 대 한 일반 정보에서 해당 항목을 참조 하세요 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)합니다.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 연관 된 커서의 이름을 반환 하는 지정 된 *hstmt*합니다. **SQLGetCursorName** 때문에 Visual FoxPro ODBC 드라이버 API에 포함 되어 핵심 수준 API 기능의 일부인; 드라이버 위치 지정된 업데이트를 지원 하지 않으므로 다른 API 함수를 사용 하 여 사용할 수 없습니다.  
  
 자세한 내용은 [SQLGetCursorName](../../odbc/reference/syntax/sqlgetcursorname-function.md) 에 *ODBC 프로그래머 참조*합니다.
