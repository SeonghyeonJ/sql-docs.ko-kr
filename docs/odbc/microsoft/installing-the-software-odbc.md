# <a name="installing-the-software-odbc"></a>소프트웨어 설치(ODBC)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거 될 예정입니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 Oracle에서 제공 하는 ODBC 드라이버를 사용 합니다.  
  
 Oracle 용 ODBC 드라이버는 데이터 액세스 구성 요소 중 하나입니다. ODBC 데이터 원본 관리자와 같은 다른 ODBC 구성 요소가 함께 제공 되며 이미 설치 되어 있어야 합니다. 이 드라이버는 [www.microsoft.com](https://www.microsoft.com)에서 Microsoft 기술 지원 서비스 온라인 웹 사이트의 "드라이버 및 기타 다운로드"에서 찾을 수도 있습니다.  
  
 네트워크 소프트웨어는 자체 설명서에 따라 설치 해야 합니다. Oracle 용 ODBC 드라이버에는 네트워크 소프트웨어가 지원 되는 경우 특별 한 설치 고려 사항이 필요 하지 않습니다.  
  
 Oracle 소프트웨어는 자체 설명서에 따라 설치 해야 합니다. Oracle 용 ODBC 드라이버는 일반적으로 드라이버에서 버전을 지 원하는 경우 특별 한 설치 고려 사항을 요구 하지 않습니다. 그러나 제품의 호환성을 유지 하려면 Oracle 용 ODBC 드라이버를 설치 하 여 최신 버전의 드라이버가 설치 되어 있는지 확인 합니다. Oracle은 다른 작업 중에 서버 제품과 함께 제공 되는 클라이언트 구성 요소 및 Oracle 서버 제품에 대 한 패치를 게시 하는 공용 FTP 사이트를 유지 관리 합니다. 이러한 패치는 여러 Microsoft 제품 및 기술에 적합 한 기능을 위해 필요 합니다. 이 사이트에 대 한 자세한 내용은 [Oracle 소프트웨어 패치](../../odbc/microsoft/oracle-software-patches.md)를 참조 하세요.  
  
> [!CAUTION]  
>  MDAC/Windows DAC를 통해 Oracle 소프트웨어를 설치 하면 MDAC의 현재 버전을 덮어쓸 수 있습니다. ODBC 구성 요소를 사용 하 여 문제가 발생 하는 경우 MDAC를 다시 설치 하십시오.
