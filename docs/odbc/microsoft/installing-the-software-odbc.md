# <a name="installing-the-software-odbc"></a>소프트웨어 설치(ODBC)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 오라클에서 제공하는 ODBC 드라이버를 사용합니다.  
  
 오라클의 ODBC 드라이버는 데이터 액세스 구성 요소 중 하나입니다. ODBC 데이터 원본 관리자와 같은 다른 ODBC 구성 요소와 함께 제공되며 이미 설치해야 합니다. 드라이버는 또한 [www.microsoft.com](https://www.microsoft.com)Microsoft 제품 지원 서비스 온라인 웹 사이트의 "드라이버 및 기타 다운로드"에서 찾을 수 있습니다.  
  
 네트워크 소프트웨어는 자체 설명서에 따라 설치해야 합니다. 오라클의 ODBC 드라이버는 네트워크 소프트웨어가 지원되는 한 특별한 설치 고려 사항이 필요하지 않습니다.  
  
 오라클 소프트웨어는 자체 설명서에 따라 설치해야 합니다. 오라클의 ODBC 드라이버는 일반적으로 드라이버가 버전을 지원하는 한 특별한 설치 고려 사항이 필요하지 않습니다. 그러나 제품을 호환하려면 오라클의 ODBC 드라이버를 마지막으로 설치하여 최신 버전의 드라이버를 사용할 수 있도록 합니다. 오라클은 오라클 서버 제품 및 서버 제품과 함께 제공되는 클라이언트 구성 요소에 대한 패치를 게시하는 공개 FTP 사이트를 유지 관리합니다. 이러한 패치는 여러 Microsoft 제품 및 기술의 적절한 작동을 위해 필요합니다. 이 사이트에 대한 자세한 내용은 [Oracle 소프트웨어 패치를](../../odbc/microsoft/oracle-software-patches.md)참조하십시오.  
  
> [!CAUTION]  
>  MDAC/Windows DAC에 Oracle 소프트웨어를 설치하면 현재 버전의 MDAC를 덮어쓸 수 있습니다. ODBC 구성 요소를 사용하여 문제가 발생하면 MDAC를 다시 설치하십시오.
