# <a name="native-error"></a>원시 오류
긍정적인 기본 오류 코드는 데스크톱 데이터베이스 드라이버에서 생성됩니다. 부정적인 오류 코드는 마이크로 소프트 제트에 의해 생성됩니다. 자세한 내용은 Microsoft *Jet 데이터베이스 엔진 프로그래머 가이드를*참조하십시오.
