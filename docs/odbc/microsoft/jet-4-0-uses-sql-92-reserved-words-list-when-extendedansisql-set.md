# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisql_set"></a>Jet 4.0은 ExtendedAnsiSQL_Set을 사용할 때 SQL-92 예약어 목록을 사용합니다.
Extendedansisql을 플래그가 설정 되 면 Jet 4.0는 SQL-92 예약어 목록을 사용 합니다. 따옴표 붙지 않은 개체 이름으로 SQL-92 예약어를 사용 하려고 하면 구문 오류가 발생 합니다. Extendedansisql을 플래그가 해제 되 면 새 예약어는 이전과 같이 개체 이름으로 사용 될 수 있습니다.
