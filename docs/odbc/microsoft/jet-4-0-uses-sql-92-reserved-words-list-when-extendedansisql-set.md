# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisqlset"></a>Jet 4.0은 ExtendedAnsiSQL_Set을 사용할 때 SQL-92 예약어 목록을 사용합니다.
Extendedansisql을 플래그를 설정 하는 경우 Jet 4.0 SQL-92 예약어 목록을 사용 합니다. SQL-92 사용 하려고 예약어 따옴표가 없는 개체 이름에 구문 오류가 발생 됩니다. Extendedansisql을 플래그를 해제 하 고, 새 예약 된 단어 앞으로 개체 이름으로 사용할 수 있습니다.
