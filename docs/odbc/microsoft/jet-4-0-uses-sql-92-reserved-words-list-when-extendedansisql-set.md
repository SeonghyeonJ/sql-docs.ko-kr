# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisql_set"></a>Jet 4.0은 ExtendedAnsiSQL_Set을 사용할 때 SQL-92 예약어 목록을 사용합니다.
ExtendedAnsiSQL 플래그가 켜져 있으면 Jet 4.0은 SQL-92 예약된 단어 목록을 사용합니다. SQL-92 예약된 단어를 인용되지 않은 개체 이름으로 사용하려고 하면 구문 오류가 발생합니다. ExtendedAnsiSQL 플래그가 꺼지면 새 예약된 단어를 이전과 같이 개체 이름으로 사용할 수 있습니다.
