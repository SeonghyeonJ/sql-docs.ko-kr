# <a name="querying-and-updating-visual-foxpro-data-from-microsoft-access"></a>Microsoft Access에서 Visual FoxPro 데이터 쿼리 및 업데이트
링크 테이블 옵션을 사용하여 Microsoft Access 데이터베이스에서 Visual FoxPro 데이터베이스에 저장된 데이터를 쿼리하고 업데이트할 수 있습니다.  
  
### <a name="to-link-a-visual-foxpro-database-to-a-microsoft-access-database"></a>비주얼 FoxPro 데이터베이스를 Microsoft 액세스 데이터베이스에 연결하려면  
  
1.  Microsoft 액세스 데이터베이스를 엽니다.  
  
2.  테이블 탭에서 새로 단추를 클릭합니다.  
  
3.  새 테이블 대화 상자에서 링크 테이블을 선택하고 확인을 클릭합니다.  
  
4.  대화 상자링크에서 형식 목록의 파일에서 ODBC 데이터베이스를 선택합니다.  
  
5.  SQL 데이터 원본 대화 상자에서 쿼리할 Visual FoxPro 데이터에 연결하는 데이터 원본을 선택하고 확인을 클릭합니다.  
  
6.  테이블 링크 대화 상자에서 쿼리및 업데이트할 테이블을 선택하고 확인을 클릭합니다. 연결된 Visual FoxPro 테이블은 Microsoft Access 데이터베이스의 테이블 탭에 표시됩니다.  
  
 이제 Microsoft Access를 사용하여 연결된 Visual FoxPro 테이블에서 데이터를 쿼리하고 업데이트할 수 있습니다. 연결된 데이터에 대한 변경 내용은 Visual FoxPro 데이터 원본으로 다시 전송됩니다.  
  
 Visual FoxPro 데이터 원본의 데이터에 영향을 미치도록 Microsoft Access에서 변경하지 않으려면 [Visual FoxPro 데이터를 Microsoft 액세스로 가져오기](../../odbc/microsoft/importing-visual-foxpro-data-into-microsoft-access.md)를 참조하십시오.
