# <a name="querying-and-updating-visual-foxpro-data-from-microsoft-access"></a>Microsoft Access에서 Visual FoxPro 데이터 쿼리 및 업데이트
연결 테이블 옵션을 사용 하 여 Microsoft Access 데이터베이스에서 Visual FoxPro 데이터베이스에 저장 된 데이터를 쿼리하고 업데이트할 수 있습니다.  
  
### <a name="to-link-a-visual-foxpro-database-to-a-microsoft-access-database"></a>Visual FoxPro 데이터베이스를 Microsoft Access 데이터베이스에 연결 하려면  
  
1.  Microsoft Access 데이터베이스를 엽니다.  
  
2.  테이블 탭에서 새로 만들기를 클릭 합니다.  
  
3.  새 테이블 대화 상자에서 테이블 연결을 선택 하 고 확인을 클릭 합니다.  
  
4.  링크 대화 상자의 파일 형식 목록에서 ODBC 데이터베이스를 선택 합니다.  
  
5.  SQL 데이터 원본 대화 상자에서 쿼리 하려는 Visual FoxPro 데이터에 연결 하는 데이터 원본을 선택 하 고 확인을 클릭 합니다.  
  
6.  테이블 연결 대화 상자에서 쿼리하고 업데이트할 테이블을 선택 하 고 확인을 클릭 합니다. Microsoft Access 데이터베이스의 테이블 탭에 연결 된 Visual FoxPro 테이블이 표시 됩니다.  
  
 이제 Microsoft Access를 사용 하 여 연결 된 Visual FoxPro 테이블의 데이터를 쿼리하고 업데이트할 수 있습니다. 연결 된 데이터에 대 한 변경 내용은 Visual FoxPro 데이터 원본으로 다시 전송 됩니다.  
  
 Microsoft Access에서 변경 하 여 Visual FoxPro 데이터 원본의 데이터에 영향을 주지 않으려면 [Visual Foxpro 데이터를 Microsoft access로 가져오기](../../odbc/microsoft/importing-visual-foxpro-data-into-microsoft-access.md)를 참조 하세요.
