# <a name="parameterized-query-limitations"></a>매개 변수가 있는 쿼리 제한 사항
Microsoft Access 드라이버를 사용 하는 경우 다음 구문을 사용 하 여 매개 변수가 있는 쿼리를 호출할 수 있습니다. CALL *query-name* [(*parameter*[,*parameter*]...)].
