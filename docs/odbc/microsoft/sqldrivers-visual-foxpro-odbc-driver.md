 ODBC API 규칙: 수준 2  
  
 드라이버 설명 및 드라이버 특성 키워드를 나열 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [sqldrivers](../../odbc/reference/syntax/sqldrivers-function.md) 를 참조 하세요.
