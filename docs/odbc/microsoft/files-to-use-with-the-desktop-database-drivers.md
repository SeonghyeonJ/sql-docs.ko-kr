# <a name="files-to-use-with-the-desktop-database-drivers"></a>데스크톱 데이터베이스 드라이버와 사용할 파일
ODBC 데스크톱 데이터베이스 드라이버는 마이크로 소프트 제트 데이터베이스 엔진, 설치 ISAM 파일 및 마이크로 소프트 ODBC 파일과 함께 사용됩니다. 제품 재배포에 필요한 것은 아니며 모든 구성 요소가 핵심 구성 요소인 것은 아닙니다.  
  
|원하는 정보|참조|  
|---------------------------|---------|  
|각 데스크톱 데이터베이스 드라이버에서 사용하는 파일|ODBC 드라이버 파일|  
|각 데스크톱 데이터베이스 드라이버에서 사용하는 Microsoft Jet 파일|마이크로 소프트 제트 파일|  
|마이크로소프트 제트와 다른 DBMS 간의 데이터 액세스를 제공하는 IISAM 드라이버 파일|설치 가능한 ISAM 드라이버 파일|
