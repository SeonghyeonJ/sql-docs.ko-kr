# <a name="date-arithmetic"></a>날짜 산술
드라이버는 날짜, 시간 또는 타임 스탬프 열에서 정수를 추가 하 고 빼는 것을 지원 합니다. 날짜 열의 경우 정수는 더하거나 뺄 일 수를 지정 합니다. 시간 또는 타임 스탬프 열의 경우 정수는 더하거나 뺄 시간 (초)을 지정 합니다.
