# <a name="date-arithmetic"></a>날짜 산술
드라이버는 더하기 및 빼기에서 날짜, 시간 또는 타임 스탬프 열 사이의 정수를 지원 합니다. 날짜 열에 대 한 정수를 추가 하거나 뺄 일 수를 지정 합니다. 시간 또는 타임 스탬프 열에 대 한 정수를 추가 하거나 뺄 시간 (초) 수를 지정 합니다.
