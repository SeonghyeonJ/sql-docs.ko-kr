# <a name="date-arithmetic"></a>날짜 산술
드라이버는 날짜, 시간 또는 TIMESTAMP 열에서 정수 추가 및 빼기를 지원합니다. DATE 열의 경우 정수는 추가하거나 뺄 일 수를 지정합니다. TIME 또는 TIMESTAMP 열의 경우 정수는 추가하거나 뺄 수 있는 초 수를 지정합니다.
