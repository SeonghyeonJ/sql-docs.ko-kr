# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 특정 *hstmt와*연관된 처리를 *중지하고, hstmt와*연결된 열린 커서를 닫고, 보류 중인 결과를 삭제하고, 선택적으로 문 핸들과 연관된 모든 리소스를 해제합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLFreeStmt를](../../odbc/reference/syntax/sqlfreestmt-function.md) 참조하십시오.
