# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 수준 2  
  
 행 집합에서 커서 위치를 설정 합니다. **SQLSetPos** 를 [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 와 함께 사용 하 여 행 집합의 특정 행에 커서를 놓은 후 바인딩되지 않은 열에서 행을 검색할 수 있습니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*의 [SQLSetPos](../../odbc/reference/syntax/sqlsetpos-function.md) 를 참조 하십시오.
