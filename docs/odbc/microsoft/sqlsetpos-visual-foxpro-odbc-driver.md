# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 수준 2  
  
 행 집합에서 커서 위치를 설정합니다. [SQLGetData를](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 사용하여 커서를 행 집합의 특정 행에 배치한 후 바인딩되지 않은 열에서 행을 검색할 수 있습니다. **SQLSetPos**  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLSetPos](../../odbc/reference/syntax/sqlsetpos-function.md) 를 참조하십시오.
