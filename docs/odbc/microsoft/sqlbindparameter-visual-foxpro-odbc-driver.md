# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 레벨 1  
  
 버퍼를 SQL 문의 매개 변수 마커와 연결합니다. Visual FoxPro ODBC 드라이버는 *fParamType* 인수에 지정된 입력 매개 변수를 지원합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) 를 참조하십시오.
