# <a name="drop-table-statement-limitations"></a>DROP TABLE 문 제한 사항
Microsoft Excel 5.0, 7.0 또는 97 드라이버를 사용 하면 DROP TABLE 문이 워크시트를 지우지만 워크시트 이름을 삭제 하지 않습니다. 통합 문서에 워크시트 이름이 여전히 있으므로 동일한 이름을 사용 하 여 다른 워크시트를 만들 수 없습니다.
