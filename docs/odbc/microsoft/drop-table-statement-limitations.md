# <a name="drop-table-statement-limitations"></a>DROP TABLE 문 제한 사항
Microsoft Excel 5.0, 7.0 또는 97 드라이버를 사용하면 DROP TABLE 문이 워크시트를 지울 수 있지만 워크시트 이름은 삭제하지 않습니다. 워크시트 이름은 여전히 통합 북에 있으므로 동일한 이름으로 다른 워크시트를 만들 수 없습니다.
