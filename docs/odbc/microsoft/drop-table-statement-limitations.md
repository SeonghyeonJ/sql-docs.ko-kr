# <a name="drop-table-statement-limitations"></a>DROP TABLE 문 제한 사항
Microsoft Excel 97 5.0, 7.0 드라이버를 사용 하는 DROP TABLE 문 워크시트 지워지지만 워크시트 이름을 삭제 하지 않습니다. 워크시트 이름이 통합 문서에 계속 존재 하기 때문에 동일한 이름의 다른 워크시트를 만들 수 없습니다.
