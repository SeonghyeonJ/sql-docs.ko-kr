# <a name="where-clause-limitations"></a>WHERE 절 제한 사항
WHERE 절의 최대 절 수는 40입니다.  
  
 가 중에는가는 최대 255 자 길이 보다 최대 자의 리터럴과 비교할 수 있지만 매개 변수를 사용 하 여 비교할 수는 없습니다.
