# <a name="scalar-functions"></a>스칼라 함수
Microsoft ODBC 데스크톱 데이터베이스 드라이버 다음 스칼라 함수를 지원합니다.  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 인수 및 스칼라 함수의 반환 값에 대 한 자세한 참조의 부록 F를 *ODBC 프로그래머 참조입니다.*
