# <a name="sqlstate"></a>SQLSTATE
데이터 원본에서 발생 하는 오류의 경우 ODBC 드라이버는 반환 된 네이티브 오류를 적절 한 SQLSTATE에 매핑합니다. 드라이버 또는 드라이버 관리자에서 검색 된 오류의 경우 ODBC 드라이버 또는 드라이버 관리자에서 적절 한 SQLSTATE를 생성 합니다.
