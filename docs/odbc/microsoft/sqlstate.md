# <a name="sqlstate"></a>SQLSTATE
데이터 원본에서 발생 하는 오류에 대 한 ODBC 드라이버는 반환 된 원시 오류를 적절 한 SQLSTATE에 매핑됩니다. 오류에 대 한 드라이버 또는 드라이버 관리자에서 ODBC 드라이버에서 검색 되는 또는 드라이버 관리자는 적절 한 SQLSTATE를 생성 합니다.
