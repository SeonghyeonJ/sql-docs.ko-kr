# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 준비된 SQL [문(SQLPrepare에서](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)이미 준비한 명령문)을 실행합니다. 드라이버는 문안에 매개 변수가 있는 경우 매개 변수 마커 변수의 현재 값을 사용합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLExecute를](../../odbc/reference/syntax/sqlexecute-function.md) 참조하십시오.
