# <a name="drop-index-statement"></a>DROP INDEX 문
Microsoft Access, dBASE 또는 Paradox 드라이버를 사용 하는 경우 DROP INDEX 문의 구문은 "DROP INDEX a on b"입니다. 여기서 "a"는 인덱스의 이름이 고 "b"는 DROP INDEX *인덱스 이름이*아니라 테이블의 이름입니다.  
  
 Paradox 드라이버를 사용 하는 경우 DROP INDEX 문은 Paradox 보조 인덱스 파일을 삭제 합니다.  
  
 Microsoft Excel 또는 텍스트 드라이버에는 DROP INDEX 문이 지원 되지 않습니다.
