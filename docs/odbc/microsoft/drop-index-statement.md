# <a name="drop-index-statement"></a>DROP INDEX 문
Microsoft Access, dBASE 또는 역설 드라이버를 사용하는 경우 DROP INDEX 문의 구문은 "DROP INDEX a on b"이고 여기서 "a"는 인덱스의 이름이고 "b"는 테이블의 이름(DROP INDEX *인덱스 이름*아님)입니다.  
  
 역설 드라이버를 사용 하는 경우 DROP INDEX 문은 역설 보조 인덱스 파일을 삭제 합니다.  
  
 DROP INDEX 문은 Microsoft Excel 또는 텍스트 드라이버에 대해 지원되지 않습니다.
