# <a name="jet-outer-joins"></a>Jet: 외부 조인
SELECT 문에는 외부 조인 절 목록이 포함 될 수 있습니다. 중첩 된 외부 조인이 지원 됩니다. 외부 조인에 대 한 자세한 내용은 *Microsoft Jet 데이터베이스 엔진 프로그래머 가이드*를 참조 하세요.
