# <a name="sqlsetpos-desktop-database-drivers"></a>SQLSetPos(데스크톱 데이터베이스 드라이버)
**SQLSetPos** 호출에 대 한 대량 모델 의미 체계0과 *irow* 인수가 지원 됩니다.  
  
 SQL_LOCK_NO_CHANGE *fLock에*대해 지원됩니다. SQL_LOCK_EXCLUSIVE 및 SQL_LOCK_UNLOCK 지원되지 않습니다.  
  
 **SQLSetPos는** 업데이터 조인을 지원합니다. 자세한 내용은 *Microsoft Jet 데이터베이스 엔진 프로그래머 가이드를*참조하십시오.
