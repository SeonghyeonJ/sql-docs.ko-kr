# <a name="sqlsetpos-desktop-database-drivers"></a>SQLSetPos(데스크톱 데이터베이스 드라이버)
*Irow* 인수가 0 인 **SQLSetPos** 호출에 대 한 대량 모델 의미 체계가 지원 됩니다.  
  
 *FLock*에 대 한 SQL_LOCK_NO_CHANGE 지원 됩니다. SQL_LOCK_EXCLUSIVE 및 SQL_LOCK_UNLOCK 지원 되지 않습니다.  
  
 **SQLSetPos** 는 업데이트할 수 있는 조인을 지원 합니다. 자세한 내용은 *Microsoft Jet 데이터베이스 엔진 프로그래머 가이드*를 참조 하세요.
