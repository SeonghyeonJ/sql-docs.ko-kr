# <a name="sqlallocenv-visual-foxpro-odbc-driver"></a>SQLAllocEnv(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 환경 핸들, *henv에*메모리를 할당하고 응용 프로그램에서 사용할 ODBC 호출 수준 인터페이스를 초기화합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLAllocEnv를](../../odbc/reference/syntax/sqlallocenv-function.md) 참조하십시오.
