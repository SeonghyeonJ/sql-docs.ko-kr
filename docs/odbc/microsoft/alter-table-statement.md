# <a name="alter-table-statement"></a>ALTER TABLE 문
Microsoft Access, dBASE 또는 Paradoxdriver를 사용 하는 경우 ALTER TABLE 문은 add 및 drop table 제약 조건 정의 절을 사용 하 여 지원 됩니다. Microsoft Excel 또는 텍스트 드라이버에서는 ALTER TABLE 문이 지원 되지 않습니다.  
  
> [!NOTE]  
>  Borland 데이터베이스 엔진를 구현 하지 않고 Paradox 드라이버를 사용 하는 경우 ALTER TABLE 문은 지원 되지 않습니다. 읽기 및 추가 문만 사용할 수 있습니다.
