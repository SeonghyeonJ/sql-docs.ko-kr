# <a name="error-messages-odbc-driver-for-oracle"></a>오류 메시지(Oracle용 ODBC 드라이버)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거 될 예정입니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 Oracle에서 제공 하는 ODBC 드라이버를 사용 합니다.  
  
 오류가 발생 하면 Microsoft ODBC Driver for Oracle에서 SQLSTATE (ODBC 오류 코드) 및 오류 메시지를 반환 합니다. 드라이버는 드라이버에서 검색 한 오류 및 Oracle 서버에서 반환 된 오류에서이 정보를 모두 파생 시킵니다.  
  
 이 섹션에는 다음 항목이 포함 되어 있습니다.  
  
-   [Oracle용 ODBC 드라이버에서 반환한 메시지](../../odbc/microsoft/messages-returned-by-the-odbc-driver-for-oracle.md)
