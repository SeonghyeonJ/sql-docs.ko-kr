# <a name="data-truncation-detection-enabled-using-extendedansisql"></a>ExtendedAnsiSQL을 사용하여 활성화된 데이터 잘림 검색
Extendedansisql을 플래그를 설정 하 고 응용 프로그램 데이터를 삽입 하는 문자 또는 이진 열에 데이터가 잘리고, 경우 잘림을 검색 됩니다. Extendedansisql을 플래그를 해제 하 고, 이전 버전의 ODBC 데스크톱 데이터베이스 드라이버에 포함 되어 경고 없이 데이터가 잘립니다.
