# <a name="microsoft-odbc-desktop-database-drivers"></a>Microsoft ODBC 데스크톱 데이터베이스 드라이버
ODBC는 데이터베이스 액세스 언어로 구조적 쿼리 언어 (SQL)를 사용 하는 API입니다. 응용 프로그램의 소스 코드에 직접 통합 되는 동일한 ODBC 소스 코드를 사용 하 여 다양 한 Dbms (데이터베이스 관리 시스템)에 액세스할 수 있습니다. Microsoft ODBC 데스크톱 데이터베이스 드라이버를 사용 하 여 ODBC 사용 응용 프로그램 사용자는 ODBC 인터페이스를 통해 데스크톱 데이터베이스를 열고 쿼리하고 업데이트할 수 있습니다.  
  
 Microsoft ODBC 데스크톱 데이터베이스 드라이버는 Microsoft Jet 기반 ODBC 드라이버 집합입니다. Microsoft ODBC 데스크톱 데이터베이스 드라이버 2.0에는 16 비트 및 32 비트 드라이버가 모두 포함 되어 있지만 버전 3.0 이상에는 Windows 95 이상, Windows NT 워크스테이션 또는 서버 버전 4.0, Windows 2000 Professional 또는 Windows 2000 서버에서 작동 하는 32 비트 드라이버만 포함 되어 있습니다. 이러한 드라이버는 다음과 같은 유형의 데이터 원본에 대 한 액세스를 제공 합니다.  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   Paradox  
  
-   dBASE  
  
-   텍스트  
  
 Microsoft Visual FoxPro® ODBC 드라이버에 대 한 자세한 설명서는 [Visual FOXPRO Odbc 드라이버](../../odbc/microsoft/visual-foxpro-odbc-driver.md) 를 참조 하십시오.  
  
> [!NOTE]  
>  Lotus 1-2-3, Microsoft Exchange 및 HTML과 같은 다른 데이터 원본에 대 한 액세스는 설치 가능한 ISAM (IISAM) 드라이버를 통해 사용 하도록 설정 됩니다. 이러한 드라이버에 대 한 자세한 내용은 *Microsoft Jet 데이터베이스 엔진 프로그래머 참조*에서 "외부 데이터 액세스"를 참조 하세요. ODBC 데스크톱 데이터베이스 드라이버 4.0는 Btrieve 및 EMS 데이터 형식을 지원 하지 않습니다.
