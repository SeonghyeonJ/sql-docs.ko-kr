# <a name="microsoft-odbc-desktop-database-drivers"></a>Microsoft ODBC 데스크톱 데이터베이스 드라이버
ODBC는 SQL(구조화 쿼리 언어)을 데이터베이스 액세스 언어로 사용하는 API입니다. 응용 프로그램의 소스 코드에 직접 통합되는 동일한 ODBC 소스 코드로 다양한 DBMS(데이터베이스 관리 시스템)에 액세스할 수 있습니다. Microsoft ODBC 데스크톱 데이터베이스 드라이버를 사용하면 ODBC 지원 응용 프로그램의 사용자가 ODBC 인터페이스를 통해 데스크톱 데이터베이스를 열고 쿼리하고 업데이트할 수 있습니다.  
  
 마이크로 소프트 ODBC 데스크톱 데이터베이스 드라이버는 ODBC 드라이버의 마이크로 소프트 제트 기반 세트입니다. Microsoft ODBC 데스크톱 데이터베이스 드라이버 2.0에는 16비트 및 32비트 드라이버가 모두 포함되어 있는 반면, 버전 3.0 이상에는 Windows 95 이상에서 작동하는 32비트 드라이버, Windows NT 워크스테이션 또는 서버 버전 4.0, Windows 2000 Professional 또는 Windows 2000 서버가 포함됩니다. 이러한 드라이버는 다음과 같은 유형의 데이터 원본에 대한 액세스를 제공합니다.  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   역설  
  
-   Dbase  
  
-   텍스트  
  
 마이크로소프트 비주얼 [폭스프로® ODBC 드라이버에](../../odbc/microsoft/visual-foxpro-odbc-driver.md) 대한 자세한 설명서는 비주얼 폭스프로 ODBC 드라이버를 참조하십시오.  
  
> [!NOTE]  
>  로터스 1-2-3, 마이크로소프트 익스체인지 및 HTML과 같은 다른 데이터 원본에 대한 액세스는 설치 가능한 ISAM(IISAM) 드라이버에 의해 활성화됩니다. 이러한 드라이버에 대한 자세한 내용은 *Microsoft Jet 데이터베이스 엔진 프로그래머 의 참조에서*"외부 데이터 액세스"를 참조하십시오. ODBC 데스크톱 데이터베이스 드라이버 4.0은 Btrieve 및 EMS 데이터 형식을 지원하지 않습니다.
