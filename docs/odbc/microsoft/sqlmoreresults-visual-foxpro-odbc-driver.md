# <a name="sqlmoreresults-visual-foxpro-odbc-driver"></a>SQLMoreResults(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 수준 2  
  
 문 핸들, *hstmt*, SELECT, UPDATE, INSERT 또는 DELETE 문에서 보류 중인 결과가 더 있는지 여부를 확인 하 고, 해당 하는 경우 해당 결과에 대 한 처리를 초기화 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [SQLMoreResults](../../odbc/reference/syntax/sqlmoreresults-function.md) 를 참조 하세요.
