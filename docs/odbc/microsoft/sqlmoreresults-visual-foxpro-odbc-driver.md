# <a name="sqlmoreresults-visual-foxpro-odbc-driver"></a>SQLMoreResults(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 수준 2  
  
 SELECT, UPDATE, INSERT 또는 DELETE 문을 포함하는 문 핸들, *hstmt,* 에 더 많은 결과가 보류 중인지 여부를 결정하고, 그렇다면 해당 결과에 대한 처리를 초기화합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLMoreResults](../../odbc/reference/syntax/sqlmoreresults-function.md) 를 참조하십시오.
