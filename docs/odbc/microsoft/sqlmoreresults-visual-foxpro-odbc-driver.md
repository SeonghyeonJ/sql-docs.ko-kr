# <a name="sqlmoreresults-visual-foxpro-odbc-driver"></a>SQLMoreResults(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에서는 Visual FoxPro ODBC 드라이버 관련 정보를 포함합니다. 이 함수에 대 한 일반 정보에서 해당 항목을 참조 하세요 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)합니다.  
  
 지원: 전체  
  
 ODBC API 규칙: 수준 2  
  
 더 많은 결과 보류 중인지 여부를 결정 문 핸들에 대해 *hstmt*SELECT, UPDATE, INSERT 또는 DELETE 문을 포함 하 고 그렇다면 이러한 결과 대 한 처리를 초기화 합니다.  
  
 자세한 내용은 [SQLMoreResults](../../odbc/reference/syntax/sqlmoreresults-function.md) 에 *ODBC 프로그래머 참조*합니다.
