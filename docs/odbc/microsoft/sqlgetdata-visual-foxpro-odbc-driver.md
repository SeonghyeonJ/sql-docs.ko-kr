# <a name="sqlgetdata-visual-foxpro-odbc-driver"></a>SQLGetData(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 수준 1  
  
 지정 된 결과 집합의 현재 레코드에서 단일 필드의 값을 검색 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*의 [SQLGetData](../../odbc/reference/syntax/sqlgetdata-function.md) 를 참조 하세요.
