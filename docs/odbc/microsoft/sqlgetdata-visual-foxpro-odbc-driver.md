> [!NOTE]

 지원: 전체  
  
 ODBC API 규칙: 수준 1  
  
 지정 된 결과 집합의 현재 레코드에서 단일 필드의 값을 검색 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*의 [SQLGetData](../../odbc/reference/syntax/sqlgetdata-function.md) 를 참조 하세요.
