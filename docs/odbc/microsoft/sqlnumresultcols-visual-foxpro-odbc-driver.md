# <a name="sqlnumresultcols-visual-foxpro-odbc-driver"></a>SQLNumResultCols(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 코어 레벨  
  
 결과 집합 커서의 열 수를 반환합니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLNumResultCols를](../../odbc/reference/syntax/sqlnumresultcols-function.md) 참조하십시오.
