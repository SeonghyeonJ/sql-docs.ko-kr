# <a name="sorting-limitations"></a>정렬 제한 사항
그룹 BY 절, ORDER BY 절, SELECT DISTINCT 문 또는 외부 조인에서 정렬 키의 최대 길이는 255바이트입니다. 정렬 행의 모든 정렬 키의 최대 길이는 65,500바이트입니다.
