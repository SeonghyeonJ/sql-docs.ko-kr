# <a name="sorting-limitations"></a>정렬 제한 사항
GROUP BY 절, ORDER BY 절, SELECT DISTINCT 문 또는 outer join에서 정렬 키의 최대 길이는 255 바이트입니다. 정렬 행에서 모든 정렬 키의 최대 길이는 65500 바이트입니다.
