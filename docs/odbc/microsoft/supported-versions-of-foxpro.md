# <a name="supported-versions-of-foxpro"></a>지원되는 FoxPro 버전
Microsoft® Visual FoxPro® ODBC 드라이버를 사용하여 FoxPro 테이블에 저장된 데이터에 액세스할 수 있습니다. FoxPro 데이터의 다음 버전이 지원됩니다.  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   비주얼 폭스프로 (모든 버전)  
  
 Visual FoxPro에 저장된 데이터에 액세스할 때 [테이블이](../../odbc/microsoft/visual-foxpro-terminology.md) 0개 이상인 [데이터베이스](../../odbc/microsoft/visual-foxpro-terminology.md) 또는 [무료 테이블](../../odbc/microsoft/visual-foxpro-terminology.md)디렉터리에 연결하도록 선택할 수 있습니다.  
  
 데이터 원본에 연결하는 방법에 대한 자세한 내용은 [Visual FoxPro 데이터 원본 추가를](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)참조하십시오.  
  
 드라이버를 다운로드하려면 [Visual FoxPro ODBC 드라이버 다운로드 사이트를](https://go.microsoft.com/fwlink/?LinkId=121318)참조하십시오.
