# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 레벨 1  
  
 테이블의 경우 지정된 테이블 또는 테이블의 열 목록인 결과 집합을 만듭니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLColumns를](../../odbc/reference/syntax/sqlcolumns-function.md) 참조하십시오.
