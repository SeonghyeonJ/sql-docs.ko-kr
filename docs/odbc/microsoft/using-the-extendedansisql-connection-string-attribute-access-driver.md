# <a name="using-the-extendedansisql-connection-string-attribute-access-driver"></a>ExtendedAnsiSQL 연결 문자열 특성 사용(Access 드라이버)
Jet 4.0에는 ODBC 데스크톱 데이터베이스 드라이버 버전 4.0에서 기본적으로 활성화되지 않은 몇 가지 새로운 기능이 있습니다. 이러한 새로운 기능을 활용하려면 응용 프로그램에서 새 연결 특성을 설정하거나 연결 문자열에 새 특성을 추가하거나 DSN에서 특성을 설정해야 합니다. 이 특성은 ExtendedAnsiSQL이라고 하며 Microsoft ODBC 데스크톱 데이터베이스 드라이버 4.0 이상에만 관련됩니다.

-   [ExtendedAnsiSQL을 사용하여 뷰 및 프로시저 만들기 및 삭제](../../odbc/microsoft/creating-and-dropping-views-and-procedures-using-extendedansisql.md)

-   [Jet 4.0은 확장AnsiSQL 세트를 사용할 때 SQL-92 예약 된 단어 목록을 사용합니다.](../../odbc/microsoft/jet-4-0-uses-sql-92-reserved-words-list-when-extendedansisql-set.md)
