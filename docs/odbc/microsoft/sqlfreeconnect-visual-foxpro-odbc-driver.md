# <a name="sqlfreeconnect-visual-foxpro-odbc-driver"></a>SQLFreeConnect(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 연결 핸들을 해제 하 고 핸들에 할당 된 모든 메모리를 해제 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [SQLFreeConnect](../../odbc/reference/syntax/sqlfreeconnect-function.md) 를 참조 하세요.
