# <a name="using-connection-strings"></a>연결 문자열 사용
Visual FoxPro 데이터 원본에 연결할 연결 문자열을 사용할 수 있습니다.  
  
 예를 들어 TasTrade 데이터 원본 및 데이터 소스와 연결 된 전용의 현재 설정을 재정의 연결 하려면 문자열을 사용 합니다.  
  
```  
DSN=TasTrade;Exclusive=Yes  
```  
  
 특성 키워드 및 연결 문자열에 포함할 수 있습니다 하는 값의 목록을 참조 하세요 [SQLDriverConnect](../../odbc/microsoft/sqldriverconnect-visual-foxpro-odbc-driver.md)합니다.  
  
 에 대 한 전체 설명은 연결 문자열 구문, 참조 [SQLBrowseConnect](../../odbc/reference/syntax/sqlbrowseconnect-function.md) 에 *ODBC 프로그래머 참조*합니다.
