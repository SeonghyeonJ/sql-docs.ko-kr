# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>Visual FoxPro 데이터를 사용하여 Microsoft Word에서 우편 레이블 만들기
당신은 윈도우 95 또는 윈도우 98 문서에 대한 마이크로 소프트 워드에서 비주얼 FoxPro 데이터를 사용할 수 있습니다. 예를 들어 Visual FoxPro 테이블에 저장된 고객 정보에서 메일링 레이블을 만들 수 있습니다.  
  
### <a name="to-create-mailing-labels"></a>메일링 레이블을 만들려면  
  
1.  Microsoft Word에서 새 빈 문서를 만듭니다.  
  
2.  도구 메뉴에서 메일 병합을 선택합니다.  
  
3.  메일 병합 도우미에서 만들기를 선택한 다음 메일링 레이블을 선택합니다.  
  
4.  기본 문서에서 활성 창을 선택합니다.  
  
5.  데이터 원본에서 데이터 가져옵니다를 선택한 다음 데이터 원본 열기를 선택합니다.  
  
6.  데이터 원본 열기 대화 상자에서 MS 쿼리를 선택합니다.  
  
7.  데이터 원본 선택 대화 상자에서 Visual FoxPro 데이터 원본을 선택한 다음 사용을 클릭합니다.  
  
8.  데이터 원본에서 액세스하는 데이터베이스에 테이블이 포함된 경우 테이블 추가 대화 상자에서 테이블을 선택합니다. Microsoft 쿼리는 쿼리 디자이너의 위쪽 절반에 추가된 테이블을 표시합니다.  
  
9. 테이블에서 디자이너의 아래쪽 절반으로 끌어 쿼리필드를 선택합니다.  
  
10. 파일 메뉴에서 데이터를 Microsoft Word로 반환을 선택합니다. Microsoft 쿼리가 닫히고 선택한 데이터를 메일 병합 문서에서 사용할 수 있습니다.  
  
11. 기본 문서에서 설정을 선택합니다.  
  
12. 레이블 옵션 대화 상자에서 원하는 프린터 및 레이블 정보를 선택한 다음 확인을 클릭합니다.  
  
13. 레이블 만들기 대화 상자에서 메일링 레이블에 인쇄할 필드를 선택한 다음 확인을 클릭합니다.  
  
14. 메일 병합 도우미에서 문서와 데이터 병합 아래에서 병합을 클릭합니다.  
  
15. 대화 상자 병합에서 원하는 옵션을 선택한 다음 병합을 클릭합니다.
