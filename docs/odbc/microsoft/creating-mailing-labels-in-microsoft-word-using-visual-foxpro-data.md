# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>Visual FoxPro 데이터를 사용하여 Microsoft Word에서 우편 레이블 만들기
Windows 95 또는 Windows 98 문서에 대 한 Microsoft Word에서 Visual FoxPro 데이터를 사용할 수 있습니다. 예를 들어, Visual FoxPro 테이블에 저장 된 고객 정보에서 우편물 레이블을 만들 수 있습니다.  
  
### <a name="to-create-mailing-labels"></a>우편물 레이블을 만들려면  
  
1.  Microsoft Word에서 비어 있는 새 문서를 만듭니다.  
  
2.  도구 메뉴에서 메일 병합을 선택 합니다.  
  
3.  메일 병합 도우미에서 만들기를 선택 하 고 우편물 레이블을 선택 합니다.  
  
4.  주 문서에서 활성 창을 선택 합니다.  
  
5.  데이터 원본에서 데이터 가져오기를 선택한 다음 데이터 원본 열기를 선택 합니다.  
  
6.  데이터 원본 열기 대화 상자에서 MS Query를 선택 합니다.  
  
7.  데이터 소스 선택 대화 상자에서 Visual FoxPro 데이터 원본을 선택한 다음 사용을 클릭 합니다.  
  
8.  데이터 원본에 의해 액세스 되는 데이터베이스에 테이블이 포함 되어 있는 경우 테이블 추가 대화 상자에서 테이블을 선택 합니다. Microsoft Query는 쿼리 디자이너의 위쪽 절반에 추가 된 테이블을 표시 합니다.  
  
9. 테이블에서 디자이너의 아래쪽으로 끌어서 놓는 방법으로 쿼리 필드를 선택 합니다.  
  
10. 파일 메뉴에서 Microsoft Word로 데이터 반환을 선택 합니다. Microsoft 쿼리를 닫고 선택한 데이터를 메일 병합 문서에서 사용할 수 있습니다.  
  
11. 주 문서에서 설치를 선택 합니다.  
  
12. 레이블 옵션 대화 상자에서 원하는 프린터와 레이블 정보를 선택 하 고 확인을 클릭 합니다.  
  
13. 레이블 만들기 대화 상자에서 메일링 레이블에 인쇄할 필드를 선택 하 고 확인을 클릭 합니다.  
  
14. 메일 병합 도우미의 문서와 데이터 병합에서 병합을 클릭 합니다.  
  
15. 병합 대화 상자에서 원하는 옵션을 선택 하 고 병합을 클릭 합니다.
