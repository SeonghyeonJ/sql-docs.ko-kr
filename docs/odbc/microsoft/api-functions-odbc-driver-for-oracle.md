# <a name="api-functions-odbc-driver-for-oracle"></a>API 함수(Oracle용 ODBC 드라이버)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거 될 예정입니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 Oracle에서 제공 하는 ODBC 드라이버를 사용 합니다.  
  
 Oracle 용 Microsoft ODBC 드라이버는 핵심 수준, 수준 1 및 수준 2 API 함수를 지원 합니다. 이러한 함수는 [ODBC 규칙 수준](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)에 나열 됩니다.  
  
-   CLI ( [핵심 수준 인터페이스](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) ) 규칙은 ISO CLI 사양에 정의 된 기능과 오픈 그룹 CLI 사양에 정의 된 필수 기능을 제공 합니다.  
  
-   [수준 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) 규칙은 트랜잭션과 같은 추가 기능 뿐만 아니라 핵심 수준 인터페이스 기능을 제공 합니다.  
  
-   [수준 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) 규칙은 수준 1 기능 뿐만 아니라 책갈피, 동적 매개 변수 및 ODBC 함수의 비동기 실행과 같은 추가 기능을 제공 합니다.
