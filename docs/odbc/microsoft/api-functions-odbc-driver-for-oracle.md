# <a name="api-functions-odbc-driver-for-oracle"></a>API 함수(Oracle용 ODBC 드라이버)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 오라클에서 제공하는 ODBC 드라이버를 사용합니다.  
  
 오라클용 Microsoft ODBC 드라이버는 코어 레벨, 레벨 1 및 레벨 2 API 기능을 지원합니다. 이러한 함수는 [ODBC 적합성 수준에](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)나열됩니다.  
  
-   [코어 레벨](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) 인터페이스(CLI) 적합성은 ISO CLI 사양에 정의된 기능과 개방형 그룹 CLI 사양에 정의된 필수 기능을 제공합니다.  
  
-   [수준 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) 준수는 핵심 수준 인터페이스 기능뿐만 아니라 트랜잭션과 같은 추가 기능을 제공합니다.  
  
-   [수준 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) 준수는 레벨 1 기능뿐만 아니라 북마크, 동적 매개 변수 및 ODBC 함수의 비동기 실행과 같은 추가 기능을 제공합니다.
