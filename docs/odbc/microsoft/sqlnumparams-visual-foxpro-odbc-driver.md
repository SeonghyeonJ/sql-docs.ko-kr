# <a name="sqlnumparams-visual-foxpro-odbc-driver"></a>SQLNumParams(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 수준 2  
  
 SQL 문의 매개 변수 수를 반환합니다. 매개 변수 의 수는 [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)에 전달 된 SQL 문의 물음표의 수와 같아야 합니다.  
  
 SQL 문법에 대한 자세한 내용은 [지원되는 ODBC SQL 문법](../../odbc/microsoft/supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md)을 참조하십시오. 이 함수에 대한 자세한 내용은 *ODBC 프로그래머 의 참조에서* [SQLNumParams](../../odbc/reference/syntax/sqlnumparams-function.md) 을 참조하십시오.
