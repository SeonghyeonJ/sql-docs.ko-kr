# <a name="sqlgetfunctions-visual-foxpro-odbc-driver"></a>SQLGetFunctions(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 레벨 1  
  
 지원되는 모든 함수에 대해 TRUE를 반환합니다.  
  
 비주얼 FoxPro ODBC 드라이버는 모든 ODBC API 코어 및 레벨 1 기능을 지원합니다. 다음 표는 드라이버가 특정 수준 2 기능을 지원하는지 여부를 나타냅니다.  
  
|*기능*|지원됨|  
|----------------|---------------|  
|SQL_API_SQLBROWSECONNECT|예|  
|SQL_API_SQLCOLUMNPRIVELEGES|예|  
|SQL_API_SQLDATASOURCES|예|  
|SQL_API_SQLDESCRIBEPARAM|예|  
|SQL_API_SQLDRIVERS|예|  
|SQL_API_SQLEXTENDEDFETCH|예|  
|SQL_API_SQLFOREIGNKEYS|예|  
|SQL_API_SQLMORERESULTS|예|  
|SQL_API_SQLNATIVESQL|예|  
|SQL_API_SQLNUMPARAMS|예|  
|SQL_API_SQLPARAMOPTIONS|예|  
|SQL_API_SQLPRIMARYKEYS|예|  
|SQL_API_SQLPROCEDURECOLUMNS|예|  
|SQL_API_SQLPROCEDURES|예|  
|SQL_API_SQLSETPOS|예|  
|SQL_API_SQLSETSCROLLOPTIONS|예|  
|SQL_API_SQLTABLEPRIVILEGES|예|  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLGetFunctions를](../../odbc/reference/syntax/sqlgetfunctions-function.md) 참조하십시오.
