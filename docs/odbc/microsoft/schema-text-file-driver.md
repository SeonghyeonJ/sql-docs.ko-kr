# <a name="schema-text-file-driver"></a>스키마(텍스트 파일 드라이버)
스키마에는 테이블 형식, 열 형식을 결정하기 위해 스캔할 행 수, 테이블의 첫 번째 행에 열 이름이 포함되어 있는지 여부, OEM 또는 ANSI 코드 페이지를 사용하여 원본 파일을 작성했는지 여부, 각 열의 이름, 데이터 형식 및 너비를 비롯한 데이터 원본의 각 테이블(텍스트 파일)에 대한 정보가 포함됩니다.
