# <a name="schema-text-file-driver"></a>스키마(텍스트 파일 드라이버)
이 스키마에는 테이블의 형식, 열 유형을 결정 하기 위해 검색할 행 수, 테이블의 첫 행에 열 이름이 포함 되는지 여부, OEM 또는 ANSI 코드 페이지를 사용 하 여 원본 파일을 쓸지 여부, 각 열의 이름, 데이터 형식 및 너비를 비롯 하 여 데이터 원본의 각 테이블 (텍스트 파일)에 대 한 정보가 포함 됩니다.
