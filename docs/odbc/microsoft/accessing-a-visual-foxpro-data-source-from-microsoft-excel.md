# <a name="accessing-a-visual-foxpro-data-source-from-microsoft-excel"></a>Microsoft Excel에서 Visual FoxPro 데이터 원본에 액세스
Microsoft Query를 설치한 경우 Visual FoxPro 데이터에 연결 하는 데이터 원본을 Microsoft Excel에서 만들 수 있습니다.  
  
### <a name="to-access-visual-foxpro-data-from-microsoft-excel"></a>Microsoft Excel에서 Visual FoxPro 데이터에 액세스 하려면  
  
1.  Microsoft Excel 스프레드시트를 엽니다.  
  
2.  데이터 메뉴에서 외부 데이터 가져오기를 선택 합니다. Microsoft Query가 열립니다.  
  
3.  데이터 소스 선택 대화 상자에서 기타를 클릭 합니다.  
  
4.  ODBC 데이터 원본 대화 상자에서 새로 만들기를 클릭 합니다.  
  
5.  데이터 소스 추가 대화 상자의 설치 된 ODBC 드라이버 목록 상자에서 Microsoft Visual FoxPro 드라이버를 선택 하 고 확인을 클릭 합니다.  
  
6.  [ODBC Visual FoxPro 설정 대화 상자](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md)에서 데이터 원본 이름을 입력 하 고 데이터베이스 유형을 선택한 다음 데이터베이스 또는 디렉터리의 경로를 입력 하 고 확인을 클릭 합니다.  
  
     ODBC 데이터 원본 대화 상자의 데이터 원본 입력 텍스트 상자에 새 데이터 원본 이름이 표시 됩니다.  
  
7.  확인을 클릭합니다.  
  
     데이터 원본 선택 대화 상자의 사용 가능한 데이터 원본 텍스트 상자에서 새 데이터 원본 이름이 선택 됩니다.  
  
8.  사용을 클릭합니다.
