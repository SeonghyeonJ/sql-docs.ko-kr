# <a name="accessing-a-visual-foxpro-data-source-from-microsoft-excel"></a>Microsoft Excel에서 Visual FoxPro 데이터 원본에 액세스
Microsoft 쿼리를 설치한 경우 Visual FoxPro 데이터에 연결하는 데이터 원본을 Microsoft Excel에서 만들 수 있습니다.  
  
### <a name="to-access-visual-foxpro-data-from-microsoft-excel"></a>마이크로 소프트 엑셀에서 비주얼 폭스 프로 데이터에 액세스하려면  
  
1.  Microsoft Excel 스프레드시트를 엽니다.  
  
2.  데이터 메뉴에서 외부 데이터 받기를 선택합니다. Microsoft 쿼리가 열립니다.  
  
3.  데이터 원본 선택 대화 상자에서 기타를 클릭합니다.  
  
4.  ODBC 데이터 원본 대화 상자에서 새로 를 클릭합니다.  
  
5.  데이터 원본 추가 대화 상자에서 설치된 ODBC 드라이버 목록 상자에서 Microsoft Visual FoxPro 드라이버를 선택하고 확인을 클릭합니다.  
  
6.  [ODBC Visual FoxPro 설치 대화 상자에서](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md)데이터 원본 이름을 입력하고, 데이터베이스 유형을 선택하고, 데이터베이스 또는 디렉터리로 경로를 입력하고, 확인을 클릭합니다.  
  
     새 데이터 원본 이름은 ODBC 데이터 원본 대화 상자의 데이터 원본 텍스트 입력 상자에 표시됩니다.  
  
7.  확인을 클릭합니다.  
  
     새 데이터 원본 이름은 데이터 원본 선택 대화 상자의 사용 가능한 데이터 원본 텍스트 상자에서 선택됩니다.  
  
8.  사용을 클릭합니다.  
  
 이제 열려 있는 쿼리에 테이블을 추가할 수 있습니다. 쿼리 빌드에 대한 자세한 내용은 [Visual FoxPro 데이터베이스에서 Microsoft Excel로 데이터 가져오기](../../odbc/microsoft/importing-data-into-microsoft-excel-from-a-visual-foxpro-database.md)를 참조하십시오.
