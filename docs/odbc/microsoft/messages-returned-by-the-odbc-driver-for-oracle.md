# <a name="messages-returned-by-the-odbc-driver-for-oracle"></a>Oracle용 ODBC 드라이버에서 반환한 메시지
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. 대신 오라클에서 제공하는 ODBC 드라이버를 사용합니다.  
  
 오라클 오류 메시지를 사용할 수 있는 경우 [Microsoft], [오라클용 ODBC 드라이버], [오라클] 태그 앞에 반환됩니다. 그렇지 않으면 다음 예제와 같이 [Oracle] 태그 없이 메시지가 반환됩니다.  
  
## <a name="oracle-error-message"></a>오라클 오류 메시지:  
  
```  
[Microsoft][ODBC driver for Oracle][Oracle]ORA-nnnnn message-text  
```  
  
## <a name="odbc-driver-for-oracle-error-message"></a>오라클 오류 메시지에 대한 ODBC 드라이버 :  
  
```  
[Microsoft][ODBC driver for Oracle]  
```
