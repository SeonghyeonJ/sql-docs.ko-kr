# <a name="numeric-functions-visual-foxpro-odbc-driver"></a>숫자 함수(Visual FoxPro ODBC 드라이버)

|ODBC 문법|비주얼 폭스프로 문법|  
|------------------|---------------------------|  
|ABS *(numeric_exp)*||  
|ACOS *(float_exp)*||  
|아신 *(float_exp)*||  
|아탄 *(float_exp)*||  
|아탄2 *(float_exp1, float_exp2)*|ATN2 *(float_exp1, float_exp2)*|  
|*천장(numeric_exp)*||  
|코스 *(float_exp)*||  
|COT *(float_exp)*||  
|도 *(numeric_exp)*|RTOD *(numeric_exp)*|  
|전황 *(float_exp)*||  
|*플로어(numeric_exp)*||  
|*로그(float_exp)*||  
|LOG10 *(float_exp)*||  
|MOD *(integer_exp1, integer_exp2)*||  
|PI *( )*||  
|라디안 *(numeric_exp)*|DTOR *(numeric_exp)*|  
|*랜드([integer_exp])*||  
|라운드 *(numeric_exp, integer_exp)*||  
|기호 *(numeric_exp)*||  
|죄 *(float_exp)*||  
|SQRT *(float_exp)*||  
|TAN *(float_exp)*||
