# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures(데스크톱 데이터베이스 드라이버)
**Sqlprocedures** 은 인수를 하나 이상 포함 하는 프로시저에 대 한 행만 반환 합니다. 인수가 없는 프로시저는 뷰로 처리 됩니다.  
  
|열|주석|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|데이터베이스 파일의 경로입니다.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|구분 기호로 분리 된 프로시저 이름|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
