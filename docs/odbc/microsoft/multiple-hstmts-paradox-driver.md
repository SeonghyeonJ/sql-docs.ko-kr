# <a name="multiple-hstmts-paradox-driver"></a>다중 hstmts(Paradox 드라이버)
ODBC Paradox 드라이버를 사용 하는 경우 두 개 이상의 *hstmt* 를 사용 하 여 테이블에 대 한 쿼리를 실행 하려면 테이블에 고유 인덱스 (Paradox 기본 키)가 있어야 합니다.
