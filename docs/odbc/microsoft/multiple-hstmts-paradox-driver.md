# <a name="multiple-hstmts-paradox-driver"></a>다중 hstmts(Paradox 드라이버)
둘 이상의 사용 하려는 경우, ODBC Paradox 드라이버 되 *hstmt* 테이블에서 쿼리를 실행 하려면 테이블에는 고유 인덱스 (Paradox 기본 키) 있어야 합니다.
