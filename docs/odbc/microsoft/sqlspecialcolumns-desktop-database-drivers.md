# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns(데스크톱 데이터베이스 드라이버)
*Fcoltype*의 SQL_BEST_ROWID 플래그에 대해 고유 인덱스가 반환 됩니다 (있는 경우). SQL_ROWVER 플래그에 대해 결과 집합이 반환 되지 않습니다.  
  
 모든 행 Id의 범위는 SQL_SCOPE_CURROW입니다.  
  
 *Sztablequalifier* 또는 *sztablename* 인수에는 패턴 일치가 지원 되지 않습니다.
