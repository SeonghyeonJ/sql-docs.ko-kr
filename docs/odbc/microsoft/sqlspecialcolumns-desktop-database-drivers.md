# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns(데스크톱 데이터베이스 드라이버)
*fColType에서*SQL_BEST_ROWID 플래그에 대한 고유 인덱스가 반환됩니다(있는 경우). SQL_ROWVER 플래그에 대한 결과 집합이 반환되지 않습니다.  
  
 모든 행 아이디의 범위는 SQL_SCOPE_CURROW.  
  
 패턴 일치는 *szTableQualifier* 또는 *szTableName* 인수에 대해 지원되지 않습니다.
