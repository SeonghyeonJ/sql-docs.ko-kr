# <a name="sqlputdata-visual-foxpro-odbc-driver"></a>SQLPutData(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함 되어 있습니다. 이 함수에 대 한 일반 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절 한 항목을 참조 하세요.  
  
 지원: 전체  
  
 ODBC API 규칙: 수준 1  
  
 응용 프로그램에서 문 실행 시 매개 변수 또는 열에 대 한 데이터를 드라이버에 보낼 수 있도록 합니다.  
  
 자세한 내용은 *ODBC 프로그래머 참조*에서 [sqlputdata](../../odbc/reference/syntax/sqlputdata-function.md) 를 참조 하세요.
