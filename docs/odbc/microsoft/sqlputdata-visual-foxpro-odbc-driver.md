# <a name="sqlputdata-visual-foxpro-odbc-driver"></a>SQLPutData(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에는 Visual FoxPro ODBC 드라이버 관련 정보가 포함되어 있습니다. 이 함수에 대한 일반적인 정보는 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)에서 적절한 항목을 참조하십시오.  
  
 지원: 전체  
  
 ODBC API 적합성: 레벨 1  
  
 응용 프로그램이 명령문 실행 시간에 매개 변수 또는 열에 대한 데이터를 드라이버에 보낼 수 있습니다.  
  
 자세한 내용은 *ODBC 프로그래머의 참조에서* [SQLPutData](../../odbc/reference/syntax/sqlputdata-function.md) 를 참조하십시오.
