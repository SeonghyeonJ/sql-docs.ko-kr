# <a name="sqlgetcursorname-desktop-database-drivers"></a>SQLGetCursorName(데스크톱 데이터베이스 드라이버)
**SQLGetCursorName** 는 지원 되지만 커서 라이브러리가 사용 되는 경우에만 사용할 수 있습니다. 위치 지정 작업은 드라이버에서 지원 되지 않기 때문입니다.
