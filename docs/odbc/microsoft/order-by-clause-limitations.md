# <a name="order-by-clause-limitations"></a>ORDER BY 절 제한 사항
SELECT 문의 GROUP BY 절 및 ORDER BY 절에 있으면 ORDER BY 절은 결과 집합의 열만 또는 GROUP BY 절에 식을 포함할 수 있습니다.
