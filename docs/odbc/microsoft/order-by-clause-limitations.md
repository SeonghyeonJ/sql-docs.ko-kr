# <a name="order-by-clause-limitations"></a>ORDER BY 절 제한 사항
SELECT 문에 GROUP BY 절과 ORDER BY 절이 포함 된 경우 ORDER BY 절은 결과 집합의 열 이나 GROUP BY 절의 식만 포함할 수 있습니다.
