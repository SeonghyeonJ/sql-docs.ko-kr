# <a name="index-name-limitations"></a>인덱스 이름 제한 사항
Paradox 드라이버를 사용 하는 경우 기본 인덱스의 이름이 정의 된 테이블과 동일 해야 합니다. 다른 고유 또는 고유 하지 않은 인덱스는 정의 된 테이블의 이름과 같아야 합니다.
