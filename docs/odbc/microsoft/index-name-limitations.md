# <a name="index-name-limitations"></a>인덱스 이름 제한 사항
Paradox 드라이버를 사용할 때 기본 인덱스는 정의된 테이블과 이름이 같아야 합니다. 다른 고유 또는 고유하지 않은 인덱스는 정의된 테이블과 이름이 같아야 합니다.
