# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 드라이버 설정
Visual FoxPro ODBC 드라이버를 다운로드하고 설치하는 방법에 대한 자세한 내용은 MSDN 다운로드의 [Visual FoxPro ODBC 드라이버를](https://go.microsoft.com/fwlink/?LinkId=121318) 참조하십시오.  
  
 Microsoft Visual FoxPro ODBC 드라이버 설정 프로그램을 사용하여 다음을 수행합니다.  
  
-   새 구성요소를 추가합니다.  
  
-   설치된 구성 요소를 제거합니다.  
  
-   누락된 파일 및 설정을 복원하려면 다시 설치합니다.  
  
-   이전에 설치한 모든 구성 요소를 제거합니다.  
  
 시스템에 드라이버를 설치한 후 설치 프로그램은 설치된 드라이버 구성 요소를 인식하고 드라이버 구성을 변경할 수 있는 추가 대화 상자를 제공합니다.
