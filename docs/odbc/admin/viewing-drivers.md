# <a name="viewing-drivers"></a>드라이버 보기
데이터 원본을 구성하려면 시스템에 하나 이상의 드라이버를 설치해야 합니다. **ODBC 데이터 원본 관리자** 대화 상자의 **드라이버** 탭에는 각 드라이버의 이름, 버전, 회사, 파일 이름 및 파일 생성 날짜를 포함하여 컴퓨터에 설치된 모든 드라이버가 나열됩니다. 시스템에서 드라이버를 추가하거나 삭제하려면 드라이버의 설치 프로그램을 사용해야 합니다. 드라이버 수정에 대한 자세한 내용은 [데이터 원본 관리를](../../odbc/admin/managing-data-sources.md)참조하십시오.
