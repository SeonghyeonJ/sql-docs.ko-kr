# <a name="viewing-drivers"></a>드라이버 보기
데이터 원본을 구성 하려면 시스템에 드라이버를 하나 이상 설치 해야 합니다. **ODBC 데이터 원본 관리자** 대화 상자의 **드라이버** 탭에는 각 드라이버의 이름, 버전, 회사, 파일 이름 및 파일 생성 날짜를 비롯 하 여 컴퓨터에 설치 된 모든 드라이버가 나열 됩니다. 드라이버의 설치 프로그램을 사용 하 여 시스템에서 드라이버를 추가 하거나 삭제 해야 합니다. 드라이버를 수정 하는 방법에 대 한 자세한 내용은 [데이터 원본 관리](../../odbc/admin/managing-data-sources.md)를 참조 하세요.
