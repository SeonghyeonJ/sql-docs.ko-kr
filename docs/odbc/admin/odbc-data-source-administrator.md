# <a name="odbc-data-source-administrator"></a>ODBC 데이터 원본 관리자
Microsoft® ODBC 데이터 원본 관리자는 데이터베이스 드라이버 및 데이터 원본을 관리합니다. 이 응용 프로그램은 **관리 도구**아래의 Windows 제어판에 있습니다. Windows 8에서 시작하여 아이콘은 **ODBC 데이터 소스로**명명되며 64비트 운영 체제에는 32비트 및 64비트 버전이 있습니다.  
  
 자세한 ODBC 관리자 절차에 대한 자세한 내용을 보려면 [ODBC 데이터 원본 관리자](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333) 대화 상자를 열고 **도움말을**클릭합니다.  
  
 PowerShell 명령을 사용하여 드라이버 및 데이터 원본을 관리할 수 있습니다. 이러한 PowerShell 명령에 대한 자세한 내용은 [Windows 데이터 액세스 구성 요소 PowerShell 명령을](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)참조하십시오.
