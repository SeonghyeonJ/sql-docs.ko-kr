# <a name="odbc-data-source-administrator"></a>ODBC 데이터 원본 관리자
Microsoft® ODBC 데이터 원본 관리자 데이터베이스 드라이버 및 데이터 소스를 관리합니다. 이 응용 프로그램에서 Windows 제어판에 위치한 **관리 도구**합니다. Windows 8 부터는 아이콘 라고 **ODBC 데이터 원본**, 64 비트 운영 체제는 32 비트 및 64 비트 버전 및 합니다.  
  
 자세한 ODBC 관리자 절차에 대 한 내용은 엽니다는 [ODBC 데이터 원본 관리자](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333) 대화 상자를 클릭 **도움말**합니다.  
  
 드라이버 및 데이터 원본을 관리할 PowerShell 명령을 사용할 수 있습니다. 다음 PowerShell 명령에 대 한 자세한 내용은 참조 하세요. [Windows 데이터 액세스 구성 요소 PowerShell 명령을](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)합니다.
