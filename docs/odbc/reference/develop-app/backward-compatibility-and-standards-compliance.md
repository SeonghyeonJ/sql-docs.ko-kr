# <a name="backward-compatibility-and-standards-compliance"></a>이전 버전과의 호환성 및 표준 준수
이전 버전과의 호환성은 최신 ODBC 구성 요소가 이전 ODBC 구성 요소와 함께 작동할 수 있는 기능입니다. 다음 섹션에서는 ODBC *3.x.* 이 정보에 포함된 정보는 주로 ODBC *3.x* 응용 프로그램의 작성과 ODBC 드라이버에서 이전 버전과의 호환성 문제를 처리하는 방법을 다룹니다. 이전 버전과의 호환성 문제가 ODBC *3.x* 드라이버 작성에 미치는 영향에 대한 구체적인 지침은 [부록 G: 이전 버전과의 호환성을 위한 드라이버 지침 참조](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)
