# <a name="backward-compatibility-and-standards-compliance"></a>이전 버전과의 호환성 및 표준 준수
이전 버전과 호환성은 이전 ODBC 구성 요소를 사용 하는 기능 최신 ODBC 구성 요소입니다. 다음 섹션에서는 이러한 구성 요소에 주는 영향 ODBC의 변경 내용을 논의 *3.x*합니다. 기본적으로 포함 된 정보를 작성 하는 ODBC 주소 *3.x* 응용 프로그램 및 ODBC 드라이버에서 호환성 문제는 처리 하는 뒤로 하는 방법입니다. 어떻게 이전 버전과 호환성에 대 한 특정 지침은 문제로 영향을 받는 ODBC 작성할 *3.x* 드라이버 참조 [부록 g: 이전 버전과 호환성에 대 한 드라이버 지침](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)합니다.
