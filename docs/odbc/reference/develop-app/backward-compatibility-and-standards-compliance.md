# <a name="backward-compatibility-and-standards-compliance"></a>이전 버전과의 호환성 및 표준 준수
이전 버전과의 호환성은 이전 ODBC 구성 요소와 함께 작동 하는 최신 ODBC 구성 요소의 기능입니다. 다음 섹션에서는 이러한 구성 *요소가 ODBC 3.x*의 변경에 의해 영향을 받는 방법에 대해 설명 합니다. 여기에 포함 된 정보는 주로 ODBC *2.x 응용 프로그램* 작성 및 이전 버전과의 호환성 문제가 odbc 드라이버에 의해 처리 되는 방식을 다룹니다. 이전 버전과의 호환성 *문제가 ODBC 3.x* 드라이버 작성에 영향을 주는 방법에 대 한 구체적인 지침은 [부록 G: 이전 버전과의 호환성을 위한 드라이버 지침](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)을 참조 하세요.
