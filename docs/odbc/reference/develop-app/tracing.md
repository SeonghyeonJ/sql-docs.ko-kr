# <a name="tracing"></a>추적
ODBC 드라이버 관리자에는 ODBC 응용 프로그램에서 만든 함수 호출 시퀀스를 기록하고 로그 파일로 기록할 수 있는 추적 기능이 있습니다. 추적은 응용 프로그램과 드라이버 관리자 간의 호출을 캡처하는 추적 DLL과 드라이버 관리자와 드라이버 간의 호출을 캡처하여 수행됩니다. 이 추적 방법은 ODBC 2 *.x* 드라이버 관리자가 수행한 추적과 ODBC 스파이가 ODBC 2 *.x에서* 수행한 추적을 대체합니다.
