# <a name="tracing"></a>추적
ODBC 드라이버 관리자에는 ODBC 응용 프로그램에서 수행 하는 함수 호출 시퀀스를 기록 하 고 로그 파일에 transcribed 수 있는 추적 기능이 있습니다. 추적은 응용 프로그램 및 드라이버 관리자와 드라이버 관리자와 드라이버 사이에서 호출을 캡처하는 추적 DLL에 의해 수행 됩니다. 이 추적 메서드는 odbc 1.x 드라이버 관리자*에 의해 수행* 된 추적과 odbc Spy를 통해 odbc 2.x에서 수행 된 추적을*대체 합니다.*
