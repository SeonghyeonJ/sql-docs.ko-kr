# <a name="enabling-visual-studio-analyzer"></a>Visual Studio 분석기 사용
> [!IMPORTANT]  
>  비주얼 스튜디오 분석기지원은 Windows 8부터 제거되었습니다(Visual Studio 분석기는 이전 버전의 Visual Studio에만 포함되었습니다.) 대체 문제 해결 메커니즘을 보려면 BID 추적을 사용합니다.  
  
 Microsoft Visual Studio Analyzer는 모든 계층 및 시스템에서 응용 프로그램의 성능을 수준 높도록 설계된 응용 프로그램 분석 도구입니다. 구성 요소 간의 상호 작용에 중점을 둡니다. 시각적 스튜디오 분석기는 ODBC 관리자의 추적 탭에서 시작하고 중지할 수 있습니다.  
  
 ODBC가 생성하는 이벤트를 보려면 다음 단계를 따르십시오.  
  
1.  ODBC 데이터 원본 관리자에 액세스합니다.  
  
2.  **추적** 탭을 클릭합니다.  
  
3.  **비주얼 스튜디오 분석기 추적 시작을**클릭합니다.  
  
4.  **확인**을 클릭합니다.  
  
 비주얼 스튜디오 분석기 이벤트 생성은 **중지 시각적 스튜디오 분석기 추적이** 선택될 때까지 계속됩니다.  
  
 이 섹션에는 다음 항목이 포함되어 있습니다.
