# <a name="enabling-visual-studio-analyzer"></a>Visual Studio 분석기 사용
> [!IMPORTANT]  
>  Visual Studio 분석기에 대 한 지원은 Windows 8부터 제거 되었습니다 (Visual Studio 분석기 이전 버전의 Visual Studio에만 포함 됨). 다른 문제 해결 메커니즘을 사용 하려면 입찰 추적을 사용 합니다.  
  
 Microsoft Visual Studio Analyzer는 모든 계층 및 시스템에서 응용 프로그램의 성능에 대 한 개략적인 개요를 제공 하도록 설계 된 응용 프로그램 분석 도구입니다. 구성 요소 간의 상호 작용에 초점을 맞춘 것입니다. ODBC 관리자의 추적 탭에서 Visual Studio 분석기를 시작 하 고 중지할 수 있습니다.  
  
 ODBC에서 생성 하는 이벤트를 보려면 다음 단계를 수행 합니다.  
  
1.  ODBC 데이터 원본 관리자에 액세스 합니다.  
  
2.  **추적** 탭을 클릭 합니다.  
  
3.  **Visual Studio 분석기 추적 시작**을 클릭 합니다.  
  
4.  **확인**을 클릭합니다.  
  
 Visual Studio 분석기 이벤트 생성은 **Visual Studio 분석기 추적 중지** 가 선택 될 때까지 계속 됩니다.  
  
 이 섹션에는 다음 항목이 포함 되어 있습니다.  
  
-   [ODBC 드라이버 관리자에 의해 생성된 이벤트](../../../odbc/reference/develop-app/events-generated-by-the-odbc-driver-manager.md)
