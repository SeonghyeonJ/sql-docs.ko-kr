# <a name="choosing-a-level-of-interoperability"></a>상호 운용성 수준 선택
응용 프로그램이 ODBC를 사용한다는 가정하에 다음 단계는 필요한 상호 운용성 수준을 결정하는 것입니다. 기본 수준의 상호 운용성은 일반적으로 응용 프로그램 유형의 함수입니다: 사용자 지정 응용 프로그램은 상호 운용되지 않는 경향이 있고, 수직 응용 프로그램은 제한된 수의 DBMS 간에 상호 운용되는 경향이 있으며, 일반 응용 프로그램은 모든 DBMS 간에 상호 운용되는 경향이 있습니다.
