# <a name="choosing-a-level-of-interoperability"></a>상호 운용성 수준 선택
응용 프로그램에서 ODBC를 사용 하는 경우 다음 단계는 필요한 상호 운용성 수준을 결정 하는 것입니다. 상호 운용성의 기본 수준은 일반적으로 응용 프로그램 종류의 기능입니다. 사용자 지정 응용 프로그램은 상호 운용할 수 없고, 수직 응용 프로그램은 제한 된 수의 Dbms 간에 상호 운용이 가능 하 고, 제네릭 응용 프로그램은 모든 Dbms에서 상호 운용할 수 있는 경향이 있습니다.
