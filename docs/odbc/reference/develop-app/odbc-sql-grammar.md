# <a name="odbc-sql-grammar"></a>ODBC SQL 문법
상호 운용 가능한 응용 프로그램은 항상 SQL 문에서 ODBC SQL 문법을 사용해야 합니다. 그러나 이 문법 내에서도 상당한 변형이 가능합니다. 자세한 내용은 [SQL 문의 상호 운용성을](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md)참조하십시오.
