# <a name="transactions-in-odbc-odbc"></a>ODBC의 트랜잭션
ODBC의 트랜잭션은 연결 수준에서 완료 됩니다. 즉, 응용 프로그램이 트랜잭션을 완료 하면 해당 연결에서 모든 문 핸들을 통해 수행 된 모든 작업을 커밋하거나 롤백합니다.
