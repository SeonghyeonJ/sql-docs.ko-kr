# <a name="parameter-markers-in-procedure-calls"></a>프로시저 호출의 매개 변수 표식
매개 변수를 허용 하는 프로시저를 호출할 때 상호 운용 가능한 응용 프로그램은 리터럴 매개 변수 값 대신 매개 변수 표식을 사용 해야 합니다. 일부 데이터 원본은 프로시저 호출에서 리터럴 매개 변수 값을 사용 하는 것을 지원 하지 않습니다. 매개 변수에 대 한 자세한 내용은 [문 매개 변수](../../../odbc/reference/develop-app/statement-parameters.md)를 참조 하세요. 프로시저 호출에 대 한 자세한 내용은이 섹션의 뒷부분에 나오는 [프로시저 호출](../../../odbc/reference/develop-app/procedure-calls.md)을 참조 하세요.
