# <a name="parameter-markers-in-procedure-calls"></a>프로시저 호출의 매개 변수 표식
매개 변수를 허용하는 프로시저를 호출할 때 상호 운용 가능한 응용 프로그램은 리터럴 매개 변수 값 대신 매개 변수 마커를 사용해야 합니다. 일부 데이터 원본은 프로시저 호출에서 리터럴 매개 변수 값의 사용을 지원하지 않습니다. 매개 변수에 대한 자세한 내용은 [문 매개 변수 를](../../../odbc/reference/develop-app/statement-parameters.md)참조하십시오. 호출 절차에 대한 자세한 내용은 이 섹션의 다음 부분의 [프로시저 호출을](../../../odbc/reference/develop-app/procedure-calls.md)참조하십시오.
