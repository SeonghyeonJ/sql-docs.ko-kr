# <a name="comparing-bookmarks"></a>책갈피 비교
책갈피 바이트 호환 이기 때문에 같음 또는 같지 않음을 비교할 수 있습니다. 이렇게 하려면 응용 프로그램 각 책갈피를 바이트 배열로 취급 하 고 두 책갈피에서 바이트를 비교 합니다. 책갈피 결과 집합 내 에서만 고유 되도록 보장 되므로 때문에 값은 다양 한 결과 집합에서 얻은 책갈피를 비교할 의미가 없습니다.
