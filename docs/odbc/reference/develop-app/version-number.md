# <a name="version-number"></a>버전 번호
ODBC에는 여러 버전이 있으며 각 버전은 서로 다른 기능을 가지고 있습니다. 응용 프로그램은 SQL_ODBC_VER 및 SQL_DRIVER_ODBC_VER 옵션을 사용하여 **SQLGetInfo를** 호출하여 드라이버 관리자와 특정 드라이버 지원을 어떤 ODBC 버전인지 결정합니다.
