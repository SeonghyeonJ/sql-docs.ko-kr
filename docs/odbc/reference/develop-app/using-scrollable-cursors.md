# <a name="using-scrollable-cursors"></a>스크롤 가능 커서 사용
스크롤 가능 커서를 사용 하 여 이러한 세 가지 단계가 필요 합니다.  
  
1.  커서 기능을 확인 합니다.  
  
2.  커서를 설정 합니다.  
  
3.  행 스크롤 및 인출 합니다.
