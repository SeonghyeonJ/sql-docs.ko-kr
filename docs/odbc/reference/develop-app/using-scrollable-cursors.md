# <a name="using-scrollable-cursors"></a>스크롤 가능 커서 사용
스크롤 가능 커서를 사용 하려면 다음 세 단계가 필요 합니다.  
  
1.  커서 기능을 결정 합니다.  
  
2.  커서를 설정 합니다.  
  
3.  행을 스크롤하고 페치합니다.
