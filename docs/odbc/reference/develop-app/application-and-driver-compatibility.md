# <a name="application-and-driver-compatibility"></a>애플리케이션 및 드라이버 호환성
ODBC 응용 프로그램 및 드라이버는 버전 외에도 여러 범주에 속합니다. 이러한 응용 프로그램 중 일부는 일부 드라이버와 호환되지 않습니다. 다른 경우에는 응용 프로그램 또는 드라이버의 유형이 이전 버전과의 호환성 문제에 영향을 줄 수 있습니다.
