# <a name="features-to-watch-for"></a>주목할 기능
이 섹션에서는 응용 프로그램 개발자가 당연하게 여기는 여러 가지 기능에 대해 설명합니다. 실제로 이러한 기능은 DBMS 간의 지원 및 지원 방식에 따라 매우 다양합니다. 코딩하지 못하면 상호 운용 가능한 응용 프로그램에서 문제가 발생할 수 있습니다.  
  
 이 섹션에는 응용 프로그램 개발자가 고려해야 하는 모든 기능이 나열되어 있지는 않습니다. 이 정보는 [SQLGetInfo,](../../../odbc/reference/syntax/sqlgetinfo-function.md) [SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md)및 [SQLSetConnectAttr](../../../odbc/reference/syntax/sqlsetconnectattr-function.md) 함수 설명, [부록 C: SQL 문법](../../../odbc/reference/appendixes/appendix-c-sql-grammar.md)및 각 기능에 대해 설명하는 이 설명서의 섹션을 참조하십시오.
