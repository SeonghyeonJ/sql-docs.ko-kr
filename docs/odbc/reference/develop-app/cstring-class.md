# <a name="cstring-class"></a>CString 클래스
Microsoft® Visual C++®의 **CString** 클래스 개체는 서명 되 고 odbc 함수의 문자열 인수는 서명 되지 않기 때문에 **cstring** 개체를 캐스팅 하지 않고 odbc 함수에 전달 하는 응용 프로그램은 컴파일러 경고를 수신 합니다.
