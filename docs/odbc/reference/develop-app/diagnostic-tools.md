# <a name="diagnostic-tools"></a>진단 도구
두 가지 시설이 ODBC 응용 프로그램의 문제 진단을 지원할 수 있습니다. 둘 다 ODBC 관리자에서 사용할 수 있으며 드라이버 관리자가 구현합니다. *추적은* 함수 호출 시퀀스를 로그 파일에 기록하는 수단을 제공합니다. Visual Studio 분석기를 사용하면 분산 환경에서 구성 요소의 상호 작용에 대한 정보를 분석할 수 있습니다.  
  
 이 섹션에서는 다음 항목을 다룹니다.  
  
-   [추적](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
