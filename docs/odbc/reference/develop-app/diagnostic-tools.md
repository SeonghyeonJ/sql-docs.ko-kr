# <a name="diagnostic-tools"></a>진단 도구
두 가지 기능은 ODBC 응용 프로그램에서 문제를 진단 하는 데 도움이 될 수 있습니다. 둘 다 ODBC 관리자가 사용할 수 있으며 드라이버 관리자에 의해 구현 됩니다. *추적* 은 함수 호출 시퀀스를 로그 파일에 기록 하는 수단을 제공 합니다. Visual Studio 분석기를 사용 하 여 분산 환경에서 구성 요소의 상호 작용에 대 한 정보를 분석할 수 있습니다.  
  
 이 섹션에서는 다음 항목을 다룹니다.  
  
-   [추적](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
