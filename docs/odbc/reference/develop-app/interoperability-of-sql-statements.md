# <a name="interoperability-of-sql-statements"></a>SQL 문의 상호 운용성
응용 프로그램의 나머지를 같은 상호 운용 가능한 또는 DBMS 관련 SQL 문을 수 있습니다. 응용 프로그램 나머지와 같은 SQL 문의 상호 운용 가능한 방법을 선택 해야 하 고 응용 프로그램의 유형에 따라 달라 집니다. 사용자 지정 응용 프로그램은 일반적으로 하나 또는 두 Dbms의 기능을 이용 하도록 설계 되었기 때문에 상호 운용 가능한 SQL 문을 사용 하 여 가능성이 적습니다. 다양 한 Dbms 사용 하 여 작동 하도록 설계 된 것 때문에 상호 운용 가능한 SQL 문을 사용 하 여 일반 응용 프로그램입니다. 및 수직 응용 프로그램을 일반적으로 해당 위치 사이 특정 수준의 기능을 요구 하지만 상호 운용 가능한 SQL 문을 사용 합니다.
