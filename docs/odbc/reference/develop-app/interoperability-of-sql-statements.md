# <a name="interoperability-of-sql-statements"></a>SQL 문의 상호 운용성
나머지 응용 프로그램과 마찬가지로 SQL 문은 상호 운용 가능하거나 DBMS에 따라 다를 수 있습니다. 그리고 응용 프로그램의 나머지 부분과 마찬가지로 상호 운용 가능한 SQL 문을 선택하는 방법은 응용 프로그램의 유형에 따라 다릅니다. 사용자 지정 응용 프로그램은 일반적으로 하나 또는 두 개의 DBMS의 기능을 악용하도록 설계되었기 때문에 상호 운용 가능한 SQL 문을 사용할 가능성이 적습니다. 일반 응용 프로그램은 다양한 DBMS에서 작동하도록 설계되었기 때문에 상호 운용 가능한 SQL 문을 사용합니다. 그리고 수직 응용 프로그램은 일반적으로 일정 수준의 기능을 요구하지만 상호 운용 가능한 SQL 문을 사용하는 사이에 있습니다.
