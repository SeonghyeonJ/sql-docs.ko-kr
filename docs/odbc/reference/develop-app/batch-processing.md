# <a name="batch-processing"></a>일괄 처리
SQL 문의 일괄 처리에 대 한 지원 되므로 상호 운용 가능한 응용 프로그램을 사용 해야 이러한 조건에 따라 또는 전혀를 광범위 하 게 아닙니다. 자세한 내용은 [일괄 처리 실행](../../../odbc/reference/develop-app/executing-batches.md)합니다.
