# <a name="data-types-in-odbc"></a>ODBC의 데이터 형식
ODBC는 SQL 데이터 형식과 C 데이터 형식이라는 두 가지 데이터 형식 집합을 사용합니다. SQL 데이터 형식은 데이터 원본에서 사용되며 C 데이터 형식은 응용 프로그램의 C 코드에서 사용됩니다.
