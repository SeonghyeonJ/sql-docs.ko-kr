# <a name="conformance-levels"></a>적합성 수준
ODBC 드라이버는 응용 프로그램에 다양 한 데이터 원본에 대 한 액세스를 제공 합니다. 각 드라이버를 사용 하면 응용 프로그램에서 런타임에 ODBC 기능 및 드라이버와 각 데이터 원본이 지 원하는 SQL 문법을 확인할 수 있습니다. 이러한 응용 프로그램은 해당 드라이버 또는 드라이버의 기능에 간단 하 게 작성할 수 있으므로 단일 드라이버 또는 작은 알려진 드라이버 집합에서 작동 하도록 설계 된 응용 프로그램의 요구 사항은 아닙니다. 응용 프로그램에서 드라이버 및 데이터 원본 기능을 검색 하는 데 도움이 되도록 두 가지 규칙 (ODBC 인터페이스 및 SQL 문법)을 사용할 수 있습니다.
