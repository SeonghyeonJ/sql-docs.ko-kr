# <a name="conformance-levels"></a>적합성 수준
ODBC 드라이버는 다양 한 데이터 원본에 응용 프로그램 액세스를 제공합니다. 각 드라이버는 드라이버가 달라 지는 런타임 시 ODBC 기능 및 SQL 문법에 어떤 응용 프로그램 수 및 각 데이터 소스에서 지원 합니다. 이러한 응용 프로그램이은 단순히 해당 드라이버 또는 드라이버의 기능에 쓸 수 있으므로 단일 드라이버 또는 드라이버의 알려진된 작은 집합을 사용 하도록 디자인 된 응용 프로그램의 요구 사항은 아닙니다. 드라이버 및 데이터 소스 기능을 검색 하는 응용 프로그램을 위해 두 개의 영역 규칙을 사용할 수 있습니다: ODBC 인터페이스 및 SQL 문법입니다.
