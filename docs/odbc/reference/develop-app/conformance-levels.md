# <a name="conformance-levels"></a>적합성 수준
ODBC 드라이버는 응용 프로그램에 다양한 데이터 원본에 대한 액세스를 제공합니다. 각 드라이버를 사용하면 응용 프로그램이 런타임에 ODBC 기능과 드라이버및 각 데이터 원본이 지원하는 SQL 문법을 결정할 수 있습니다. 이러한 응용 프로그램은 단순히 해당 드라이버 또는 드라이버의 기능에 기록될 수 있기 때문에 단일 드라이버 또는 작고 알려진 드라이버 집합과 함께 작동하도록 설계된 응용 프로그램의 요구 사항은 아닙니다. 응용 프로그램이 드라이버 및 데이터 원본 기능을 검색할 수 있도록 ODBC 인터페이스와 SQL 문법이라는 두 가지 적합성 영역을 사용할 수 있습니다.
