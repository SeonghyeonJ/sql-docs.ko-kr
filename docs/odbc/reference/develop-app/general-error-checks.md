# <a name="general-error-checks"></a>일반 오류 검사
드라이버 관리자는 하나의 일반 오류를 확인합니다. 항상 다음 오류가 발생할 경우 SQL_ERROR를 반환 합니다. 함수는 드라이버에서 지원 되어야 합니다.
