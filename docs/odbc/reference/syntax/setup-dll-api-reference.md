# <a name="setup-dll-api-reference"></a>설정 DLL API 참조
이 섹션에서는 두 가지 함수 (**Configdriver** 및 **ConfigDSN**)로 구성 된 드라이버 설치 DLL API의 구문에 대해 설명 합니다. **Configdriver** 및 **CONFIGDSN** 는 드라이버 dll 또는 별도의 설치 DLL에 있을 수 있습니다.  
  
 또한이 섹션에서는 단일 함수 (**Configtranslator**)로 구성 된 번역기 설정 DLL API의 구문에 대해 설명 합니다. **Configtranslator** 는 변환기 dll 또는 별도의 설치 dll에 있을 수 있습니다.  
  
 각 함수에는 해당 함수가 도입 된 ODBC 버전으로 레이블이 지정 됩니다.  
  
 이 섹션에서는 다음 항목을 다룹니다.  
  
-   [ConfigDriver 함수](../../../odbc/reference/syntax/configdriver-function.md)  
  
-   [ConfigDSN 함수](../../../odbc/reference/syntax/configdsn-function.md)  
  
-   [ConfigTranslator 함수](../../../odbc/reference/syntax/configtranslator-function.md)
