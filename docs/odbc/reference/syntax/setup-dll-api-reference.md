 이 섹션에서는 변환기 설치 DLL API를 단일 함수로 구성의 구문을 설명 하는 또한 (**ConfigTranslator**). **ConfigTranslator** 수 translator DLL의에서 이거나 별도의 DLL을 설치 합니다.  
  
 각 함수는 도입 하는 ODBC의 버전을 사용 하 여 레이블이 지정 됩니다.
