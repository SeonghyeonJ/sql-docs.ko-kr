 버전 도입: ODBC 1.0 표준 규정 준수: 더 이상 사용되지 않는  
  
 **요약**  
 ODBC 3. *x,* ODBC 2. *x* 함수 **SQLAllocStmt가** **SQLAllocHandle으로**대체되었습니다. 자세한 내용은 [SQLAllocHandle 함수를](../../../odbc/reference/syntax/sqlallochandle-function.md)참조하십시오.
