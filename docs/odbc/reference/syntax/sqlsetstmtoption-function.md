# <a name="sqlsetstmtoption-function"></a>SQLSetStmtOption 함수
**규칙**  
 소개 된 버전: ODBC 1.0 표준 준수: 사용 되지 않음  
  
 **요약**  
 ODBC*3.x에서 odbc*2.0 함수 **SQLSetStmtOption** 는 **SQLSetStmtAttr**로 대체 되었습니다. 자세한 내용은 [SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md)를 참조 하세요.  
  
> [!NOTE]
>  ODBC*2.x 응용 프로그램* 에서 odbc*2.x 드라이버를* 사용할 때 드라이버 관리자가이 기능을에 매핑하는 방법에 대 한 자세한 내용은 부록 G: 이전 버전과의 호환성을 위한 드라이버 지침에서 [사용 되지 않는 함수 매핑](../../../odbc/reference/appendixes/mapping-deprecated-functions.md) 을 참조 하세요.  
  
## <a name="remarks"></a>설명  
 응용 프로그램이 64 비트 운영 체제에서 실행 되는 경우 [ODBC 64 비트 정보](../../../odbc/reference/odbc-64-bit-information.md)를 참조 하세요.
