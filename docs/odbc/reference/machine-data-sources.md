# <a name="machine-data-sources"></a>머신 데이터 원본
*컴퓨터 데이터 원본은* 사용자 정의 이름으로 시스템에 저장됩니다. 데이터 원본 이름과 연결된 모든 정보는 드라이버 관리자와 드라이버가 데이터 원본에 연결하는 데 필요한 모든 정보입니다. Xbase 데이터 원본의 경우 Xbase 드라이버의 이름, Xbase 파일이 포함된 디렉터리의 전체 경로 및 단일 사용자 모드 또는 읽기 전용과 같은 해당 파일을 사용하는 방법을 드라이버에게 알려주는 몇 가지 옵션일 수 있습니다. Oracle 데이터 원본의 경우 오라클 드라이버의 이름, Oracle DBMS가 있는 서버, 사용할 SQL\*Net 드라이버를 식별하는 SQL*Net 연결 문자열 및 서버의 데이터베이스 ID일 수 있습니다.
