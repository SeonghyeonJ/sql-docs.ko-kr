# <a name="machine-data-sources"></a>머신 데이터 원본
*컴퓨터 데이터 원본은* 사용자 정의 이름을 사용 하 여 시스템에 저장 됩니다. 데이터 원본 이름과 연결 된 모든 정보는 드라이버 관리자와 드라이버에서 데이터 원본에 연결 하는 데 필요한 정보입니다. Xbase 데이터 원본의 경우에는 Xbase 드라이버의 이름, Xbase 파일을 포함 하는 디렉터리의 전체 경로 및 해당 파일을 사용 하는 방법 (예: 단일 사용자 모드 또는 읽기 전용)을 드라이버에 알리는 일부 옵션을 사용할 수 있습니다. Oracle 데이터 원본의 경우 oracle 드라이버의 이름, Oracle DBMS가 있는 서버, 사용할 SQL\*net 드라이버를 식별 하는 Sql * Net 연결 문자열 및 서버에 있는 데이터베이스의 시스템 ID가 될 수 있습니다.
