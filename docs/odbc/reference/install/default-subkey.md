# <a name="default-subkey"></a>기본 하위 키
레지스트리는 기본 하위 키를 사용 하 여 기본 데이터 원본을 지정할 수 있습니다. 이 하위 키는 데이터 원본 사양 하위 키의 특수 한 경우 이며 다른 데이터 원본 사양 하위 키와 동일한 값을 갖습니다. 유일한 차이점은 ODBC 데이터 원본 하위 키 아래에 값으로 나열 되지 않는다는 것입니다.
