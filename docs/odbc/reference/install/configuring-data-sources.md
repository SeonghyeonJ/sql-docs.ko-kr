# <a name="configuring-data-sources"></a>데이터 원본 구성
데이터 원본에 대한 정보는 시스템 레지스트리에 저장됩니다. 사용자는 관리 프로그램을 통해 데이터 원본 정보를 수정합니다. ODBC 관리자, ODBC 제어판 장치 또는 응용 프로그램 또는 드라이버 개발자가 작성한 관리 프로그램일 수 있습니다.  
  
 PowerShell 명령을 사용하여 데이터 원본을 수정할 수 있습니다. 이러한 PowerShell 명령에 대한 자세한 내용은 [Windows 데이터 액세스 구성 요소 PowerShell 명령을](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)참조하십시오.  
  
> [!NOTE]  
>  이 섹션은 Microsoft Windows® 플랫폼에만 해당됩니다. 다른 플랫폼에서 데이터 원본을 구성하는 방법은 플랫폼에 따라 다릅니다.  
  
 이 단원에는 다음 항목이 포함되어 있습니다.
