# <a name="numeric-literals"></a>숫자 리터럴
숫자 리터럴은 숫자 데이터 값은 문자열에 저장 된 경우에 사용 됩니다. 숫자 SQL 데이터를 SQL_C_CHAR 문자열로 변환 또는 SQL_CHAR 또는 SQL_VARCHAR 문자열을 숫자 C 데이터 변환에 대 한 대상에 저장 된 것을 지정 하려면 숫자 리터럴 구문 사용 됩니다. 숫자 SQL 데이터를 SQL_C_CHAR 문자열로 저장 하는 숫자 또는 숫자 C 데이터에 대 한 SQL_CHAR 문자열로 저장 된 숫자의 변환에 대 한 유효성을 검사할 원본에 저장 된 것이 구문이 사용 됩니다.
