# <a name="numeric-literals"></a>숫자 리터럴
숫자 리터럴은 숫자 데이터 값이 문자 문자열에 저장될 때 사용됩니다. 숫자 SQL 데이터를 SQL_C_CHAR 문자열로 변환하거나 숫자 C 데이터를 SQL_CHAR 또는 SQL_VARCHAR 문자열로 변환하려면 숫자 리터럴 구문을 사용하여 대상에 저장된 내용을 지정합니다. SQL_C_CHAR 문자열로 저장된 숫자를 숫자 SQL 데이터로 변환하거나 SQL_CHAR 문자열로 저장된 숫자를 숫자 C 데이터로 변환하려면 이 구문을 사용하여 원본에 저장된 내용을 확인합니다.  
  
 숫자 리터럴은 부록 C: SQL 문법의 [숫자 리터럴 구문](../../../odbc/reference/appendixes/numeric-literal-syntax.md) 섹션에 정의된 구문을 준수해야 합니다.
