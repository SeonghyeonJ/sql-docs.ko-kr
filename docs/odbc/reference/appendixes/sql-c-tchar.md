# <a name="sql_c_tchar"></a>SQL_C_TCHAR
SQL_C_TCHAR 형식 식별자는 실제로 데이터 형식을 식별하지 않습니다. 유니코드 변환을 위해 헤더 파일 내에 있는 매크로입니다. 유니코드 **#define**설정에 따라 SQL_C_CHAR 또는 SQL_C_WCHAR 대체됩니다. ANSI 및 유니코드 응용 프로그램으로 컴파일되는 문자 데이터를 전송하는 응용 프로그램에 유용합니다.
