 *기본 테이블 식별자* :: = *사용자 정의 이름*  
  
 *기본 테이블 이름* :: = *자료-테이블-식별자*

 *열 식별자* :: = *사용자 정의 이름*

 *숫자가* :: = 0 &#124; 1 &#124; 2 &#124; 3 &#124; 4 &#124; 5 &#124; 6 &#124; 7 &#124; 8 &#124; 9
