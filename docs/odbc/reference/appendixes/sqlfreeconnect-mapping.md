# <a name="sqlfreeconnect-mapping"></a>SQLFreeConnect 매핑
응용 프로그램을 호출할 때 **SQLFreeConnect** 는 ODBC 3 *.x* 드라이버에 대 한 호출  
  
```  
SQLFreeConnect(hdbc)   
```  
  
 매핑되는  
  
```  
SQLFreeHandle(SQL_HANDLE_DBC,Handle)  
```  
  
 사용 하 여 합니다 *처리할* 인수는 값으로 설정 *hdbc*합니다.
