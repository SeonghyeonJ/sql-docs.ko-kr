# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations 및 커서 라이브러리
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거 될 예정입니다. 새 개발 작업에서는이 기능을 사용 하지 않도록 하 고 현재이 기능을 사용 하는 응용 프로그램은 수정 하십시오. 드라이버의 커서 기능을 사용 하는 것이 좋습니다.  
  
 커서 라이브러리는 **SQLBulkOperations**호출을 지원 하지 않습니다. **SQLBulkOperations**에 대 한 일반 정보는 [SQLBulkOperations 함수](../../../odbc/reference/syntax/sqlbulkoperations-function.md)를 참조 하세요.
