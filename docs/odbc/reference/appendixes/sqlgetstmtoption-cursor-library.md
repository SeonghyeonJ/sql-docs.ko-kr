# <a name="sqlgetstmtoption-cursor-library"></a>SQLGetStmtOption(커서 라이브러리)
> [!IMPORTANT]  
>  이 기능은 이후 버전의 Windows에서 제거됩니다. 새 개발 작업에서 이 기능을 사용하지 말고 현재 이 기능을 사용하는 응용 프로그램을 수정할 계획입니다. 드라이버의 커서 기능을 사용하는 것이 좋습니다.  
  
 이 항목에서는 커서 라이브러리에서 **SQLGetStmtOption** 함수의 사용에 대해 설명합니다. **SQLGetStmt옵션에**대한 일반 정보는 [SQLGetStmt옵션 함수를](../../../odbc/reference/syntax/sqlgetstmtoption-function.md)참조하십시오.  
  
 커서 라이브러리는 **SQLGetStmtOption을**통해 다음 명령문 옵션을 지원합니다.  
  
|||  
|-|-|  
|SQL_BIND_TYPE|SQL_ROW_NUMBER|  
|SQL_CONCURRENCY|SQL_ROWSET_SIZE|  
|SQL_CURSOR_TYPE|SQL_SIMULATE_CURSOR|  
|SQL_GET_BOOKMARK||
