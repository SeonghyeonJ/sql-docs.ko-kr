# <a name="sqlgetstmtoption-cursor-library"></a>SQLGetStmtOption(커서 라이브러리)
> [!IMPORTANT]  
>  이 기능은 Windows의 이후 버전에서 제거 됩니다. 새 개발 작업에서는이 기능을 사용 하지 말고 현재이 기능을 사용 하는 응용 프로그램은 수정 합니다. 드라이버의 커서 기능을 사용 하는 것이 좋습니다.  
  
 이 항목에서는의 사용을 설명 합니다 **SQLGetStmtOption** 커서 라이브러리의 함수입니다. 에 대 한 일반 정보에 대 한 **SQLGetStmtOption**를 참조 하십시오 [SQLGetStmtOption 함수](../../../odbc/reference/syntax/sqlgetstmtoption-function.md)합니다.  
  
 커서 라이브러리를 사용 하 여 다음 문 옵션을 지 원하는 **SQLGetStmtOption**:  
  
|||  
|-|-|  
|SQL_BIND_TYPE|SQL_ROW_NUMBER|  
|SQL_CONCURRENCY|SQL_ROWSET_SIZE|  
|SQL_CURSOR_TYPE|SQL_SIMULATE_CURSOR|  
|SQL_GET_BOOKMARK||
