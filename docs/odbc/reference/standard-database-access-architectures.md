# <a name="standard-database-access-architectures"></a>표준 데이터베이스 액세스 아키텍처
이전 섹션에서 설명한 데이터베이스 액세스 구성 요소를 살펴보려고 판명는 두 개의 프로그래밍 인터페이스 및 데이터 스트림 프로토콜은 표준화에 적합 합니다. 다른 두 구성 요소-IPC 메커니즘 및 네트워크 프로토콜-너무 낮은 수준에서 뿐만 아니라 있어야 하지만 둘 다의 네트워크 및 운영 체제에 따라 크게 달라 집니다. 표준화에 대 한 가능성을 제공 하는 세 번째 방법은-게이트웨이에 이기도 합니다.
