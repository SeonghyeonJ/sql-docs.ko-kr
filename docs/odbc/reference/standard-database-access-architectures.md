# <a name="standard-database-access-architectures"></a>표준 데이터베이스 액세스 아키텍처
이전 섹션에 설명 된 데이터베이스 액세스 구성 요소를 살펴보면 두 가지 프로그래밍 인터페이스와 데이터 스트림 프로토콜 중 두 가지를 표준화 하는 것이 좋습니다. 다른 두 가지 구성 요소 (IPC 메커니즘 및 네트워크 프로토콜)는 너무 낮은 수준에 있을 뿐만 아니라 네트워크 및 운영 체제에 따라 달라 집니다. 표준화에 대 한 가능성을 제공 하는 세 번째 방법인 게이트웨이도 있습니다.
