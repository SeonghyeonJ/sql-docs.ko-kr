# <a name="standard-data-stream-protocol"></a>표준 데이터 스트림 프로토콜
표준 데이터 스트림 프로토콜을 수 있는 방법은 유형이 다른 Dbms의 데이터에 액세스 합니다. 실제로 표준 데이터 스트림 프로토콜을 이미 있습니다.

- ANSI/ISO 원격 데이터베이스 액세스 RDA () 표준: [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). ANSI/ISO 시스템 프라미스를 보여 주지만, 구현 되지 않은 광범위 하 게 지금 합니다.
