# <a name="standard-data-stream-protocol"></a>표준 데이터 스트림 프로토콜
표준 데이터 스트림 프로토콜은 이기종 DBMS의 데이터에 액세스하는 한 가지 방법입니다. 실제로 표준 데이터 스트림 프로토콜은 이미 존재합니다.

- ANSI/ISO 원격 데이터베이스 액세스(RDA) 표준: [ISO/IEC 9579:2000.](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615) ANSI/ISO 시스템은 약속을 보여주지만 오늘날 널리 구현되지는 않습니다.
