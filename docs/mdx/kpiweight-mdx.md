# <a name="kpiweight-mdx"></a>KPIWeight(MDX)


  지정된 KPI(핵심 성과 지표)의 가중치를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>인수  
 *KPI_Name*  
 KPI의 이름을 지정하는 유효한 문자열 식입니다.  
  
## <a name="remarks"></a>Remarks  
 반환된 값은 부모에 대한 KPI의 기여도입니다.
