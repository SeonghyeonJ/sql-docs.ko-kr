# <a name="mdx-data-manipulation-statements-mdx"></a>MDX 데이터 조작 문(MDX)


  MDX에서 데이터 조작 문은 다차원 개체로부터 데이터를 검색하고 조작합니다. 다음 표에서는 MDX의 데이터 조작 문을 나열합니다.  
  
## <a name="in-this-section"></a>섹션 내용  
  
|항목|Description|  
|-----------|-----------------|  
|[CALL 문이 MDX를 &#40;&#41;](../mdx/mdx-data-manipulation-call.md)|현재 범위 또는 지정된 큐브에서 void를 반환하는 저장 프로시저를 실행합니다.|  
|[MDX &#40;CLEAR 계산 문&#41;](../mdx/mdx-data-manipulation-clear-calculations.md)|큐브에서 모든 계산을 제거하고 큐브를 계산 패스 0으로 되돌립니다.|  
|[DRILLTHROUGH 문이 MDX를 &#40;&#41;](../mdx/mdx-data-manipulation-drillthrough.md)|큐브에서 지정된 셀을 만드는 데 사용된 행 집합을 검색합니다.|  
|[SELECT 문 &#40;MDX&#41;](../mdx/mdx-data-manipulation-select.md)|지정한 큐브에서 데이터를 검색합니다.|  
|[MDX &#40;MDX 문으로 업데이트&#41;](../mdx/mdx-data-manipulation-update-cube.md)|큐브의 지정된 리프 또는 리프가 아닌 셀의 값을 업데이트합니다. 지정된 리프가 아닌 셀 값을 종속되는 여러 리프 셀에 할당할 수도 있습니다.|
