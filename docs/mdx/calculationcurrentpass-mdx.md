# <a name="calculationcurrentpass-mdx"></a>CalculationCurrentPass(MDX)


  지정한 쿼리 컨텍스트에 대한 큐브의 현재 계산 패스를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
CalculationCurrentPass()  
```  
  
## <a name="remarks"></a>설명  
 **CalculationCurrentPass** 함수는 현재 쿼리 컨텍스트에 대 한 계산 패스의 인덱스 (0부터 시작)를 반환 합니다. 자동 재귀 해결을 사용 하는 경우이 함수는 실제로 거의 사용 되지 않습니다.
