# <a name="-positive-mdx"></a>+(양수)(MDX)


  숫자 식의 양수 값을 반환하는 단항 연산을 수행합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
+ Numeric_Expression  
```  
  
#### <a name="parameters"></a>매개 변수  
 *Numeric_Expression*  
 숫자 값을 반환하는 유효한 MDX 식입니다.  
  
## <a name="return-value"></a>반환 값  
 지정한 매개 변수의 데이터 형식을 갖는 양수 값입니다.
