# <a name="mdx-function-reference-mdx"></a>MDX 함수 참조(MDX)


  Analysis Services는 MDX (Multidimensional Expressions) 구문에서 함수를 사용할 때 제공 됩니다. 유효한 모든 MDX 문에서 함수를 사용할 수 있으며, 쿼리, 사용자 지정 롤업 정의 및 기타 계산에서도 함수가 자주 사용됩니다. 이 섹션에서는 MDX 함수에 대 한 정보를 제공 합니다.  
  
 다음 표를 사용하여 반환 값의 범주별로 함수를 찾을 수도 있고, 목차의 알파벳순 목록에서 이름으로 함수를 선택할 수도 있습니다.
