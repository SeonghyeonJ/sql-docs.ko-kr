# <a name="assignment-operators"></a>대입 연산자


  MDX에서 대입 연산자는 등호(=)입니다.  
  
 대입 연산자를 사용하여 MDX 스크립트의 하위 큐브에 값을 할당합니다. 자세한 내용은 [MDX 스크립팅 기본 사항&#40;Analysis Services&#41;](../analysis-services/multidimensional-models/mdx/mdx-scripting-fundamentals-analysis-services.md)을 참조하세요.
