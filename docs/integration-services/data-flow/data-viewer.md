# <a name="data-viewer"></a>데이터 뷰어

[!INCLUDE[ssis-appliesto](../../includes/ssis-appliesto-ssvrpluslinux-asdb-asdw-xxx.md)]


  경로에서 데이터 뷰어를 사용하도록 구성한 경우 데이터 뷰어에서는 데이터가 두 데이터 흐름 구성 요소 간을 이동할 때 데이터를 버퍼별로 표시합니다.  
  
## <a name="options"></a>옵션  
 **녹색 화살표**  
 다음 버퍼에 데이터를 표시하려면 클릭합니다. 데이터를 단일 버퍼로 이동할 수 있는 경우 이 옵션은 사용할 수 없습니다.  
  
 **분리**  
 데이터 뷰어를 분리합니다.  
  
 **참고** 데이터 뷰어를 분리해도 데이터 뷰어가 삭제되지는 않습니다. 데이터 뷰어를 분리한 경우 데이터는 계속해서 경로를 통해 전달되지만 뷰어의 데이터는 각 버퍼의 데이터와 일치하도록 업데이트되지 않습니다.  
  
 **연결**  
 데이터 뷰어를 연결합니다.  
  
 **참고** 데이터 뷰어를 연결하면 뷰어에서 데이터 흐름에 있는 각 버퍼의 정보를 표시한 다음 일시 중지합니다. 녹색 화살표를 사용하여 버퍼 내에서 이동할 수 있습니다.  
  
 **데이터 복사**  
 현재 버퍼의 데이터를 클립보드로 복사합니다.
