# <a name="lesson-6-3-test-the-lesson-6-package"></a>6-3단원: 6단원 패키지 테스트
런타임에서 패키지는 **VarFolderName** 매개 변수로부터 **Directory** 속성 값을 가져옵니다.  
  
패키지가 **Directory** 속성을 업데이트했는지 확인하려면 패키지를 실행하세요. 새 디렉터리에 세 개의 샘플 데이터 파일을 복사했으므로 데이터 흐름이 세 번 실행됩니다.
  
## <a name="check-the-package-layout"></a>패키지 레이아웃 확인  
패키지를 테스트하려면 먼저 6단원 패키지의 제어 흐름과 데이터 흐름이 다음 다이어그램에 표시된 개체와 비슷한지 확인하세요.

## <a name="test-the-lesson-6-package"></a>6단원 패키지 테스트  
  
1.  **디버그** 메뉴에서 **디버깅 시작**을 선택합니다.  
  
2.  패키지의 실행이 완료된 후에 **디버그** 메뉴에서 **디버깅 중지**를 선택합니다.
