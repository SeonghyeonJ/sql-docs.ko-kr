# <a name="operators---logical"></a>연산자 - 논리
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  DMX (데이터 마이닝 확장) 식에서 논리 연산자를 사용 하 여 값을 평가 하 고에서 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]부울 값을 반환할 수 있습니다.  
  
 다음 표에서는 DMX에서 지원하는 논리 연산자를 설명합니다.  
  
|연산자|Description|  
|--------------|-----------------|  
|[및 &#40;DMX&#41;](../dmx/and-dmx.md)|두 숫자 식에 논리 결합을 수행합니다.|  
|[&#40;DMX&#41;](../dmx/not-dmx.md)|숫자 식에 논리 부정을 수행합니다.|  
|[또는 &#40;DMX&#41;](../dmx/or-dmx.md)|두 숫자 식에 논리 분리를 수행합니다.|
