# <a name="reserved-database-names-in-parallel-data-warehouse"></a>병렬 데이터 웨어하우스의 예약 된 데이터베이스 이름
예약 된 데이터베이스 이름은 SQL Server PDW의 사용자 데이터베이스에 할당할 수 없는 이름입니다.  
  
예약 된 데이터베이스 이름 목록:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
