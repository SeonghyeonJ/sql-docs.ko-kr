# <a name="reserved-database-names-in-parallel-data-warehouse"></a>병렬 데이터 웨어하우스에서 예약 된 데이터베이스 이름
예약된 된 데이터베이스 이름에는 SQL Server PDW에서 사용자 데이터베이스에 할당 될 수 없는 이름이입니다.  
  
예약 된 데이터베이스 이름의 목록:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
