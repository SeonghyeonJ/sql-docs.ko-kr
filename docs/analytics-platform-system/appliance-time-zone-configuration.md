# <a name="appliance-time-zone-configuration---analytics-platform-system"></a>표준 시간대 구성-Analytics Platform System appliance
합니다 **시간대** 페이지를 사용 하면 Analytics Platform System (APS) 어플라이언스 모든 노드에 대 한 표준 시간대를 설정할 수 있습니다.  
  
## <a name="to-set-the-time-zone"></a>표준 시간대를 설정 하려면  
  
1.  구성 관리자를 시작 합니다. 자세한 내용은 [구성 관리자 시작 &#40;Analytics Platform System&#41;](launch-the-configuration-manager.md)합니다.  
  
2.  사용 하 여 어플라이언스 서비스를 중지 합니다 **서비스 상태** Configuration Manager에서 페이지. 참조 [PDW 서비스 상태 &#40;Analytics Platform System&#41; ](pdw-services-status.md) 지침에 대 한 합니다.  
  
3.  구성 관리자의 왼쪽된 창에서 클릭 **시간대**합니다. 원하는 표준 시간대를 선택 합니다 **시간대** 드롭 다운 메뉴. 현재 위치에 따라 수도 있습니다 옆에 선택 합니다 **일광 절약 시간제 시계를 자동으로 조정할**합니다.  
  
4.  클릭 **적용** 변경 내용을 저장 합니다.  
  
5.  사용 하 여 어플라이언스 서비스를 다시 시작 합니다 **서비스 상태** Configuration Manager에서 페이지입니다. 권한을 변경도 하려는 경우에 어플라이언스를 다시 시작 하기 전에 수행할 수 있습니다.
