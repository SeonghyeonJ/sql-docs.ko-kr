# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>Analytics Platform System의 관리 콘솔 경고 이해
어플라이언스에서 경고 표시 **관리 콘솔** 고 **System Center Operations Manager** (SCOM)입니다. 경고의이 목록을 사용 하 여 추가 확인이 필요한 경고를 식별할 수 있도록 합니다.  
  
연결에 대 한 정보에 대 한 합니다 **관리 콘솔** Internet Explorer를 사용 하 여 참조 [관리자 콘솔을 사용 하 여 어플라이언스 모니터링 &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-the-admin-console.md). 에 대 한 자세한 **SCOM**를 참조 하십시오 [System Center Operations Manager를 사용 하 여 어플라이언스 모니터링 &#40;분석 플랫폼 시스템&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
Transact를 사용 하 여 경고 정보를 가져오는 방법은\-SQL 참조 [시스템 뷰를 사용 하 여 어플라이언스 모니터링 &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-views.md)합니다.  
  
## <a name="types-of-alerts"></a>경고 유형  
일반 상태를 나타내는 경고 이름은 조사를 일반적으로 필요 하지 않습니다. 경우에 따라 초과 단어를 포함 하는 경고 이름 작업이 필요 합니다. 조사는 다른 모든 유형의 경고에 대 한 필요 합니다.
