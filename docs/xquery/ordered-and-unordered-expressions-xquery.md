# <a name="ordered-and-unordered-expressions-xquery"></a>순차 식 및 비순차 식(XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  기본적으로의 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 모든 작업에 대 한 정렬 모드는 **정렬**됩니다. 따라서 **order by** 절을 사용 하지 않고 경로 식 및 FLWOR 식에서 반환 된 노드 시퀀스는 문서 순서로 되어 있습니다.  
  
 XQuery 사양에서 설명 하는 **순서가 지정** 되거나 **순서가** 지정 되지 않은 추가 구문은 지원 되지 않습니다.
