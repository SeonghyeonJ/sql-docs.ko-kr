# <a name="task-8-creating-a-composite-domain-rule"></a>태스크 8: 복합 도메인 규칙 만들기
  이 작업에 대 한 규칙을 만듭니다는 **Address Validation** 복합 도메인입니다. 도메인 간 규칙을 정의: 경우 **도시** 됩니다 **Los Angeles**, **상태** 여야 합니다 **CA** 여기서 **City** 및 **상태** 는 두 개의 도메인입니다.  
  
1.  오른쪽 창에서 전환 하는 **CD 규칙** 탭 합니다.  
  
2.  클릭 **새 도메인 규칙을 추가** 도구 모음에서 합니다.  
  
3.  형식 **City-State Rule** 에 대 한 **이름** 누릅니다 **ENTER**합니다.  
  
4.  에 **규칙 작성** 창 **City** 조건을 선택 하 고 도메인 목록에서 **값이 같지** 형식과 **Los Angeles** 에 대 한 합니다 값입니다.  
  
5.  에 **한 다음** 창 **상태** 도메인 목록에서 선택 **값은**, 형식 **CA** 값 및 키를 눌러 **탭**합니다.  
  
     ![복합 도메인 규칙](../../2014/tutorials/media/et-creatingacompositedomainrule.jpg "복합 도메인 규칙")  
  
6.  클릭 **닫기** DQS 클라이언트의 기본 페이지로 전환 페이지의 맨 위에 있는 단추입니다. 이 기술 자료는 다음 단원에서 게시합니다. 이 기술 자료는 잠긴 상태(자물쇠 아이콘)입니다.
