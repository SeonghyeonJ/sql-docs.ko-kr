# <a name="edit-the-advanced-properties"></a>고급 속성 편집
  **고급** 탭을 사용하여 CDC 인스턴스에 특별 속성을 추가할 수 있습니다.  
  
#### <a name="to-add-advanced-properties-to-the-cdc-instance"></a>CDC 인스턴스에 고급 속성을 추가하려면  
  
1.  **이름** 열에 속성의 이름을 입력합니다.  
  
2.  **값** 열에 속성에 대해 유효한 값을 입력합니다.  
  
    > [!NOTE]  
    >  뷰어에서 고급 속성을 볼 수 있습니다. 뷰어를 사용할 때 정보는 읽기 전용입니다. 또한 뷰어에는 테이블에서 캡처된 열 목록이 포함되어 있습니다.  
  
 입력할 수 있는 속성에 대한 자세한 내용은 [cdc.xdbcdc_config](the-oracle-cdc-databases.md#BKMK_cdcxdbcdc_config)에서 사용 가능한 옵션 테이블을 참조하십시오.
