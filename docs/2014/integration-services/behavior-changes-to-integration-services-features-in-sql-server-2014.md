  이 항목에서는 [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)]의 동작 변경 내용에 대해 설명합니다. 동작 변경 내용은 이전 버전의 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]와 비교해 서 현재 릴리스의에서 기능이 작동 하 고 상호 작용 하는 방법에 영향을 줍니다.  
  
 Integration Services 기능의 동작은 SQL Server 2014에서 변경되지 않았습니다.
