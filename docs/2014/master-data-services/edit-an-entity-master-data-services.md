# <a name="change-an-entity-name-master-data-services"></a>엔터티 이름 변경(Master Data Services)
  [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]에서 엔터티의 이름을 변경할 수 있습니다.  
  
> [!NOTE]  
>  연결된 준비 테이블의 이름은 업데이트되지 않습니다.  
  
## <a name="prerequisites"></a>사전 요구 사항  
 이 절차를 수행하려면  
  
-   **시스템 관리** 기능 영역에 액세스할 수 있는 권한이 있어야 합니다.  
  
-   모델 관리자여야 합니다. 자세한 내용은 [관리자&#40;Master Data Services&#41;](administrators-master-data-services.md)를 참조하세요.  
  
### <a name="to-change-an-entity-name"></a>엔터티 이름을 변경하려면  
  
1.  [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)]에서 **시스템 관리**를 클릭합니다.  
  
2.  **모델 뷰** 페이지의 메뉴 모음에서 **관리** 를 가리키고 **엔터티**를 클릭합니다.  
  
3.  **엔터티 유지 관리** 페이지의 **모델** 목록에서 모델을 선택합니다.  
  
4.  이름을 변경할 엔터티의 행을 클릭합니다.  
  
5.  **선택한 엔터티 편집**을 클릭합니다.  
  
6.  에 **엔터티 이름** 상자에 엔터티의 업데이트 된 이름을 입력 합니다.  
  
7.  **엔터티 저장**을 클릭합니다.
