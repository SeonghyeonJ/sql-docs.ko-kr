# <a name="add-metadata-master-data-services"></a>메타데이터 추가(Master Data Services)
  [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], 사용자 정의 메타 데이터를 추가 합니다 **메타 데이터** 모델 개체를 설명 하는 모델입니다.  
  
## <a name="prerequisites"></a>사전 요구 사항  
 이 절차를 수행하려면  
  
-   **탐색기** 기능 영역에 액세스할 수 있는 권한이 있어야 합니다.  
  
-   에 대 한 합니다 **메타 데이터** 해야 모델 **추가** 메타 데이터 엔터티 또는 리프 멤버 유형에 추가할 수 있는 권한이 있습니다.  
  
-   버전의 상태가 **열림**이어야 합니다.  
  
### <a name="to-add-metadata"></a>메타데이터를 추가하려면  
  
1.  에 [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] 에서 홈 페이지를 **모델** 목록에서를 **메타 데이터** 모델.  
  
2.  **버전** 목록에서 버전을 선택합니다.  
  
3.  **탐색기**를 클릭합니다.  
  
4.  메뉴 모음에서 가리킨 **엔터티** 추가할 메타 데이터 엔터티를 클릭 합니다.  
  
5.  클릭 합니다 **멤버 추가** 단추입니다. 합니다 **새 멤버 추가** 페이지가이 페이지의 오른쪽에 표시 됩니다.  
  
6.  새 이름, 코드 및 설명을 입력 한 다음 클릭 **확인**합니다.
