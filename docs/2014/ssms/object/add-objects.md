# <a name="add-objects"></a>개체 추가
  원하는 개체 유형만 개체 선택 대화 상자에 표시되도록 제한합니다.  
  
## <a name="options"></a>옵션  
 **특정 개체**  
 개체 검색 방법을 추가로 정의할 수 있는 **개체 선택** 대화 상자를 엽니다.  
  
 **형식의 모든 개체**  
 기본 목록에 포함할 개체 유형을 지정할 수 있는 **개체 유형 선택** 대화 상자를 엽니다.  
  
 **서버 서버**  _\<이름>_  
 지정한 서버의 해당 개체를 모두 추가합니다. 
  **로그인 - 신규** 대화 상자와 같은 서버 범위 대화 상자에서 여는 경우에만 나타납니다.  
  
 **스키마에 속하는 모든 개체**  
 
  **스키마 이름** 상자에 지정된 스키마가 소유하는 모든 개체를 추가합니다. 
  **데이터베이스 사용자 - 신규** 대화 상자와 같은 데이터베이스 범위 대화 상자에만 나타납니다.
