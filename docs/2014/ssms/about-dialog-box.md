# <a name="about-dialog-box"></a>대화 상자 정보
  **정보** 대화 상자에는 제품 정보가 제공되며 이 대화 상자에서 제품을 실행 중인 시스템에 대한 정보를 액세스할 수 있습니다. 평가본의 경우 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 항목은 설치가 만료되기 전까지 남은 일수를 나열합니다. 이 대화 상자는 **도움말** 메뉴에서 사용할 수 있습니다.  
  
## <a name="options"></a>옵션  
 **구성 요소 이름**  
 설치된 각 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 항목의 이름을 나열합니다.  
  
 **버전**  
 설치된 각 구성 요소의 버전을 나열합니다.  
  
 **정보 복사**  
 구성 요소 이름과 버전을 [!INCLUDE[msCoName](../includes/msconame-md.md)] Windows 클립보드에 복사합니다.
