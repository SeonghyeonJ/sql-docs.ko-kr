  이 대화 상자를 사용하면 INSERT VALUES 쿼리를 사용하여 업데이트할 테이블을 지정할 수 있습니다. 이 대화 상자는 **다이어그램** 창에 두 개 이상의 테이블이 표시되어 있는 상태에서 쿼리 형식을 INSERT VALUES 쿼리로 변경하는 경우에 나타납니다.
