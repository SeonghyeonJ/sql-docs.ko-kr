# <a name="remove-columns-from-query-results-visual-database-tools"></a>쿼리 결과에서 열 제거(Visual Database Tools)
  선택 쿼리에 사용 중인 열을 결과 집합에 표시하지 않으려는 경우 즉, 쿼리의 선택 목록에 포함하지 않으려는 경우 해당 열을 출력에서 제거할 수 있습니다. 쿼리의 출력에서 열을 제거한 후에도 이 열을 검색 조건에 사용하거나 정렬 필드로 사용할 수 있습니다.  
  
> [!NOTE]  
>  쿼리의 열을 함께 제거하려면 [쿼리에서 열 제거&#40;Visual Database Tools&#41;](visual-database-tools.md)를 참조하세요.  
  
### <a name="to-remove-a-column-from-the-query-output"></a>쿼리 결과에서 열을 제거하려면  
  
-   **조건 창**에서 제거하려는 데이터 열의 **출력** 열에 있는 확인란을 선택 취소합니다. **출력** 열을 다시 선택하면 쿼리 결과에 열을 다시 추가할 수 있습니다.  
  
     -또는-  
  
-   [SQL 창](sql-pane-visual-database-tools.md)의 출력 목록에서 열을 제거합니다.
