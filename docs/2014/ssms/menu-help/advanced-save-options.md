# <a name="advanced-save-options"></a>저장 고급 옵션
  이 대화 상자를 사용하여 언어 인코딩 스키마를 선택하고 각 줄 끝에 삽입할 문자를 지정할 수 있습니다. **고급 저장 옵션** 대화 상자에 액세스하려면 **파일** 메뉴에서 **다른 이름으로** \<query name> **.sql 저장**을 클릭하여 **다른 이름으로 파일 저장** 대화 상자를 열고, **저장** 단추에서 아래쪽 화살표를 클릭한 다음, **인코딩하여 저장**을 클릭합니다.  
  
## <a name="options"></a>옵션  
 **인코딩**  
 이 목록 상자를 사용하여 대상 파일에 새 언어 인코딩 스키마를 적용할 수 있습니다.  
  
 **줄 끝**  
 대상 파일이 특정 운영 체제에서 실행되도록 하기 위해 각 줄 끝에 캐리지 리턴(Macintosh 시스템의 경우)을 표시할지 줄 바꿈(Unix 시스템의 경우)을 표시할지 둘 다( [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows 시스템의 경우) 표시할지 선택합니다.
