# <a name="details-ssas"></a>세부 정보(SSAS)
  **테이블 가져오기 마법사** 의 이 페이지에서는 성공하거나 실패한 데이터 가져오기 작업에 대한 정보를 볼 수 있습니다. [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]에서 마법사에 액세스하려면 **모델** 메뉴에서 **데이터 원본에서 가져오기**를 클릭합니다.
