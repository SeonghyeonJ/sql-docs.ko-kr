# <a name="specify-mining-model-column-usage-data-mining-wizard"></a>마이닝 모델 열 사용법 지정(데이터 마이닝 마법사)
  **마이닝 모델 열 사용법 지정** 페이지를 사용하여 마이닝 모델 열이 사용되는 방법을 지정하고 중첩 테이블을 추가할 수 있습니다.  
  
> [!NOTE]  
>  이 페이지는 마법사의 **정의 방법 선택** 페이지에서 **기존 큐브 사용**을 선택한 경우에만 나타납니다.  
  
 **참조 항목:** [데이터 마이닝 마법사 &#40;Analysis Services-데이터 마이닝&#41;](data-mining/data-mining-wizard-analysis-services-data-mining.md)하십시오 [관계형 마이닝 구조 만들기](data-mining/create-a-relational-mining-structure.md)  
  
## <a name="options"></a>변수  
 **테이블/열**  
 마법사에서 이전에 정의한 테이블 및 열의 이름을 표시합니다.  
  
 **Input**  
 테이블 또는 열을 마이닝 모델의 입력으로 사용하려면 선택합니다.  
  
 **예측 가능한**  
 **입력** 테이블 또는 열을 사용하여 테이블 또는 열의 값을 예측하려면 선택합니다.  
  
 **중첩된 테이블 추가**  
 **새 중첩 테이블 추가** 대화 상자를 사용하여 테이블을 추가하려면 클릭합니다.  
  
 **중첩된 테이블 제거**  
 선택한 중첩 테이블을 제거하려면 클릭합니다. 이 단추는 중첩 테이블을 선택한 경우에만 사용할 수 있습니다.
