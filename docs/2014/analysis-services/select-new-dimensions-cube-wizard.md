# <a name="select-new-dimensions-cube-wizard"></a>새 차원 선택(큐브 마법사)
  **새 차원 선택** 페이지를 사용하여 만들 새로 만들 차원을 선택할 수 있습니다.  
  
## <a name="options"></a>변수  
 **Dimensions**  
 만들 차원을 선택합니다. 기본적으로 모든 차원이 선택되어 있습니다.  
  
 트리의 루트 노드는 새로 만들 차원을 나타냅니다. 차원 노드 아래에는 차원을 만드는 데 사용할 관련 테이블이 표시됩니다.  
  
 차원 이름을 바꾸려면 차원 이름을 클릭하고 새 이름을 입력합니다. 테이블 이름은 바꿀 수 없습니다.
