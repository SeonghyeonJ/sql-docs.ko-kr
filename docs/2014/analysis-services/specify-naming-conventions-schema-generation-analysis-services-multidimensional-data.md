# <a name="specify-naming-conventions-schema-generation-wizard-analysis-services---multidimensional-data"></a>명명 규칙 지정(스키마 생성 마법사)(Analysis Services - 다차원 데이터)
  **명명 규칙 지정** 페이지를 사용하여 스키마 생성 마법사에서 스키마 개체를 만들 때 사용하는 명명 규칙을 정의할 수 있습니다.  
  
## <a name="options"></a>옵션  
 **옵션**  
 마법사에서 사용할 명명 규칙을 지정합니다. 다음 표에서는 지정할 수 있는 옵션에 대해 설명합니다.  
  
|옵션|설명|  
|------------|-----------------|  
|**구분 기호**|개체 이름에서 단어를 구분하는 문자를 지정합니다. **값** 열에서 **밑줄**, **공백**또는 **없음**을 선택합니다. 기본값은 **밑줄**입니다.|  
|**기본 키 열 접두사**|각 기본 키 열의 이름 접두사로 사용할 문자열을 지정합니다. 기본값은 **PK**입니다.|  
|**외래 키 열 접두사**|각 외래 키 열의 이름 접두사로 사용할 문자열을 지정합니다. 기본값은 **FK**입니다.|  
|**특성 이름 접미사**|각 특성 열의 이름에 추가할 문자열을 지정합니다. 기본값은 **Name**입니다.|  
|**사용자 지정 롤업 접미사**|각 롤업 열의 이름에 추가할 문자열을 지정합니다. 기본값은 **CustomRollup**입니다.|  
|**사용자 지정 롤업 속성 접미사**|각 롤업 속성 열의 이름에 추가할 문자열을 지정합니다. 기본값은 **CustomRollupProperties**입니다.|  
|**단항 연산자 접미사**|각 단항 연산자 열의 이름에 추가할 문자열을 지정합니다. 기본값은 **UnaryOperator**입니다.|  
  
 **값**  
 **옵션** 에서 스키마 생성 시 사용할 옵션 값을 지정합니다.
