# <a name="importing-ssas"></a>가져오기(SSAS)
  
  **테이블 가져오기 마법사** 의 이 페이지에서는 가져오기 작업의 진행률을 볼 수 있습니다. 
  [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]에서 마법사에 액세스하려면 **모델** 메뉴에서 **데이터 원본에서 가져오기**를 클릭합니다.  
  
## <a name="uielement-list"></a>UIElement 목록  
 **세부 정보**  
 각 데이터 가져오기 작업에 대한 다음 정보를 표시합니다.  
  
|열|Description|  
|------------|-----------------|  
|**작업 항목**|가져오고 있는 테이블 또는 뷰의 이름을 표시합니다.|  
|**상태**|테이블 또는 뷰를 성공적으로 가져왔는지 여부와 가져온 행 수를 나타냅니다.|  
|**메시지**|테이블 또는 뷰 가져오기에 실패한 경우 자세한 정보에 대한 링크를 표시합니다. 이 정보는 세부 정보 창에 표시됩니다.<br /><br /> 테이블 또는 뷰 가져오기를 다시 시도하려면 마법사를 종료하고 다시 실행합니다.|  
  
 **가져오기 중지**  
 완료되기 전에 가져오기 작업을 중지하려면 클릭합니다. 이미 가져온 테이블 및 뷰는 [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)] 디자이너에 표시됩니다. 아직 가져오지 않은 테이블 및 뷰는 가져오지 않게 됩니다.
