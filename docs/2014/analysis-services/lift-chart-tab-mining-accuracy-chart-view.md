# <a name="lift-chart-tab-mining-accuracy-chart-view"></a>리프트 차트 탭(마이닝 정확도 차트 뷰)
  
  **리프트 차트** 창을 사용하여 선택한 마이닝 구조에서 선택한 모든 마이닝 모델을 비교하는 차트를 볼 수 있습니다.  
  
 모델에서 불연속 특성을 예측하는 경우 이 창에 리프트 차트나 수익 차트가 표시될 수 있습니다. 리프트 차트에 대한 자세한 내용은 [리프트 차트&#40;Analysis Services - 데이터 마이닝&#41;](data-mining/lift-chart-analysis-services-data-mining.md)를 참조하세요.  
  
 수익 차트를 만들려면 마이닝 모델의 권장 사항을 구현하는 비용과 예상 수익에 대한 추가 정보를 제공해야 합니다. 자세한 내용은 [수익 차트&#40;Analysis Services - 데이터 마이닝&#41;](data-mining/profit-chart-analysis-services-data-mining.md)를 참조하세요.  
  
 모델에서 연속 특성을 예측하는 경우에는 이 창에 산점도 그래프가 표시됩니다.  
  
 마이닝 모델의 정확도 측정 방법에 대한 일반적인 내용은 [리프트 차트&#40;Analysis Services - 데이터 마이닝&#41;](data-mining/lift-chart-analysis-services-data-mining.md)를 참조하세요.  
  
## <a name="options"></a>옵션  
 **차트 종류**  
 뷰어에 표시할 차트의 종류를 설정합니다. 
  **리프트 차트** 또는 **수익 차트**를 선택할 수 있습니다.  
  
 **설정**  
 수익 차트를 구성하는 데 사용할 수 있는 **수익 차트 설정** 대화 상자를 엽니다.  
  
 
  **열 매핑** 탭에서 예측 가능한 연속 열을 선택한 경우에는 수익 차트를 사용할 수 없습니다.  
  
 **Copy**  
 차트를 클립보드에 복사합니다.
