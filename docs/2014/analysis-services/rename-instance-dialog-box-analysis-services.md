# <a name="rename-instance-dialog-box-analysis-services"></a>인스턴스 이름 바꾸기 대화 상자(Analysis Services)
  
  **인스턴스 이름 바꾸기** 대화 상자를 사용하여 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]의 기존 인스턴스 이름을 바꿀 수 있습니다. C:\Program Files\Microsoft SQL Server\100\Tools\Binn\VSShell\Common7\IDE에서 **인스턴스 이름 바꾸기** 도구(asinstancerename.exe)를 실행하면 **인스턴스 이름 바꾸기** 대화 상자를 표시할 수 있습니다.  
  
## <a name="options"></a>옵션  
  
|용어|정의|  
|----------|----------------|  
|**이름을 바꿀 인스턴스**|이름을 바꿀 인스턴스를 선택합니다.|  
|**새 인스턴스 이름**|원하는 인스턴스 이름을 입력합니다. 서버 이름은 포함하지 마십시오. \<즉, 서버 이름>\\<인스턴스 이름을\>입력 하는 대신 \<인스턴스 이름> 입력 합니다.<br /><br /> 이름을 바꾸려는 인스턴스가 기본 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 인스턴스가 되도록 하려면 이름을 비워 둡니다.|  
|**이름**|서비스에서 시작하는 데 사용할 계정을 표시합니다. 사용자 이름은 변경할 수 없습니다.|  
|**암호**|서비스 계정의 암호를 입력합니다.|
