# <a name="process-progress-dialog-box-analysis-services---multidimensional-data"></a>처리 진행률 대화 상자(Analysis Services - 다차원 데이터)
  
  **및** 의 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] 처리 진행률 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 대화 상자를 사용하여 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]에서의 처리 상황을 모니터링할 수 있습니다. 
  **처리 진행률** 대화 상자는 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 개체 처리를 시작할 때 표시됩니다.  
  
## <a name="options"></a>옵션  
 **상태 트리 뷰**  
 시작 시간, 중지 시간 및 처리 정보를 비롯하여 처리 중인 개체에 대한 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 인스턴스의 상태 메시지를 표시합니다. 상태 메시지의 상세 정보를 클립보드에 복사하려면 항목을 마우스 오른쪽 단추로 클릭한 다음 **복사**를 선택하고, **자세히 보기** 대화 상자를 표시하려면 항목을 두 번 클릭합니다. 
  **자세히 보기** 대화 상자에 대한 자세한 내용은 [자세히 보기 대화 상자&#40;Analysis Services - 다차원 데이터&#41;](view-details-dialog-box-analysis-services-multidimensional-data.md)를 참조하세요.  
  
 **상태 설명**  
 처리 중인 작업의 현재 상태를 표시합니다.  
  
 **중지**  
 처리 작업을 중지하려면 클릭합니다.  
  
 **다시 처리**  
 
  **처리 진행률** 대화 상자에 표시된 처리 작업을 반복하려면 클릭합니다.  
  
 **세부 정보 보기**  
 
  **자세히 보기** 대화 상자를 열어 **상태 트리 뷰**에서 선택한 항목의 상세 정보를 표시하려면 클릭합니다. 
  **자세히 보기** 대화 상자에 대한 자세한 내용은 [자세히 보기 대화 상자&#40;Analysis Services - 다차원 데이터&#41;](view-details-dialog-box-analysis-services-multidimensional-data.md)를 참조하세요.
