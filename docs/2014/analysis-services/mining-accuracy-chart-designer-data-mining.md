# <a name="mining-accuracy-chart-designer-data-mining"></a>마이닝 정확도 차트 디자이너(데이터 마이닝)
  **마이닝 정확도 차트** 뷰를 사용하여 선택한 마이닝 구조를 기반으로 작성된 마이닝 모델의 성능을 비교할 수 있습니다.  
  
 **마이닝 정확도 차트** 뷰에는 다음과 같은 탭이 들어 있습니다.  
  
-   [마이닝 정확도 차트 뷰의 입력 선택 탭 &#40;&#41;](input-selection-tab-mining-accuracy-chart-view.md)  
  
     리프트 차트, 수익 차트, 산점도 또는 분류 행렬에 표시할 하나 이상의 모델을 선택합니다. 테스트 데이터를 지정하고 외부 데이터의 열을 모델에 매핑합니다.  
  
-   [리프트 차트 탭 &#40;마이닝 정확도 차트 뷰&#41;](lift-chart-tab-mining-accuracy-chart-view.md)  
  
     `Input Selection` 탭에서 선택한 모델에 대한 리프트 차트, 수익 차트 또는 산점도를 생성합니다.  
  
-   [마이닝 정확도 차트 뷰를 &#40;분류표 탭&#41;](classification-matrix-tab-mining-accuracy-chart-view.md)  
  
     `Input Selection` 탭에서 선택한 모델에 대한 분류 행렬을 표시합니다.  
  
-   [교차 유효성 검사 탭&#40;마이닝 정확도 차트 뷰&#41;](cross-validation-tab-mining-accuracy-chart-view.md)  
  
     선택한 마이닝 구조에 대해 교차 유효성 검사 옵션을 설정하고 결과를 생성합니다.  
  
 이 탭에는 다음과 같은 도구 및 대화 상자가 들어 있습니다.  
  
-   [마이닝 구조 선택 대화 상자 &#40;마이닝 정확도 차트 뷰&#41;](select-mining-structure-dialog-box-mining-accuracy-chart-view.md)  
  
-   [마이닝 정확도 차트 뷰를 &#40;테이블 선택 대화 상자&#41;](select-table-dialog-box-mining-accuracy-chart-view.md)  
  
-   [마이닝 정확도 차트 뷰의 중첩 조인 지정 대화 상자 &#40;&#41;](specify-nested-join-dialog-box-mining-accuracy-chart-view.md)  
  
-   [수익 차트 설정 대화 상자 &#40;마이닝 정확도 차트 뷰&#41;](profit-chart-settings-dialog-box-mining-accuracy-chart-view.md)  
  
-   [매핑 수정 대화 상자 &#40;마이닝 정확도 차트&#41;](modify-mapping-dialog-box-mining-accuracy-chart.md)  
  
-   [필터 대화 상자 &#40;마이닝 정확도 차트&#41;](filter-dialog-box-mining-accuracy-chart.md)  
  
-   [데이터 집합 필터 또는 모델 필터 대화 상자](data-set-filter-or-model-filter-dialog-box.md)
