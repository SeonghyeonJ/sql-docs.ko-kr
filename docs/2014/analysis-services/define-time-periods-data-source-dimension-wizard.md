# <a name="define-time-periods-data-source-dimension-wizard"></a>기간 정의(데이터 원본)(차원 마법사)
  
  **기간 정의** 페이지를 사용하여 **차원 유형 선택** 페이지에서 지정한 테이블의 열로 시간 차원의 기간을 나타내는 특성을 정의할 수 있습니다.  
  
> [!NOTE]  
>  이 페이지는 **차원 정의** 페이지의 **데이터 원본을 사용하여 차원 생성** 및 **차원 유형 선택** 페이지의 **시간 차원** 선택한 경우에만 나타납니다.  
  
## <a name="options"></a>옵션  
 **시간 속성 이름**  
 시간 차원의 기간을 표시하는 데 사용되는 특성 유형을 표시합니다. 특성 유형에 대한 자세한 내용은 [Type 요소&#40;DimensionAttribute&#41;&#40;ASSL&#41;](https://docs.microsoft.com/bi-reference/assl/properties/type-element-dimensionattribute-assl)를 참조하세요.  
  
> [!NOTE]  
>  
  `Date` 특성 유형은 DateTime 데이터 형식의 열에만 사용해야 합니다.  
  
 **시간 테이블 열**  
 해당 특성 유형이 기반으로 할 열 목록을 보여 줍니다.  
  
 열을 변경하려면 열을 클릭한 다음 목록에서 다른 열을 선택합니다.
