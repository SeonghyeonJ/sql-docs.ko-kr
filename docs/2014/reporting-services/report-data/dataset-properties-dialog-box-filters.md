# <a name="dataset-properties-dialog-box-filters"></a>데이터 세트 속성 대화 상자, 필터
  
  **데이터 세트 속성** 대화 상자에서 **필터**를 선택하여 데이터 세트에 대한 필터를 만들 수 있습니다.  
  
## <a name="options"></a>옵션  
 **추가**  
 새 필터 절을 목록에 추가합니다.  
  
 **Delete**  
 선택한 필터 절을 목록에서 삭제합니다.  
  
 **위쪽 화살표**  
 선택한 필터를 목록에서 위로 이동합니다.  
  
 **아래쪽 화살표**  
 선택한 필터를 목록에서 아래로 이동합니다.  
  
 **식**  
 필터를 적용할 식을 입력하거나 선택합니다. 식을 편집 하려면 식 단추 (**fx**)를 클릭 합니다.  
  
 **데이터 형식**  
 
  **값**에 대한 데이터 형식을 선택합니다. 가능하면 **식**에 대한 데이터 형식과 일치하는 데이터 형식을 선택합니다. 자세한 내용은 [필터 수식 예&#40;보고서 작성기 및 SSRS&#41;](../report-design/filter-equation-examples-report-builder-and-ssrs.md)를 참조하세요.  
  
 **연산자**  
 식과 값을 비교하는 데 사용할 연산자를 선택합니다.  
  
 **값**  
 
  **식** 상자에 지정된 식을 계산할 때 사용할 값이나 식을 입력합니다. 식을 편집 하려면 식 단추 (**fx**)를 클릭 합니다.
