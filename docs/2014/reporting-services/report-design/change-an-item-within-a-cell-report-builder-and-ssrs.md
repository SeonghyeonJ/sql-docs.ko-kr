# <a name="change-an-item-within-a-cell-report-builder-and-ssrs"></a>셀 내 항목 변경(보고서 작성기 및 SSRS)
  입력란, 선 또는 이미지와 같은 비컨테이너 항목만 새 보고서 항목으로 대체할 수 있습니다. 예를 들어 테이블을 입력란으로 끌어 테이블로 입력란을 대체할 수 있습니다.  
  
 셀에 사각형, 목록, 테이블, 행렬 등과 같은 컨테이너 항목이 포함되어 있는 경우 새 항목은 포함 항목을 대체하지 않고 항목에 추가됩니다. 컨테이너 항목을 새 항목으로 대체하려면 컨테이너를 삭제합니다. 컨테이너 항목을 삭제하면 다른 항목으로 대체할 수 있는 입력란으로 대체됩니다.  
  
 기본적으로 테이블, 행렬 또는 목록 데이터 영역의 모든 셀에는 입력란이 포함됩니다.  
  
> [!NOTE]  
>  [!INCLUDE[ssRBRDDup](../../includes/ssrbrddup-md.md)]  
  
### <a name="to-change-an-item-within-a-cell"></a>셀에서 항목을 변경하려면  
  
-   **삽입** 탭의 **데이터 영역** 또는 **보고서 항목** 그룹에서 보고서에 추가할 항목을 클릭한 다음 보고서를 클릭합니다. 보고서에 항목이 추가됩니다.  
  
> [!NOTE]  
>  이미지 보고서 항목을 끌어서 셀에 놓으면 **이미지 속성** 대화 상자가 열립니다. 이 대화 상자에서 이미지를 셀에 추가하기 전에 이미지 원본 등의 속성을 설정할 수 있습니다.
