# <a name="report-wizard-help"></a>보고서 마법사 도움말
  이 섹션에서는 보고서 마법사에 대한 F1 도움말을 설명합니다. 보고서 마법사에서는 테이블 형식 보고서 및 행렬 보고서 생성 프로세스를 자동화합니다. 다음 표에서는 이 섹션에서 다루는 항목에 대해 설명합니다.  
  
 보고서 마법사를 열려면 **프로젝트** 메뉴에서 **새 항목 추가**를 클릭합니다.
