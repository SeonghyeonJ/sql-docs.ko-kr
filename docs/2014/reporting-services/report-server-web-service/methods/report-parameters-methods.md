# <a name="report-parameters-methods"></a>보고서 매개 변수 메서드
  다음 메서드를 사용하여 보고서의 매개 변수를 설정하고 검색할 수 있습니다.  
  
|메서드|작업|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.GetItemParameters%2A>|지정된 항목에 대한 매개 변수 속성을 반환합니다. 이 메서드를 사용하여 지정된 항목의 매개 변수에 대해 매개 변수 값의 유효성을 검사할 수도 있습니다.|  
|<xref:ReportService2010.ReportingService2010.ListParameterStates%2A>|지원되는 매개 변수 상태 목록을 반환합니다.|  
|<xref:ReportService2010.ReportingService2010.ListParameterTypes%2A>|지원되는 매개 변수 유형 목록을 반환합니다.|  
|<xref:ReportService2010.ReportingService2010.SetItemParameters%2A>|지정된 항목에 대한 매개 변수 속성을 설정합니다.|
