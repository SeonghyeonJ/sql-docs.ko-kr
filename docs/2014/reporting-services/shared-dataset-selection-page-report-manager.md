# <a name="shared-dataset-selection-page-report-manager"></a>공유 데이터 세트 선택 페이지(보고서 관리자)
  공유 데이터 세트 선택 페이지를 사용하여 현재 보고서와 연결된 공유 데이터 세트를 검토하고 수정할 수 있습니다.  
  
 이 페이지에서 보고서에 공유 데이터 세트를 더 추가하거나 보고서에서 공유 데이터 세트를 제거할 수는 없습니다.  
  
## <a name="navigation"></a>탐색  
 사용자 인터페이스(UI)에서 이 위치를 탐색하려면 다음 절차를 사용하십시오.  
  
###### <a name="to-open-the-shared-dataset-selection-page"></a>공유 데이터 세트 선택 페이지를 열려면  
  
1.  보고서 관리자를 열고 공유 데이터 세트를 검토할 보고서를 찾습니다.  
  
2.  보고서 위로 마우스를 이동하여 드롭다운 화살표를 클릭합니다.  
  
3.  드롭다운 메뉴에서 **관리**를 클릭합니다. 보고서의 일반 속성 페이지가 열립니다.  
  
4.  **공유 데이터 세트** 탭을 선택합니다.  
  
## <a name="options"></a>옵션  
 **찾아보기**  
 보고서의 각 공유 데이터 세트 이름에 대해 대상 공유 데이터 세트의 현재 폴더 경로와 이름을 검토할 수 있습니다.  
  
 데이터 세트 이름에서 참조하는 공유 데이터 세트를 변경하려면 **찾아보기** 단추를 클릭합니다.  
  
 **찾아보기** 단추를 누르면 보고서 서버의 폴더 구조가 열립니다. **위치** 필드에 전체 경로를 추가할 공유 데이터 세트를 클릭합니다.  
  
 공유 데이터 세트 선택을 마치려면 **확인** 단추를 누르고, 공유 데이터 세트 찾아보기를 취소하려면 **취소**를 클릭합니다.  
  
 **적용**  
 변경 내용을 저장합니다.
