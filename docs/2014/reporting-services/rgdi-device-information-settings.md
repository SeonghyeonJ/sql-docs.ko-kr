# <a name="rgdi-device-information-settings"></a>RGDI 디바이스 정보 설정
  다음 표는 RGDI 형식으로 렌더링하기 위한 디바이스 정보 설정을 나열합니다.  
  
|설정|값|  
|-------------|-----------|  
|`Page`|렌더링할 보고서의 페이지를 지정합니다. 기본값은 `1`입니다.|  
|`RGDIVersion`|이 설정은 아무런 영향을 미치지 않습니다.|
