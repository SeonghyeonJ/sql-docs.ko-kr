# <a name="delete-columns-dialog-box"></a>열 삭제 대화 상자
  **열 삭제** 대화 상자를 사용하여 테이블, 행렬 또는 목록 데이터 영역에서 열을 삭제할 수 있습니다. 그룹과 연결된 열의 경우에는 그룹 정의도 삭제할 수 있습니다.  
  
## <a name="options"></a>변수  
 **열 및 관련된 그룹 삭제**  
 열 및 관련 그룹 정의를 삭제하려면 이 옵션을 선택합니다.  
  
 **열만 삭제**  
 선택한 열만 삭제하려면 이 옵션을 선택합니다.
