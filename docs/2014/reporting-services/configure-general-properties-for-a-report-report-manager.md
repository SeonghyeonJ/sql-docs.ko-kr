# <a name="configure-general-properties-for-a-report-report-manager"></a>일반적인 보고서 속성 구성(보고서 관리자)
    
### <a name="to-configure-general-report-properties"></a>일반적인 보고서 속성을 구성하려면  
  
1.  [보고서 관리자&#40;SSRS 기본 모드&#41;](../../2014/reporting-services/report-manager-ssrs-native-mode.md)를 시작합니다.  
  
2.  보고서 관리자에서 **내용** 페이지로 이동합니다. 일반 속성을 구성하려는 보고서로 이동한 후 해당 보고서를 엽니다.  
  
3.  **속성** 탭을 클릭합니다.  
  
     또는 합니다 **내용을** 페이지가 자세히 보기로 열려, 속성 페이지 아이콘을 클릭 합니다.  
  
     ![속성 페이지 아이콘](media/prop.gif "속성 페이지 아이콘")  
  
4.  합니다 **일반** 속성 페이지가 표시 되 고 다음과 같은 속성을 구성할 수 있습니다.  
  
    -   에 **속성** 섹션에서는 보고서 이름과 설명을 수정할 수 있습니다.  
  
    -   선택할 수 있습니다 합니다 **목록 뷰에서 숨기기** 가로 및 세로로 페이지 항목을 정렬 하는 기본 페이지 레이아웃 (목록 보기)에서 페이지를 열 때 항목을 숨기려면 확인란을 선택 합니다.  
  
    -   에 **보고서 정의** 섹션에서 클릭 **편집** 보고서 정의의 복사본을 추출 하 합니다. 로컬에서 수정한 보고서 정의 내용은 보고서 서버에 저장되지 않습니다.  
  
         또는.rdl 파일에서 보고서 정의 업데이트 하려면 클릭 **업데이트**합니다.  
  
        > [!NOTE]  
        >  보고서 정의를 업데이트하는 경우에는 업데이트를 완료한 후 데이터 원본 설정을 다시 설정해야 합니다.  
  
    -   사용 된 **삭제** 또는 **이동** 단추를 삭제 하거나 보고서를 이동 합니다.  
  
    -   또한 링크된 보고서를 만들 수도 있습니다.  
  
5.  보고서에 대 한 일반 속성을 구성 했으면 클릭 **적용**합니다.
