# <a name="license-terms"></a>사용 조건
  사용 조건 페이지를 사용하여 이 릴리스의 사용 조건에 동의할 수 있습니다.  
  
## <a name="options"></a>옵션  
 설치를 계속하려면 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 사용 조건에 동의해야 합니다.  
  
 **인쇄** 단추를 사용하여 이 릴리스의 사용 조건 복사본을 인쇄할 수 있습니다. 설치를 종료하려면 **취소**를 클릭합니다.
