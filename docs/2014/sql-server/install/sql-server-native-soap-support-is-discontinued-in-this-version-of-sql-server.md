# <a name="sql-server-native-soap-support-is-discontinued-in-this-version-of-sql-server"></a>SQL Server 기본 SOAP 지원 기능은 이 SQL Server 버전에서 더 이상 제공되지 않습니다.
  업그레이드 관리자가 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 네이티브 XML 웹 서비스 사용을 발견했습니다.  
  
## <a name="component"></a>구성 요소  
 [!INCLUDE[ssDE](../../includes/ssde-md.md)]  
  
## <a name="description"></a>Description  
 이 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 릴리스에서는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 네이티브 XML 웹 서비스가 제거되었습니다.  
  
## <a name="discovering-where-you-use-native-xml-web-services"></a>네이티브 XML 웹 서비스를 사용하는 경우 확인  
 다음과 같은 경우 애플리케이션에서 네이티브 XML 웹 서비스를 사용하는 것을 확인할 수 있습니다.  
  
-   업그레이드 관리자를 실행하는 경우  
  
-   [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 버전을 업그레이드하는 경우  
  
## <a name="corrective-action"></a>수정 동작  
 현재 네이티브 XML 웹 서비스를 사용하는 애플리케이션을 수정합니다.
