# <a name="late-breaking-upgrade-issues"></a>최신 업그레이드 문제
  문서화가 완료된 후 검색 논리가 추가 또는 변경되었을 수 있습니다. 업그레이드 관리자 보고서 뷰어에서 이 항목을 발견했거나 최신 항목을 보려는 경우 업그레이드 관리자와 함께 설치된 추가 정보 파일에서 추가 정보를 확인하십시오.
