# <a name="complete---repair"></a>완료 - 복구
  **완료 – 복구** 페이지를 사용하여 복구가 완료되었음을 확인할 수 있습니다.  
  
## <a name="options"></a>옵션  
 이 대화 상자에는 이 작업에 대한 설치 로그 파일 링크가 포함됩니다. 계속하려면 **닫기**를 클릭합니다.
