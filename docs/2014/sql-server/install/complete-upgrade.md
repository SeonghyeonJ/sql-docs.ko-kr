# <a name="complete---upgrade"></a>완료 - 업그레이드
  완료(업그레이드) 페이지를 사용하여 완료된 업그레이드를 확인할 수 있습니다.  
  
## <a name="options"></a>옵션  
 이 대화 상자에는 이 작업에 대한 설치 로그 파일 링크가 포함됩니다. 계속하려면 **닫기**를 클릭합니다.
