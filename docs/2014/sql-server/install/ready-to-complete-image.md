# <a name="ready-to-complete-image"></a>이미지 완료 준비
  **이미지 완료 준비** 페이지에서는 준비 인스턴스의 구성에 적용되는 구성 설정에 대한 요약 정보를 제공합니다.  
  
## <a name="options"></a>변수  
 이 페이지에서는 다음 옵션을 선택할 수 있습니다.  
  
-   설치 프로그램을 계속하려면 **설치**를 클릭합니다.  
  
-   설치 설정을 변경하려면 **뒤로**를 클릭합니다.  
  
-   설치 프로그램을 끝내려면 **취소**를 클릭합니다.
