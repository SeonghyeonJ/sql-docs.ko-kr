# <a name="upgrade-advisor-errors"></a>업그레이드 관리자 오류
  보고서 뷰어에 업그레이드 관리자 오류가 표시될 수 있습니다. 이러한 오류는 이 도움말 항목에 링크됩니다.  
  
 보고서 뷰어에 표시되는 문제 중 이 항목에 링크되는 문제는 무시해도 됩니다.
