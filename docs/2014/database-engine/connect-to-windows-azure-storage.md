# <a name="connect-to-azure-storage"></a>Azure Storage에 연결
  **연결 Azure Storage** 대화 상자를 사용 하 여 저장소 계정을 지정 하 고 Azure에 대 한 연결의 유효성을 검사 합니다.  
  
## <a name="options"></a>변수  
 Azure 계정에 대 한 다음 정보를 지정 하 고 다음 을 클릭 하 여 계속 합니다.  
  
1.  **저장소 계정** - 저장소 계정 이름을 지정합니다.  
  
2.  **계정 키** - 지정한 저장소 계정의 계정 키를 지정합니다.  
  
3.  **보안 엔드포인트 사용(HTTPS)** – 이 옵션은 암호화된 통신 및 네트워크 웹 서버의 보안 ID를 사용합니다.  
  
4.  **계정 키 저장** - 이 옵션은 암호화된 파일에 암호를 저장합니다.
