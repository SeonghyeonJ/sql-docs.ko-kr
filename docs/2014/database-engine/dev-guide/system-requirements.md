# <a name="system-requirements"></a>시스템 요구 사항
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Management Objects (SMO)를 사용 하 여 [!INCLUDE[msCoName](../../includes/msconame-md.md)] `System.Data.SqlClient` 개체 드라이버를 연결 하 고 인스턴스와 통신 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]합니다.  
  
 SMO 클라이언트에는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 및 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2.0과 함께 제공되는 [!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)] Native Client가 필요합니다.  
  
 SMO를 사용하여 애플리케이션을 개발하려면 [!INCLUDE[msCoName](../../includes/msconame-md.md)][!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 2008이 설치되어 있어야 합니다.
