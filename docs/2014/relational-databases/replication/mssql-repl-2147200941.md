# <a name="mssqlrepl-2147200941"></a>MSSQL_REPL-2147200941

|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|-2147200941|  
|이벤트 원본|MSSQLServer|  
|심볼 이름||  
|메시지 텍스트|병합 프로세스에서 구독자 파티션 유효성 검사 식을 평가하는 중 일치하지 않는 부분을 발견했습니다. 구독을 다시 초기화하여 문제를 해결할 수 있습니다.|  
  
## <a name="explanation"></a>설명  
 병합 프로세스에서 구독자 파티션 유효성 검사 식을 평가하는 중에 불일치가 발견되었습니다.  
  
## <a name="user-action"></a>사용자 동작  
 구독을 다시 초기화합니다.
