# <a name="sqlstatistics"></a>SQLStatistics
  **SQLStatistics** 는 정적 커서에 대해 실행할 수 있습니다. 업데이트 가능한 커서(키 집합 또는 동적 커서)에 대해 **SQLStatistics** 를 실행하려고 하면 커서 유형이 변경되었음을 나타내는 SQL_SUCCESS_WITH_INFO가 반환됩니다.
