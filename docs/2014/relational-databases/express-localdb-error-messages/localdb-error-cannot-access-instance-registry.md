|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|262|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|로컬 데이터베이스 인스턴스 레지스트리 구성에 액세스하는 동안 예기치 않은 오류가 발생했습니다. 자세한 오류 정보는 Windows 애플리케이션 로그를 참조하십시오.|  
  
## <a name="explanation"></a>설명
