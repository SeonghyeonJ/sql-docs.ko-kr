# <a name="cursors-event-category"></a>Cursors 이벤트 범주
  **Cursors** 이벤트 범주에는 커서 동작을 모니터링하는 데 사용하는 이벤트 클래스가 포함됩니다.  
  
## <a name="in-this-section"></a>섹션 내용  
  
|항목|Description|  
|-----------|-----------------|  
|[CursorClose 이벤트 클래스](cursorclose-event-class.md)|API(애플리케이션 프로그래밍 인터페이스) 커서에서 발생하는 커서 닫기 이벤트에 대해 설명합니다.|  
|[CursorExecute 이벤트 클래스](cursorexecute-event-class.md)|API 커서에서 발생하는 커서 실행 이벤트에 대해 설명합니다.|  
|[CursorImplicitConversion 이벤트 클래스](cursorimplicitconversion-event-class.md)|API 또는 [!INCLUDE[tsql](../../includes/tsql-md.md)] 커서에서 발생하는 커서 암시적 변환 이벤트에 대해 설명합니다.|  
|[CursorOpen 이벤트 클래스](cursoropen-event-class.md)|API 커서에서 발생하는 커서 열기 이벤트에 대해 설명합니다.|  
|[CursorPrepare 이벤트 클래스](cursorprepare-event-class.md)|API 커서에서 발생하는 커서 준비 이벤트에 대해 설명합니다.|  
|[CursorRecompile 이벤트 클래스](cursorrecompile-event-class.md)|API 커서에서 발생하는 커서 다시 컴파일 이벤트에 대해 설명합니다.|  
|[CursorUnprepare 이벤트 클래스](cursorunprepare-event-class.md)|API 커서에서 발생하는 커서 준비 취소 이벤트에 대해 설명합니다.|
