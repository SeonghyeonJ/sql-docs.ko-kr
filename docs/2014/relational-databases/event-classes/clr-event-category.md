  **CLR** 이벤트 범주에는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 내의 [!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)] CLR(공용 언어 런타임) 개체를 실행하여 만들어지는 이벤트 클래스가 포함됩니다.
