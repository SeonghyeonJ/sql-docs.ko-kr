# <a name="date-property-clientnetlibinfo-class"></a>Date 속성(ClientNetLibInfo 클래스)
  클라이언트 네트워크 라이브러리 파일의 날짜를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.Date [= value]  
```  
  
## <a name="parts"></a>부분  
 *개체가*  
 클라이언트 네트워크 라이브러리에 대한 정보를 나타내는 [ClientNetLibInfo 클래스](clientnetlibinfo-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 클라이언트 네트워크 라이브러리 파일의 날짜를 지정하는 uint32 값입니다.
