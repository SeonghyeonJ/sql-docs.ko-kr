# <a name="instancename-property-clientsettings-class"></a>InstanceName 속성(ClientSettings 클래스)
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트 인스턴스의 [!INCLUDE[msCoName](../../includes/msconame-md.md)] 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트 인스턴스의 설정을 나타내는 `ClientSettings` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트 인스턴스의 이름을 지정하는 문자열 값입니다.
