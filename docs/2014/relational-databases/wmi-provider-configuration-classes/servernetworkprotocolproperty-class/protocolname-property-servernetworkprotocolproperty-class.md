# <a name="protocolname-property-servernetworkprotocolproperty-class"></a>ProtocolName 속성(ServerNetworkProtocolProperty 클래스)
  참조된 속성을 소유하는 프로토콜의 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>부분  
 *개체가*  
 인스턴스의 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]네트워크 프로토콜 특성을 나타내는 [servernetworkprotocolproperty 클래스](servernetworkprotocolproperty-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 속성을 소유하는 프로토콜의 이름을 지정하는 문자열 값입니다.
