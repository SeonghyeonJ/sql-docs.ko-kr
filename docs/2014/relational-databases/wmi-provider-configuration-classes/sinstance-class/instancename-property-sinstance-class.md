# <a name="instancename-property-sinstance-class"></a>InstanceName 속성(SInstance 클래스)
   [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]인스턴스의 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 [SInstance 클래스](sinstance-class.md) 서버 인스턴스를 나타내는 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]의 인스턴스 이름을 지정하는 문자열 값입니다.
