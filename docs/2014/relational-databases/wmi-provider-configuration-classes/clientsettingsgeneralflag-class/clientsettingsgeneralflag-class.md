# <a name="clientsettingsgeneralflag-class"></a>ClientSettingsGeneralFlag 클래스
  합니다 [ClientSettingsGeneralFlag 클래스](clientsettingsgeneralflag-class.md) 인스턴스의 일반 플래그 설정을 나타냅니다 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 클라이언트입니다.  
  
 합니다 [ClientSettingsGeneralFlag 클래스](clientsettingsgeneralflag-class.md) 다음 작업에 사용할 수 있습니다.  
  
-   플래그 이름을 봅니다.  
  
-   플래그 인덱스 값을 봅니다.  
  
-   플래그 값을 보고 수정합니다.
