# <a name="context-property-securitycertificate-class"></a>Context 속성(SecurityCertificate 클래스)
  보안 인증서의 컨텍스트를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.Context [= value]  
```  
  
## <a name="parts"></a>부분  
 *개체가*  
 보안 인증서를 나타내는 [SecurityCertificate 클래스](securitycertificate-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 보안 인증서의 컨텍스트를 지정하는 `sint32` 배열 값입니다.
