# <a name="mssqlserver_2536"></a>MSSQLSERVER_2536

|attribute|값|  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|2536|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBCC_SUMMARY_FIRST_LINE|  
|메시지 텍스트|'COMMAND'의 DBCC 결과입니다.|
