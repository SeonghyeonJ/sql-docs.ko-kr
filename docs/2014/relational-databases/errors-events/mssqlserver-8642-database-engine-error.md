# <a name="mssqlserver8642"></a>MSSQLSERVER_8642
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|8642|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|EXCHNGSTART_ERR|  
|메시지 텍스트|쿼리 프로세서에서 병렬 쿼리 실행에 필요한 스레드 리소스를 시작할 수 없습니다.|  
  
## <a name="explanation"></a>설명  
서버의 스레드 리소스가 부족합니다.  
  
## <a name="user-action"></a>사용자 동작  
서버의 로드를 줄이고 쿼리를 다시 실행하십시오.
