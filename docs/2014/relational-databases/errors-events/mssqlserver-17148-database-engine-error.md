# <a name="mssqlserver_17148"></a>MSSQLSERVER_17148

|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|17148|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|STARTUP_SERVERUNINSTALL|  
|메시지 텍스트|서비스 제어 관리자로부터 '중지' 요청을 받아 SQL Server를 종료하고 있습니다. 이 메시지는 정보 제공용이므로 추가적인 조치가 필요하지 않습니다.|
