# <a name="mssqlserver7916"></a>MSSQLSERVER_7916
    
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|7916|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBCC2_REPAIR_RECORD_DELETED|  
|메시지 텍스트|복구: 개체 ID O_ID, 인덱스 ID I_ID, 파티션 ID PN_ID, 할당 단위 ID A_ID (type 유형), 페이지 P_ID, 슬롯 S_ID에서 삭제 된 레코드입니다. 인덱스를 다시 작성합니다.|
