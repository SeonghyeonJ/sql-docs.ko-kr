# <a name="mssqlserver_12302"></a>MSSQLSERVER_12302
    
## <a name="details"></a>세부 정보  
  
|attribute|값|  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|12302|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|메시지 텍스트|PRIMARY KEY 제약 조건의 일부인 열 업데이트는 '*construct*'에서 지원되지 않습니다.|  
  
## <a name="user-action"></a>사용자 동작  
 기본 키 제약 조건의 일부인 열을 업데이트하지 마십시오.
