|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|12302|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|HK_UNSUPPORTED_COMPUTED_COLUMNS|
